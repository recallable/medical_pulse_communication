# 心律失常医疗知识库

# 一、心律失常概述

心律失常（Arrhythmia）是指心脏电信号的起源、频率、节律、传导速度或传导路径发生异常，导致心脏跳动过快、过慢、不规则的一组临床综合征。心脏的正常节律由窦房结发起，称为窦性心律，当这一电信号传导过程出现紊乱，就会引发心律失常。作为心血管系统常见疾病，心律失常可单独发病，也可伴随冠心病、高血压性心脏病、心肌病等基础疾病出现，我国患病群体超2亿，且随年龄增长患病率显著升高，重症可诱发晕厥、心力衰竭甚至心源性猝死。

心律失常临床表现差异较大，轻者可无明显症状，仅在体检时发现；重者可出现心悸、头晕、黑朦、胸闷、气短等症状，严重时可发生晕厥或猝死。本病诊断依赖心电图、动态心电图等电生理检查，治疗需结合病因、病情严重程度，采用药物、介入或手术等方式，实现节律控制、症状缓解及并发症预防的目标。

# 二、心律失常基础常识

## 2.1 疾病分型与核心特征

临床常根据心率快慢、起源部位及传导异常类型对心律失常进行分类，核心亚型及特征如下，不同分型的病情严重程度及治疗方案差异显著：

|分型方式|疾病亚型|核心表现与特征|
|---|---|---|
|按心率快慢|快速性心律失常|心率＞100次/分，含窦性心动过速、房性心动过速、心房颤动（房颤）、室上性心动过速、室性心动过速等，常表现为心悸、心慌，重症可致头晕|
||缓慢性心律失常|心率＜60次/分，含窦性心动过缓、房室传导阻滞、窦性停搏等，轻者无症状，重者可出现黑朦、晕厥，甚至心脏骤停|
|按起源部位|室上性心律失常|起源于窦房结、心房、房室结及希氏束以上部位，如房颤、房扑、室上速，病情相对缓和，但若持续时间长可引发血栓栓塞|
||室性心律失常|起源于心室，如室性早搏（室早）、室性心动过速、心室颤动，危险性较高，心室颤动可直接导致心源性猝死|
## 2.2 流行病学与高危人群

- **高发人群**：65岁以上老年人；有心律失常家族史者；冠心病、高血压性心脏病、心肌病、心力衰竭、先天性心脏病患者；甲状腺功能亢进、电解质紊乱（低钾、低镁）者；长期服用某些药物（如洋地黄类、抗抑郁药）者；长期熬夜、情绪波动大、过量饮酒、吸烟及浓茶咖啡依赖者。

- **人群与地域特征**：我国心律失常患病率随年龄增长显著升高，65岁以上人群患病率达30%以上，80岁以上人群超50%；房颤作为最常见的持续性心律失常，在成人中患病率为1%-2%，且北方患病率略高于南方，与基础心血管疾病分布一致。

- **疾病负担**：心律失常易反复发作，影响患者生活质量，重症病例需紧急救治及长期管理，其导致的血栓栓塞、心力衰竭等并发症，不仅增加医疗费用，还显著升高致残率和病死率，是心血管疾病死亡的重要诱因之一。

# 三、心律失常病因与发病机制

## 3.1 核心病因

心律失常的发生与心脏自身结构异常、神经体液调节紊乱、外部诱因等密切相关，部分病例病因不明（特发性心律失常），核心诱因如下：

- **心脏结构与功能异常**：冠心病、心肌梗死导致心肌缺血缺氧，破坏心脏电信号传导路径；高血压性心脏病、心肌病引起心肌肥厚、纤维化，影响电信号起源与传导；心脏瓣膜病、先天性心脏病导致心脏结构畸形，干扰正常电生理活动。

- **全身性疾病与代谢异常**：甲状腺功能亢进（甲亢）激活交感神经，加快心率、诱发心律失常；电解质紊乱（低钾、低镁、高钾）直接影响心肌细胞电活动，引发室早、室速甚至心室颤动；糖尿病、慢性肾病可损伤心脏自主神经，增加心律失常风险。

- **药物与外部诱因**：洋地黄类药物中毒、抗心律失常药滥用、部分抗生素（如喹诺酮类）及抗抑郁药，可干扰心脏电生理；长期熬夜、过度劳累、情绪剧烈波动、过量饮酒、吸烟、浓茶咖啡刺激，可激活交感神经，诱发心律失常；寒冷刺激、剧烈运动也可作为临时诱因。

- **遗传因素**：部分心律失常与遗传相关，如长QT综合征、Brugada综合征等遗传性心律失常疾病，多为常染色体显性遗传，常在青少年或中青年期发病，易诱发猝死。

## 3.2 核心发病机制

心律失常的核心发病机制是心脏电信号的起源异常、传导异常，或两者同时存在。正常情况下，窦房结作为心脏起搏点，发出电信号经心房、房室结、希氏束、浦肯野纤维传导至心室，引发心脏规律收缩。当这一过程出现紊乱，就会导致心律失常。

起源异常是指电信号并非来自窦房结，而是由心脏其他部位（异位起搏点）发起，如房性早搏、室性早搏，多因心肌细胞自律性增高或触发活动异常所致。传导异常包括传导阻滞（电信号传导延迟或中断，如房室传导阻滞）和折返激动（电信号沿异常路径折返传导，形成心动过速，如室上速、房颤）。此外，心肌缺血、缺氧、纤维化等可导致心肌细胞电生理特性改变，降低电信号传导稳定性，进一步加重心律失常。

# 四、心律失常预防与健康管理

## 4.1 一级预防（病因预防）

针对普通人群及高危群体，通过综合干预减少心律失常诱发因素，降低发病风险，核心措施如下：

- **生活方式调理**：规律作息，保证每日7-8小时睡眠，避免熬夜和过度劳累；情绪稳定，避免长期焦虑、激动、抑郁，学会调节心理压力；戒烟限酒，避免过量饮用浓茶、咖啡，减少对心脏的刺激；适度运动，每周进行150分钟中等强度有氧运动（快走、太极拳、游泳），避免剧烈运动，心功能不全者需在医生指导下运动。

- **基础疾病控制**：积极治疗冠心病、高血压、糖尿病、甲亢等基础疾病，严格控制血压、血糖、血脂及甲状腺功能达标，减少心脏结构和功能损伤；避免电解质紊乱，日常均衡饮食，必要时在医生指导下补充钾、镁等电解质。

- **药物与体检管理**：遵医嘱用药，避免自行服用可能诱发心律失常的药物；40岁以上人群每年进行1次心电图检查，高危人群（有基础心脏病、家族史者）每6个月复查，早期发现异常。

## 4.2 二级预防（早筛查、早干预）

- **定期筛查**：常规心电图可初步筛查心律失常，对于心悸、头晕等症状不典型者，可进行24小时动态心电图监测，捕捉阵发性心律失常；必要时进行心脏超声、甲状腺功能、电解质检查，明确病因及心脏结构功能状态。

- **症状监测与早期干预**：警惕心律失常早期信号，如心悸、心慌、头晕、黑朦、胸闷、气短等，及时就医排查；确诊轻症患者（如偶发室早），尽早启动生活方式干预，必要时给予药物治疗，避免病情进展。

- **慢病登记管理**：医疗机构建立心律失常患者分级管理体系，对确诊患者进行定期随访，评估心律控制情况及药物不良反应，调整治疗方案；社区卫生服务中心提供健康指导，提高患者治疗依从性。

## 4.3 三级预防（慢病管理与并发症防治）

- **规范治疗与用药**：患者严格遵医嘱规律服药，不可自行停药、减药或换药，掌握抗心律失常药物的不良反应及应对方法；房颤等易引发血栓的患者，需遵医嘱服用抗凝药物，预防血栓栓塞。

- **康复与生活管理**：介入或手术治疗后患者，在医生指导下进行康复训练，逐步恢复正常活动；避免诱发因素，如寒冷刺激、感染、剧烈运动，预防心律失常复发；定期复查心电图、动态心电图、心脏超声，评估治疗效果及心功能状态。

- **并发症防治**：早期发现血栓栓塞、心力衰竭、心脏骤停等并发症，房颤患者需定期监测凝血功能，必要时调整抗凝方案；出现晕厥、胸痛等重症表现时，立即就医急救。

# 五、心律失常诊断方法

## 5.1 临床诊断（初步筛查）

结合患者病史、症状、体格检查及流行病学史进行初步判断，为后续辅助检查提供方向，适用于基层医疗机构首诊及社区筛查。

诊断要点：① 病史采集：重点询问心律失常家族史、既往基础病史（心脏病、甲亢、糖尿病）、用药史、诱因（熬夜、饮酒、运动），以及症状特点（心悸频率、持续时间、伴随症状）；② 体格检查：测量心率、心律，听诊心脏心音，评估血压、呼吸，快速性心律失常可闻及心率增快、心律不齐，缓慢性心律失常可能出现血压下降、面色苍白。

## 5.2 实验室与影像学诊断（辅助确诊与病情评估）

|诊断方法|检测目标/用途|结果解读与适用场景|注意事项|
|---|---|---|---|
|心电图（常规）|初步筛查心律失常类型|可明确窦性心律、早搏、房颤、房室传导阻滞等心律失常，适用于症状发作时或体检筛查。|检查前避免剧烈运动、情绪激动，保持安静状态，平卧配合检查。|
|24小时动态心电图|捕捉阵发性心律失常，评估发作频率|适用于心悸、头晕症状不典型，常规心电图无异常者，可明确心律失常的类型、发作时间及诱因。|监测期间保持正常生活节奏，避免洗澡、剧烈运动，准确记录症状发作时间。|
|心脏超声|评估心脏结构与功能，排查病因|可发现心肌肥厚、心室扩大、瓣膜病变、心肌病等，明确心律失常是否由心脏结构异常引发。|检查前无需特殊准备，配合医生调整体位，女性避开月经期。|
|电生理检查（EPS）|精准定位异常起搏点及传导路径|有创检查，为介入治疗（射频消融）提供精准定位，适用于药物治疗效果不佳或需明确病因的复杂心律失常。|检查前需告知医生过敏史，术后卧床休息，观察穿刺部位出血情况。|
|血生化与甲状腺功能|排查电解质紊乱、甲亢等诱因|低钾、低镁、高钾提示电解质紊乱；TSH、T3、T4异常提示甲亢/甲减，为病因治疗提供依据。|需空腹采血，采血前避免服用影响甲状腺功能的药物。|
# 六、心律失常治疗方案

## 6.1 治疗原则

心律失常治疗以“去除病因、控制节律、缓解症状、预防并发症、降低病死率”为核心。对于无症状、无基础疾病的偶发心律失常（如偶发室早），可仅进行生活方式干预，定期监测；对于有症状、影响生活质量或有猝死风险的心律失常，需结合药物、介入或手术治疗，控制心律或心率；同时积极治疗基础疾病，纠正电解质紊乱、甲亢等诱因，实现个体化精准治疗。

## 6.2 药物治疗（基础干预手段）

药物治疗适用于多数心律失常患者，需根据心律失常类型、合并症及患者耐受性选择，核心药物类别及应用如下：

|药物类别|代表药物|适用场景|不良反应与注意事项|
|---|---|---|---|
|钠通道阻滞剂|普罗帕酮、奎尼丁|室上性心动过速、房性早搏、室性早搏|可能出现头晕、恶心、腹痛，严重者可诱发心律失常，用药期间监测心电图。|
|β受体阻滞剂|美托洛尔、比索洛尔|窦性心动过速、室上速、房颤（控制心室率）、室早|可能导致心率减慢、乏力、血压下降，不可突然停药，需逐步减量，支气管哮喘患者禁用。|
|钾通道阻滞剂|胺碘酮|房颤、房扑、室性心动过速、心室颤动|可能损伤甲状腺功能、肺纤维化、肝功能异常，用药期间定期复查甲状腺功能、肝功能及胸片。|
|钙通道阻滞剂|维拉帕米、地尔硫卓|室上速、房颤（控制心室率）、窦性心动过速|可能出现头晕、下肢水肿、便秘，低血压患者慎用，避免与β受体阻滞剂联用。|
|抗凝药物|华法林、利伐沙班|房颤、房扑等易引发血栓的心律失常，预防血栓栓塞|华法林需定期监测凝血功能（INR），调整剂量；利伐沙班出血风险较低，避免与抗血小板药滥用。|
## 6.3 介入与手术治疗（复杂病例干预）

- **射频消融术**：微创介入治疗，通过导管将射频能量送至心脏异常起搏点或传导路径，使病变心肌组织凝固坏死，阻断异常电信号。适用于室上速、房颤、房扑、室性心动过速、频发室早等，具有创伤小、恢复快、治愈率高的优势，术后可减少或停用抗心律失常药物。

- **心脏起搏器植入术**：适用于缓慢性心律失常（如重度房室传导阻滞、窦性心动过缓），通过植入起搏器发放电信号，刺激心脏跳动，维持正常心率。起搏器分为临时起搏器（短期急救）和永久起搏器（长期治疗），术后需定期复查起搏器功能。

- **植入式心脏复律除颤器（ICD）**：适用于有猝死风险的严重心律失常（如心室颤动、恶性室性心动过速），可自动监测心律，当出现致命性心律失常时，发放电击除颤，挽救生命。

## 6.4 重症急救与并发症治疗

- **心室颤动急救**：立即进行心肺复苏（CPR），同时给予电除颤，恢复窦性心律；静脉输注肾上腺素、胺碘酮等药物，维持心律稳定，转入ICU监护治疗，排查并纠正诱因（如电解质紊乱、心肌梗死）。

- **严重心动过缓急救**：出现晕厥、黑朦、血压下降时，立即给予阿托品静脉注射，提升心率；必要时植入临时起搏器，维持心脏供血，避免心脏骤停。

- **血栓栓塞并发症**：房颤等患者发生脑栓塞时，立即就医，早期可行溶栓或介入取栓治疗；肢体动脉栓塞时，及时进行溶栓或手术取栓，避免肢体坏死；后续长期服用抗凝药物，预防复发。

# 七、鉴别诊断与转诊指征

## 7.1 常见疾病鉴别诊断

|鉴别疾病组合|核心鉴别点|
|---|---|
|心律失常与自主神经功能紊乱|心律失常：心悸呈阵发性或持续性，心电图/动态心电图可发现电信号异常，伴头晕、黑朦；自主神经紊乱：心悸与情绪、熬夜相关，无器质性病变，心电图正常，调节生活方式或服用谷维素后可缓解。|
|房颤与房扑|房颤：心率绝对不规则，心音强弱不等，心电图可见P波消失，代之以f波；房扑：心率相对规则，心电图可见锯齿状F波，两者治疗方案略有差异，电生理检查可明确区分。|
|室性早搏与房性早搏|室性早搏：心电图可见提前出现的宽大畸形QRS波群，代偿间歇完全，症状较明显；房性早搏：QRS波群形态正常，代偿间歇不完全，部分患者无症状，药物治疗侧重点不同。|
## 7.2 转诊指征

出现以下情况，需及时转诊至上级医院心血管内科或ICU治疗：

- 突发心悸、头晕、黑朦、晕厥，疑似心室颤动、室性心动过速、重度房室传导阻滞等重症心律失常；

- 心律失常反复发作，药物治疗效果不佳，需进一步行电生理检查或介入治疗（射频消融、起搏器植入）；

- 合并心力衰竭、心肌梗死、血栓栓塞等并发症，常规治疗无法控制；
