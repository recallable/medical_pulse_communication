# 病毒性流感医疗知识库

# 一、病毒性流感简介

病毒性流感（简称流感）是由流感病毒引起的急性呼吸道传染病，具有传染性强、传播速度快、流行范围广的特点，属于我国丙类传染病。流感病毒主要分为甲、乙、丙三型，其中甲型流感（甲流）变异频繁，易引发全球性大流行；乙型流感（乙流）变异较慢，多呈区域性季节性流行；丙型流感（丙流）致病性较弱，仅引起散发病例或轻微流行。流感临床以高热、头痛、肌肉酸痛、乏力等全身症状为主，呼吸道症状较轻，严重者可并发肺炎、呼吸衰竭等，危及生命。

全球每年约有数十亿人感染流感，我国流感流行季主要集中在冬春季（11月至次年3月），部分年份夏季可能出现小高峰。儿童、老年人、孕妇及患有慢性基础疾病者为高危人群，感染后重症风险较高。早期干预治疗可显著降低重症发生率，多数轻症患者经对症治疗后1-2周可痊愈。

# 二、病毒性流感基础常识

## 2.1 病原学与病理特征

流感病毒属于正黏病毒科，为单股负链RNA病毒，病毒颗粒呈球形，直径80-120nm，表面含血凝素（HA）和神经氨酸酶（NA）两种关键糖蛋白，这两种蛋白是病毒感染和传播的核心，也是疫苗研发和抗病毒药物作用的主要靶点。甲型流感病毒根据HA和NA的不同可分为多种亚型（如H1N1、H3N2），易发生抗原转变和抗原漂移，导致新毒株出现；乙型流感病毒仅分为Victoria系和Yamagata系，变异以抗原漂移为主；丙型流感病毒无亚型划分，致病性最弱。

流感病毒对热、紫外线、消毒剂敏感，56℃加热30分钟、75%酒精、含氯消毒剂均可有效灭活病毒，但在低温环境下可存活数月，这也是冬春季流感易流行的原因之一。病毒主要侵犯呼吸道黏膜上皮细胞，引发局部炎症反应，同时可通过血液循环诱发全身性免疫反应。

## 2.2 流行病学特征

- **传染源**：主要为流感患者和无症状感染者，患者发病后1-7天具有传染性，发病后24-48小时传染性最强，无症状感染者虽症状轻微，但仍可排毒传播病毒，是重要的隐性传染源。

- **传播途径**：以呼吸道飞沫传播为主要途径，患者咳嗽、打喷嚏时产生的飞沫被他人吸入后引发感染；同时可通过接触被病毒污染的手、餐具、衣物、环境表面等间接传播，密闭空间内气溶胶传播风险也需警惕。

- **易感人群**：人群普遍易感，其中5岁以下儿童、65岁以上老年人、孕妇、免疫功能低下者、患有慢性心肺疾病、糖尿病、肾病等基础疾病者为高危人群，感染后易发展为重症，需重点防护。

# 三、病毒性流感病因与发病机制

## 3.1 核心病因

病毒性流感的直接病因是感染甲、乙、丙型流感病毒，其中甲型和乙型流感病毒是引发流行的主要病原体，感染后是否发病及病情严重程度，与病毒毒力、感染剂量及机体免疫力密切相关，具体诱发因素如下：

- **病毒感染因素**：接触流感患者或无症状感染者，吸入含病毒飞沫，或接触被病毒污染的物品后，病毒通过呼吸道侵入人体，黏附并侵入呼吸道上皮细胞，完成复制增殖后扩散，引发感染。甲型流感病毒因变异快，易出现新毒株，人群普遍缺乏免疫力，易引发大规模流行。

- **机体免疫因素**：机体免疫力低下时，对病毒的抵抗力减弱，易受感染且病情更重。如儿童免疫系统尚未发育成熟、老年人免疫系统衰退、长期使用免疫抑制剂者、过度劳累、受凉、营养不良等，均会降低机体免疫功能，增加感染风险。

- **环境与行为因素**：冬春季气温低、湿度小，呼吸道黏膜屏障功能减弱，且人群室内活动增多、空间密闭、通风不良，导致病毒易传播；集体场所（学校、托幼机构、养老院）人群密集，易发生聚集性疫情；未接种流感疫苗者，缺乏特异性免疫力，感染风险显著高于接种者。

## 3.2 发病机制

流感病毒侵入人体后，首先通过表面血凝素（HA）与呼吸道上皮细胞表面的唾液酸受体结合，进入细胞内进行复制增殖，产生大量子代病毒，子代病毒通过神经氨酸酶（NA）的作用从感染细胞释放，进一步侵犯周围呼吸道上皮细胞，导致黏膜上皮细胞变性、坏死、脱落，引发局部炎症反应，出现咳嗽、咽痛、流涕等呼吸道症状。

病毒复制过程中产生的毒素及机体对病毒的免疫反应，会诱发全身性炎症反应，刺激机体释放白细胞介素、肿瘤坏死因子等炎症因子，这些因子作用于全身组织器官，导致高热、头痛、肌肉酸痛、乏力等全身症状。重症病例多因免疫反应过度激活引发“细胞因子风暴”，导致肺部严重炎症、急性呼吸窘迫综合征，甚至多器官功能衰竭，危及生命，此情况在高危人群中更易发生。

# 四、病毒性流感预防措施

## 4.1 一级预防（病因预防）

- **接种流感疫苗**：接种流感疫苗是预防流感最经济有效的手段，可刺激机体产生特异性抗体，显著降低感染风险及感染后重症发生率。我国推荐每年9-11月（流感流行季前）接种，重点人群优先接种；目前常用三价或四价流感疫苗，可覆盖甲、乙流主要流行毒株，建议每年定期接种。

- **做好个人防护**：流感流行季减少前往人群密集、通风不良的场所，必要时佩戴医用外科口罩；咳嗽、打喷嚏时用纸巾或肘部遮挡口鼻，避免飞沫传播；勤用流动水和洗手液规范洗手，揉搓时间不少于20秒，无洗手条件时使用含75%酒精的免洗洗手液。

- **优化环境与生活习惯**：居住、工作、学习场所每日定时通风换气，每次不少于30分钟，保持空气流通；对门把手、手机、餐具等高频接触物品，用含氯消毒剂或75%酒精擦拭消毒；保证充足睡眠、均衡饮食、适度体育锻炼，增强机体免疫力，根据气温变化及时增减衣物，避免受凉。

## 4.2 二级预防（早发现、早隔离、早治疗）

- **主动监测与隔离**：流感流行季密切关注自身及家人健康状况，出现发热、咳嗽等症状时，及时居家隔离，避免前往公共场所，隔离至体温正常后24小时且症状明显缓解，减少病毒传播；集体单位发现聚集性发热病例，需及时上报并采取管控措施。

- **暴露后预防**：与流感患者密切接触后，需居家观察7天，每日监测体温及症状，期间避免接触他人。高危人群可在医生指导下，于接触后48小时内服用奥司他韦等抗病毒药物进行预防，降低感染概率。

- **重点场所防控**：学校、托幼机构、养老院等集体场所，需加强晨午检制度，及时发现发热病例并隔离；定期开展环境消毒，避免聚集性疫情发生。

# 五、病毒性流感诊断方法

## 5.1 临床诊断（初步筛查）

结合流行病学史、临床表现进行初步判断，适用于基层医疗机构快速识别，为后续诊疗提供参考。

诊断要点：① 流行病学史：发病前7天内接触过流感患者，或处于流感流行区域、集体单位聚集性发病环境；② 临床表现：突发高热（体温≥38℃）、头痛、肌肉酸痛、乏力，伴咳嗽、咽痛、流涕等呼吸道症状，部分患者可出现恶心、呕吐、腹泻等消化道症状。

## 5.2 实验室与影像学诊断（辅助确诊与评估）

|诊断方法|检测目标/用途|优势|局限性|适用场景|
|---|---|---|---|---|
|核酸检测（RT-PCR）|流感病毒RNA，区分甲、乙、丙型及亚型|特异性高、灵敏度高，可早期确诊，为治疗提供依据|需专业设备和技术，检测时间1-4小时，基层普及度有限|临床确诊、重症病例评估、聚集性疫情溯源|

# 六、病毒性流感治疗方案

## 6.1 一般治疗与对症治疗

适用于轻症流感患者，以缓解症状、支持机体自愈为主，具体措施如下：

- **隔离休息**：居家或住院隔离，保证充足休息，减少体力消耗，避免劳累加重病情，同时避免传播他人。

- **对症处理**：高热、头痛、肌肉酸痛者，可服用对乙酰氨基酚、布洛芬等非甾体抗炎药（儿童禁用阿司匹林，以防瑞氏综合征）；咳嗽、咳痰者，给予氨溴索、乙酰半胱氨酸等止咳祛痰药物；咽痛者可使用含漱液、润喉片缓解症状。

- **补液支持**：鼓励多饮水，多摄入清淡易消化的食物，维持水电解质平衡；呕吐、腹泻严重者，给予口服补液盐，必要时静脉补液，避免脱水。

## 6.2 抗病毒治疗（核心治疗）

抗病毒治疗需早期启动，建议在发病48小时内使用，可缩短病程、减轻症状、降低重症风险，常用药物如下：

|药物名称|作用机制|用法要点|注意事项|
|---|---|---|---|
|奥司他韦|抑制神经氨酸酶，阻止病毒从感染细胞释放，阻断传播|口服，成人每次75mg，每日2次，疗程5天；儿童按体重调整剂量|不良反应轻微，偶有恶心、呕吐，饭后服用可减轻胃肠道刺激|
|扎那米韦|作用于神经氨酸酶，抑制病毒释放，发挥抗病毒作用|吸入给药，成人及12岁以上儿童，每次10mg，每日2次，疗程5天|不适用于哮喘、慢阻肺患者，可能诱发气道痉挛|
|帕拉米韦|新型神经氨酸酶抑制剂，抗病毒作用强效、持久|静脉滴注，成人单次300mg，重症患者可重复给药|用于重症病例或无法口服药物者，需监护肝肾功能|
## 6.3 重症病例治疗

高危人群感染后易发展为重症，出现呼吸困难、低氧血症、意识障碍等症状时，需立即住院治疗，采取综合救治措施：

- **呼吸支持**：给予吸氧、无创通气，病情进展者及时行气管插管、有创机械通气，严重呼吸衰竭者可采用体外膜肺氧合（ECMO）支持，维持呼吸功能。

- **抗炎与免疫调节**：针对“细胞因子风暴”，在医生指导下合理使用糖皮质激素，避免滥用导致感染加重；必要时给予免疫球蛋白，增强机体抵抗力。

- **并发症治疗**：积极治疗肺部继发感染、心肌损伤、肾功能衰竭等并发症，加强营养支持，保护重要器官功能，降低病死率。

# 七、病毒性流感临床特征与诊疗路径

## 7.1 临床分型与表现

- **轻症型**：占绝大多数，病程5-7天，主要表现为突发高热（38-40℃）、头痛、肌肉酸痛、乏力，伴轻微咳嗽、咽痛、流涕，无明显呼吸困难，症状持续3-5天后逐渐缓解，预后良好，可自愈或经对症治疗康复。

- **重症型**：发病后病情进展较快，出现持续高热（≥39℃，持续3天以上）、呼吸困难、胸闷、喘憋、口唇发绀，可伴有意识模糊、嗜睡、烦躁不安、呕吐、腹泻等症状，严重者引发急性呼吸窘迫综合征、感染性休克、多器官功能衰竭，需紧急救治。

- **特殊人群表现**：儿童感染后发热症状更明显，易出现高热惊厥，部分患儿以消化道症状（呕吐、腹泻）为主要表现；老年人感染后全身症状重，合并基础疾病者易发生重症，症状不典型者可能仅表现为低热、乏力；孕妇感染后可能影响胎儿发育，增加早产风险，自身重症率高于普通人群。

## 7.2 临床诊疗路径

流感诊疗遵循“筛查-诊断-分型治疗-随访”标准化路径，确保诊疗规范高效，具体流程如下：

1. 初步筛查：基层医疗机构结合流行病学史、临床表现，对发热患者进行流感抗原检测，快速区分流感与普通感冒、新冠病毒感染等。

2. 确诊与分型：抗原检测阳性者可初步诊断，核酸检测阳性为确诊依据，结合症状、体征及胸部CT、血氧饱和度检查，判断轻症或重症。

3. 分层治疗：轻症患者居家隔离，给予对症治疗，必要时早期服用抗病毒药物；重症患者立即转至定点医院，启动重症救治流程，加强器官功能支持。

4. 随访与康复：轻症患者治疗后3-5天随访，监测症状消退及体温恢复情况；重症患者康复后定期复查肺功能、心功能，避免后遗症，同时指导康复护理。

## 7.3 临床鉴别诊断

流感需与普通感冒、新冠病毒感染、支原体肺炎、细菌性肺炎等疾病相鉴别，核心鉴别要点如下：

|鉴别疾病|核心鉴别点|
|---|---|
|普通感冒|全身症状轻微，以鼻塞、流涕、咽痛等上呼吸道症状为主，发热多为低热，无明显肌肉酸痛，病程3-5天，流感抗原/核酸检测阴性。|
|新冠病毒感染|可出现发热、咳嗽、乏力，部分患者伴有嗅觉味觉减退、腹泻，新冠核酸/抗原检测阳性，流感检测阴性，传染性较强。|
|支原体肺炎|发热多为中低热，以刺激性干咳为主要症状，病程较长（1-2周），支原体抗体检测阳性，流感检测阴性，胸片/CT可见肺部炎症浸润影。|
