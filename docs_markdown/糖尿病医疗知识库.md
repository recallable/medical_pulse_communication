# 糖尿病医疗知识库

# 一、糖尿病概述

糖尿病是一组以慢性血葡萄糖（简称血糖）水平增高为特征的代谢性疾病，核心病因是胰岛素分泌不足或胰岛素抵抗，导致葡萄糖无法有效被机体组织利用，进而引发糖、脂肪、蛋白质代谢紊乱。作为全球高发的慢性疾病，我国成人糖尿病患病率已超11%，其中2型糖尿病占比达90%以上，且发病呈年轻化趋势。

糖尿病起病隐匿，早期多无明显症状，长期高血糖可损伤全身多系统，引发心脑血管疾病、肾脏病变、眼底病变、周围神经病变等严重并发症，是导致残疾、死亡的重要原因之一。部分急性并发症（如糖尿病酮症酸中毒、高渗性高血糖状态）发病急骤，若救治不及时，病死率极高。糖尿病需长期综合管理，通过饮食、运动、药物、监测及教育干预，维持血糖稳定，延缓并发症进展。

# 二、糖尿病基础常识

## 2.1 疾病分型与核心特征

临床根据病因、发病机制及人群特点，将糖尿病主要分为4型，其中1型、2型糖尿病最为常见，核心特征如下：

|分型|核心病因|高发人群|核心特征|
|---|---|---|---|
|1型糖尿病|自身免疫损伤胰岛β细胞，胰岛素分泌绝对不足|青少年、儿童，无明显性别差异|起病急，症状典型（多饮、多食、多尿、体重骤降），依赖胰岛素治疗，易发生酮症酸中毒|
|2型糖尿病|胰岛素抵抗为主，伴胰岛素分泌相对不足|40岁以上中老年人，肥胖者、有家族史者高发|起病缓慢，早期症状隐匿，多伴肥胖、高血压、高脂血症，早期可通过饮食、口服药控制|
|妊娠期糖尿病|妊娠期激素变化诱发胰岛素抵抗，糖代谢异常|妊娠期女性，尤其高龄、肥胖、有家族史者|多无明显症状，产后多数可恢复，但未来患2型糖尿病风险升高，需严格控制血糖保障母婴安全|
|其他特殊类型糖尿病|遗传缺陷、胰腺疾病、药物诱发等明确病因|无特定人群，与原发病相关|症状各异，需针对原发病治疗，同时控制血糖|
## 2.2 流行病学与高危人群

- **高发人群**：40岁以上中老年人，代谢功能衰退，患病风险显著升高；有糖尿病家族史者，直系亲属患病风险较普通人群高2-3倍；肥胖（BMI≥24）、中心性肥胖者，胰岛素抵抗风险增加；长期久坐、缺乏运动、饮食高油高脂高糖者；高血压、高脂血症、冠心病患者；妊娠期糖尿病病史者、多囊卵巢综合征患者。

- **人群与地域特征**：糖尿病发病率随经济发展水平升高而上升，城市人群患病率高于农村；男性略高于女性，且发病年龄逐步年轻化，青少年肥胖型2型糖尿病发病率显著增加；沿海地区、经济发达地区因饮食精细化、肥胖率高，患病率更高。

- **疾病负担**：糖尿病为终身性疾病，需长期用药、监测血糖，医疗费用较高；并发症可导致患者失明、肾衰竭、截肢等残疾，严重影响生活质量，同时占用大量医疗资源，是我国重要的公共卫生问题。

# 三、糖尿病病因与发病机制

## 3.1 核心病因

糖尿病的发生是遗传因素、环境因素及机体代谢调节异常共同作用的结果，不同分型病因各有侧重：

- **遗传因素**：糖尿病具有明显家族遗传倾向，1型糖尿病与HLA基因多态性相关，2型糖尿病受多基因调控，直系亲属患病者，个体患病风险显著升高。遗传因素主要影响胰岛素分泌功能、胰岛素受体敏感性，为疾病发生提供易感背景。

- **环境与生活方式因素**：肥胖是2型糖尿病最主要的环境诱因，尤其是中心性肥胖，可直接诱发胰岛素抵抗；长期高油高脂高糖饮食、暴饮暴食，导致热量摄入过剩，加重胰腺代谢负担，损伤胰岛功能；长期缺乏运动，可降低机体对胰岛素的敏感性，延缓葡萄糖代谢；精神压力过大、熬夜、长期应激，可升高糖皮质激素水平，拮抗胰岛素作用，诱发血糖升高。

- **自身免疫与腺体损伤**：1型糖尿病由自身免疫功能紊乱引发，机体产生胰岛自身抗体（如GADAb、ICA抗体），攻击并破坏胰岛β细胞，导致胰岛素分泌绝对不足；胰腺疾病（如胰腺炎、胰腺肿瘤）、胰腺手术切除，可直接损伤胰岛组织，影响胰岛素分泌。

- **其他诱因**：长期服用糖皮质激素、利尿剂等药物，可拮抗胰岛素作用，诱发药物性高血糖；妊娠期女性体内雌激素、孕激素水平升高，抑制胰岛素活性，诱发妊娠期糖尿病；年龄增长导致胰岛功能自然衰退，是中老年人患病的重要诱因。

## 3.2 核心发病机制

糖尿病的核心发病机制可概括为胰岛素分泌不足、胰岛素抵抗两大核心，最终导致血糖调节功能紊乱，同时引发脂肪、蛋白质代谢异常。

胰岛素分泌不足：胰岛β细胞是机体唯一分泌胰岛素的细胞，1型糖尿病中，自身免疫攻击导致胰岛β细胞大量破坏，胰岛素分泌绝对缺乏，葡萄糖无法被组织摄取利用，血糖升高；2型糖尿病早期以胰岛素抵抗为主，后期胰岛β细胞功能衰退，胰岛素分泌相对不足，血糖控制能力下降。此外，胰岛α细胞分泌胰高血糖素过多，可促进肝糖原分解，进一步升高血糖。

胰岛素抵抗：指机体组织（如肝脏、肌肉、脂肪）对胰岛素敏感性下降，即使胰岛素水平正常或升高，也无法有效促进葡萄糖摄取、利用及储存。肝脏胰岛素抵抗可导致肝糖原异生增加、糖原分解加快，空腹血糖升高；肌肉、脂肪组织胰岛素抵抗，可导致餐后葡萄糖摄取减少，餐后血糖升高。胰岛素抵抗还可引发高胰岛素血症，进一步诱发肥胖、高血压、高脂血症，形成代谢综合征。

长期高血糖可通过“糖毒性”损伤全身组织，一方面直接损伤血管内皮细胞，引发动脉粥样硬化，导致心脑血管疾病、肾脏病变；另一方面损伤神经组织，影响周围神经、自主神经功能，引发肢体麻木、感觉异常、胃肠功能紊乱等并发症。

# 四、糖尿病预防与健康管理

## 4.1 一级预防（病因预防）

针对普通人群及高危群体，通过综合干预减少致病因素，降低糖尿病发病风险，核心措施如下：

- **生活方式干预**：保持规律作息，避免熬夜，保证每日7-8小时睡眠，维持胰岛功能稳定；适度运动，每周进行150分钟中等强度有氧运动（快走、慢跑、游泳、太极拳），搭配2-3次力量训练，控制体重，改善胰岛素敏感性；调节情绪，缓解精神压力，避免长期焦虑、应激，减少糖皮质激素对胰岛素的拮抗作用。

- **饮食调理**：规律饮食，三餐定时定量，避免暴饮暴食，控制总热量摄入；均衡膳食，减少精制糖、高油高脂食物摄入，增加全谷物、新鲜蔬菜、优质蛋白（鱼、虾、瘦肉、豆制品）摄入；控制碳水化合物摄入量，选择低升糖指数食物，少食多餐，避免餐后血糖骤升；戒烟限酒，避免过量饮酒影响血糖控制，诱发低血糖。

- **高危人群防控**：有家族病史、肥胖、高血压、高脂血症等高危因素者，定期筛查血糖，建立健康档案；肥胖者逐步减重，将BMI控制在18.5-23.9之间，可显著降低发病风险；妊娠期女性定期监测血糖，及时发现并干预妊娠期糖尿病；避免长期滥用激素药物，用药需遵医嘱，监测血糖变化。

## 4.2 二级预防（早筛查、早干预）

- **定期筛查**：40岁以上人群每年进行空腹血糖、餐后2小时血糖、糖化血红蛋白筛查；高危人群每6个月复查一次，必要时行葡萄糖耐量试验，早期发现糖尿病前期；妊娠期女性在孕24-28周进行糖耐量试验，筛查妊娠期糖尿病。

- **症状监测与早期干预**：警惕糖尿病早期信号，如多饮、多食、多尿、体重不明原因下降、乏力、视物模糊、肢体麻木等，及时就医排查；确诊糖尿病前期（空腹血糖6.1-6.9mmol/L，餐后2小时血糖7.8-11.0mmol/L）者，立即通过饮食、运动干预，必要时药物治疗，逆转或延缓发展为糖尿病。

- **慢病管理前置**：糖尿病前期人群控制每日总热量，减少高糖、高油食物，增加运动，每周减重0.5-1kg；定期监测血糖，每3-6个月复查糖化血红蛋白，评估血糖控制情况；合并高血压、高脂血症者，同步控制血压、血脂，降低并发症风险。

## 4.3 三级预防（慢病管理与并发症防治）

- **规范治疗与用药**：糖尿病患者严格遵医嘱规律用药，不可自行停药、减药，根据分型及病情选择合适药物（口服降糖药或胰岛素）；掌握药物用法及不良反应，如口服降糖药可能引发低血糖、胃肠道反应，胰岛素需注意注射部位轮换，避免局部反应；定期复查血糖、糖化血红蛋白、肝肾功能，根据结果调整治疗方案。

- **血糖自我监测**：1型糖尿病、血糖波动大或使用胰岛素者，每日监测4-7次血糖（空腹、三餐后2小时、睡前，必要时凌晨2-3点）；2型糖尿病口服药控制稳定者，每周监测2-4次血糖；记录血糖变化，为医生调整治疗方案提供依据，同时警惕低血糖，随身携带糖果、饼干，出现心慌、手抖、出汗时及时补充。

- **并发症筛查与防治**：每年进行一次并发症筛查，包括眼底检查（预防视网膜病变）、尿微量白蛋白检测（预防肾病）、神经传导速度检查（预防周围神经病变）、血管超声（预防动脉粥样硬化）；控制血压、血脂、血糖达标，戒烟限酒，延缓并发症进展；出现并发症及时干预，如眼底病变早期行激光治疗，肾病患者控制蛋白摄入，必要时透析治疗。

# 五、糖尿病诊断方法

## 5.1 临床诊断（初步筛查）

结合患者病史、症状、体格检查及流行病学史进行初步判断，为后续辅助检查提供方向，适用于基层医疗机构首诊及社区筛查。

诊断要点：① 病史采集：重点询问家族病史、饮食史、运动史、用药史、妊娠史，以及症状特点（多饮、多食、多尿、体重变化、乏力、视物模糊等出现时间、持续规律）；② 体格检查：测量身高、体重、BMI、血压，评估精神状态、肢体感觉，初步判断病情及是否存在并发症迹象。

## 5.2 实验室与特殊检查（辅助确诊与病情评估）

|诊断方法|检测目标/用途|结果解读与诊断标准|注意事项|
|---|---|---|---|
|空腹血糖检测|评估基础血糖水平，初步筛查糖尿病|正常：＜6.1mmol/L；糖尿病前期：6.1-6.9mmol/L；糖尿病：≥7.0mmol/L（空腹指至少8小时未进食）。|需空腹采血，避免剧烈运动、情绪激动，采血前避免服用影响血糖的药物。|
|餐后2小时血糖检测|评估餐后血糖调节能力，诊断糖尿病及前期|正常：＜7.8mmol/L；糖尿病前期：7.8-11.0mmol/L；糖尿病：≥11.1mmol/L（从进食第一口饭开始计时）。|可与葡萄糖耐量试验同步进行，日常监测需保持饮食规律，避免刻意控制饮食。|
|糖化血红蛋白（HbA1c）|评估近2-3个月平均血糖水平，排除偶然血糖波动|正常：＜6.5%；糖尿病：≥6.5%；糖尿病前期：5.7%-6.4%。|不受进食影响，可随机采血；贫血、血红蛋白异常者结果可能不准确，需结合其他指标。|
|葡萄糖耐量试验（OGTT）|明确诊断糖尿病前期、妊娠期糖尿病，鉴别血糖异常类型|空腹血糖＜6.1mmol/L，餐后2小时血糖＜7.8mmol/L为正常；餐后2小时血糖7.8-11.0mmol/L为糖耐量异常；≥11.1mmol/L为糖尿病。|需空腹进行，服糖后定时采血（30分钟、1小时、2小时、3小时）；试验前3天正常饮食，避免高糖、高脂食物。|
|胰岛自身抗体检测|鉴别1型与2型糖尿病|GADAb、ICA抗体阳性，多提示1型糖尿病；2型糖尿病患者抗体多为阴性。|采血前无需特殊准备，抗体阳性需结合临床症状、血糖水平综合判断。|
# 六、糖尿病治疗方案

## 6.1 治疗原则

糖尿病治疗以“控制血糖、纠正代谢紊乱、预防并发症、改善生活质量”为核心，遵循“五驾马车”综合管理原则（饮食治疗、运动治疗、药物治疗、血糖监测、糖尿病教育）。1型糖尿病以胰岛素替代治疗为主，配合饮食、运动干预；2型糖尿病以饮食、运动为基础，逐步加用口服降糖药，必要时联合胰岛素治疗；妊娠期糖尿病需严格控制血糖，优先饮食、运动干预，必要时胰岛素治疗，避免使用口服降糖药。

## 6.2 药物治疗（核心干预手段）

药物治疗需根据糖尿病分型、病情、合并症及患者个体情况选择，核心药物类别及应用如下：

|药物类别|代表药物|适用场景|用法要点与不良反应|
|---|---|---|---|
|双胍类|二甲双胍|2型糖尿病一线用药，尤其适用于肥胖患者；糖尿病前期干预|饭前或饭中服用，初始剂量宜小，逐渐加量；不良反应以胃肠道反应（腹胀、腹泻）为主，肾功能不全者禁用，避免乳酸酸中毒。|
|磺脲类|格列齐特、格列美脲、格列吡嗪|2型糖尿病，胰岛功能尚存者|饭前30分钟服用，需严格控制剂量；主要不良反应为低血糖，老年患者慎用；长期使用可能导致体重增加。|
|SGLT-2抑制剂|达格列净、恩格列净、卡格列净|2型糖尿病，尤其适用于合并心血管疾病、肾病患者|晨起空腹服用，不受进食影响；可减重、降压，降低心血管事件风险；不良反应为泌尿感染、脱水、酮症酸中毒，肾功能不全者慎用。|
|GLP-1受体激动剂|利拉鲁肽、司美格鲁肽|2型糖尿病，肥胖或合并心血管疾病患者|皮下注射，每日1次或每周1次，给药方便；可减重、改善胰岛功能；不良反应为胃肠道反应（恶心、呕吐），有胰腺炎病史者慎用。|
|胰岛素|短效（普通胰岛素）、中效（低精蛋白锌胰岛素）、长效（甘精胰岛素）、预混胰岛素|1型糖尿病；2型糖尿病口服药无效、胰岛功能衰退者；妊娠期糖尿病；糖尿病急性并发症|短效胰岛素饭前30分钟皮下注射，长效胰岛素固定时间注射；需轮换注射部位，监测血糖调整剂量；主要不良反应为低血糖，过量可危及生命。|
## 6.3 饮食与运动治疗（基础干预）

- **饮食治疗**：控制总热量，每日摄入量根据身高、体重、活动量计算，肥胖者适当减少；合理分配营养素，碳水化合物占总热量50%-60%，蛋白质15%-20%，脂肪20%-30%；选择低升糖指数食物（如燕麦、芹菜、柚子），避免精制糖、蜂蜜、含糖饮料；少食多餐，每日3餐或5-6餐，避免餐后血糖骤升，同时预防低血糖。

- **运动治疗**：选择适合自身的有氧运动，如快走、慢跑、游泳、骑自行车，每周至少150分钟，每次30分钟以上，循序渐进；运动时间选择餐后1-2小时，避免空腹运动（预防低血糖）；合并并发症者需调整运动方式，如眼底病变、肾病患者避免剧烈运动，周围神经病变者避免足部受伤；运动前后监测血糖，运动中随身携带糖果，出现不适及时停止。

## 6.4 急性并发症急救与对症治疗

- **糖尿病酮症酸中毒（DKA）急救**：立即补液，首选生理盐水，纠正脱水与电解质紊乱；小剂量胰岛素静脉输注（每小时0.1U/kg），逐步降低血糖；纠正酸中毒，仅在pH＜7.1时少量补充碳酸氢钠；抗感染治疗，去除诱因（如感染、擅自停药）；密切监测血糖、电解质、血气分析、尿量，血糖降至13.9mmol/L时，改用葡萄糖+胰岛素输注，预防低血糖。

- **高渗性高血糖状态（HHS）急救**：快速补液，纠正高渗状态，首选生理盐水，必要时补充低渗盐水；小剂量胰岛素静脉输注，控制血糖下降速度；纠正电解质紊乱，监测钾、钠、氯水平；去除诱因（如感染、饮水不足），转入ICU监护，预防脑水肿、休克。

- **低血糖急救**：轻度低血糖（心慌、手抖、出汗、头晕），立即口服15-20g葡萄糖（如糖水、糖果），15分钟后复查血糖，未恢复则重复补充；严重低血糖（昏迷、意识障碍），立即静脉输注50%葡萄糖注射液，清醒后改为口服补充，同时查找低血糖诱因（如药物过量、饮食不足），调整治疗方案。

# 七、鉴别诊断与转诊指征

## 7.1 常见疾病鉴别诊断

|鉴别疾病组合|核心鉴别点|
|---|---|
|1型糖尿病与2型糖尿病|1型糖尿病：青少年发病，起病急，症状典型，胰岛自身抗体阳性，依赖胰岛素治疗，易发生酮症酸中毒；2型糖尿病：中老年发病，起病缓慢，多伴肥胖，抗体阴性，早期可通过饮食、口服药控制，胰岛素抵抗明显。|

## 7.2 转诊指征

出现以下情况，需及时转诊至上级医院内分泌科或ICU治疗：

- 突发糖尿病酮症酸中毒、高渗性高血糖状态、严重低血糖昏迷，病情危急，需紧急抢救；

- 新诊断糖尿病患者，分型不明确，或合并其他内分泌疾病，需进一步检查明确病因；

- 血糖控制不佳，口服降糖药联合胰岛素治疗后，血糖仍持续异常，或频繁出现低血糖、血糖波动过大；

- 出现严重并发症，如糖尿病视网膜病变（视物模糊、失明）、肾病（蛋白尿、肾衰竭）、神经病变（肢体麻木、溃疡）、心脑血管疾病；

- 妊娠期糖尿病患者，血糖控制不佳，或合并妊娠并发症，需专科评估母婴安全；

- 儿童、青少年糖尿病患者，需专科制定胰岛素治疗方案，指导长期管理。
