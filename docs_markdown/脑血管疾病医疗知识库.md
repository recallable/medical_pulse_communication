# 脑血管疾病医疗知识库

# 一、脑血管疾病概述

脑血管疾病是一组因脑血管破裂出血、阻塞缺血或血管畸形导致脑功能障碍的疾病总称，主要累及中枢神经系统，是神经系统最常见的高发疾病之一。其核心病理改变为脑血管结构异常或功能障碍引发的脑组织缺血、缺氧、水肿或受压损伤，临床以肢体偏瘫、言语障碍、认知下降、意识障碍等神经功能缺损为主要表现。

脑血管疾病具有高发病率、高致残率、高致死率、高复发率及低治愈率的特点，我国成人患病率约为3%-5%，且发病率随年龄增长显著升高，40岁以上人群患病风险呈指数级上升，北方地区发病率高于南方，城市高于农村。该病发病急骤，多数患者需紧急救治，幸存者中70%以上遗留不同程度的神经功能后遗症，给家庭及社会带来沉重医疗负担。脑血管疾病诊断需结合病史、症状、影像学及实验室检查综合判断，治疗以急性期挽救脑组织、恢复期功能康复为核心，强调危险因素的长期管控。

# 二、脑血管疾病基础常识

## 2.1 疾病分型与核心特征

根据发病机制及病理改变，脑血管疾病主要分为缺血性脑血管病、出血性脑血管病两大类，另有脑血管畸形等少见类型，核心分型及特征如下：

|分型类别|典型疾病|核心病因|核心特征|
|---|---|---|---|
|缺血性脑血管病（占比70%-80%）|脑梗死、短暂性脑缺血发作（TIA）|动脉粥样硬化、血栓形成、栓塞（心源性/非心源性）、血管狭窄|起病较缓（数分钟至数小时），症状持续或短暂缓解，表现为肢体麻木无力、言语不清、视物模糊等，无颅内压升高典型表现|
|出血性脑血管病（占比20%-30%）|脑出血、蛛网膜下腔出血|高血压性动脉硬化、动脉瘤破裂、血管畸形、凝血功能障碍|起病急骤（数分钟内达高峰），伴剧烈头痛、呕吐、意识障碍，颅内压升高明显，严重时危及生命|
|其他类型|脑血管畸形、脑静脉窦血栓、脑小血管病|先天性发育异常、感染、凝血异常、长期高血压|症状因人而异，可表现为反复头痛、癫痫发作、渐进性神经功能下降，部分可突发出血|
## 2.2 流行病学与高危人群

- **高发人群**：60岁以上老年人，血管弹性减退、动脉粥样硬化进展加速，患病风险显著升高；高血压、糖尿病、高血脂患者，是脑血管病核心高危群体，三者可协同加重血管损伤；吸烟、过量饮酒者，烟草有害物质及酒精可损伤血管内皮，诱发血栓形成或血管破裂；心房颤动、心脏瓣膜病患者，易形成心源性血栓，诱发脑栓塞；肥胖、长期久坐、缺乏运动者，可加重代谢紊乱，加速血管硬化；有脑血管病家族史者，遗传因素增加患病易感性。

- **人群与地域特征**：缺血性脑血管病发病率高于出血性，且随年龄增长差距更明显；北方地区因高盐饮食、寒冷气候、高血压患病率高，脑血管病发病率显著高于南方；城市人群因生活压力大、饮食结构失衡、肥胖率高，患病率高于农村；男性发病率略高于女性，女性绝经后因雌激素保护作用消失，患病风险接近男性。

- **疾病负担**：脑血管病急性期治疗费用高昂，且多数患者需长期康复护理，占用大量医疗资源；幸存者遗留的偏瘫、失语、认知障碍等后遗症，导致劳动力丧失，生活质量严重下降；复发率高达15%-20%/年，二次发病患者神经功能损伤更严重，致死致残率显著升高，是我国重大公共卫生挑战。

# 三、脑血管疾病病因与发病机制

## 3.1 核心病因

脑血管疾病病因复杂，以血管壁损伤、血液成分异常、血流动力学改变为核心诱因，不同类型疾病病因各有侧重，具体如下：

- **血管壁损伤**：动脉粥样硬化是缺血性脑血管病最主要病因，高血压、高血脂、糖尿病、吸烟等因素可损伤血管内皮，导致脂质沉积、斑块形成，使血管狭窄、闭塞；高血压性动脉硬化可导致脑小动脉玻璃样变、破裂，引发脑出血；先天性血管畸形（如动静脉畸形）、动脉瘤，因血管壁结构异常，易破裂出血；感染、炎症（如血管炎）可直接损伤血管壁，诱发血栓或出血。

- **血液成分与凝血异常**：血小板增多、血液黏稠度升高，可加速血栓形成，诱发缺血性脑血管病；凝血功能亢进、抗凝药物使用不当，可增加出血风险；心源性血栓（如心房颤动、心肌梗死附壁血栓）脱落，随血流阻塞脑血管，引发脑栓塞；镰状细胞贫血、血小板减少性紫癜等血液疾病，可分别诱发缺血或出血。

- **血流动力学改变**：高血压是脑血管病首要危险因素，长期高压可加重血管壁损伤，血压骤升时易诱发血管破裂，血压过低则可导致脑部灌注不足，诱发缺血性病变；低血压、脱水、心力衰竭等，可导致脑血流量减少，脑组织缺血缺氧；脑血管痉挛（如蛛网膜下腔出血后），可加重脑部缺血，恶化病情。

- **其他诱因**：长期熬夜、精神压力过大、情绪剧烈波动，可导致血压骤升、血管收缩，诱发脑血管事件；过量运动、用力排便等增加颅内压的行为，可诱发高血压患者脑血管破裂；药物因素（如溶栓药、抗血小板药）使用不当，可增加出血风险，毒品（如可卡因）可诱发血管痉挛或破裂。

## 3.2 核心发病机制

脑血管疾病核心发病机制围绕脑部血供异常展开，缺血性与出血性疾病机制差异显著，且存在相互影响的病理过程。

缺血性脑血管病机制：动脉粥样硬化斑块形成后，可导致血管狭窄，脑部灌注不足，引发脑组织缺血缺氧；斑块破裂后，血小板及凝血因子聚集，形成血栓，完全阻塞血管，导致脑梗死；心源性血栓脱落阻塞脑血管，可快速引发相应供血区域神经功能缺损；缺血后脑组织可出现中心坏死区及周围半暗带，若及时恢复血供，半暗带脑组织功能可挽救，否则会进展为坏死。

出血性脑血管病机制：长期高血压导致脑小动脉壁玻璃样变、纤维素样坏死，形成微小动脉瘤，血压骤升时动脉瘤破裂，血液溢出至脑组织（脑出血）或蛛网膜下腔（蛛网膜下腔出血）；血液压迫周围脑组织，引发水肿、颅内压升高，进一步加重脑组织损伤；蛛网膜下腔出血后，血液成分可诱发脑血管痉挛，减少脑部供血，加重缺血性损伤，形成“出血-痉挛-缺血”恶性循环。

共同病理过程：无论缺血还是出血，均可引发脑组织水肿、炎症反应，释放炎症因子（如肿瘤坏死因子、白细胞介素），加重神经细胞损伤；神经细胞缺血缺氧超过6小时可发生不可逆坏死，导致神经功能永久性缺损；颅内压过高时，可引发脑疝，压迫脑干等核心区域，危及生命。

# 四、脑血管疾病预防与健康管理

## 4.1 一级预防（病因预防）

针对普通人群及高危群体，通过综合干预减少致病因素，降低脑血管病发病风险，核心措施如下：

- **危险因素管控**：严格控制血压，高血压患者目标血压＜130/80mmHg，规律服药，定期监测，避免血压骤升骤降；控制血糖，糖尿病患者空腹血糖控制在4.4-7.0mmol/L，餐后2小时血糖＜10.0mmol/L，定期筛查糖化血红蛋白；调节血脂，低密度脂蛋白胆固醇目标＜1.8mmol/L，遵医嘱服用调脂药，减少脂质沉积；戒烟限酒，吸烟者完全戒烟，饮酒者男性每日酒精摄入量＜25g，女性＜15g，最好不饮酒。

- **生活方式干预**：保持规律作息，避免熬夜，保证每日7-8小时睡眠，减少血管应激；适度运动，每周进行150分钟中等强度有氧运动（快走、慢跑、游泳），避免剧烈运动，肥胖者逐步减重，BMI维持在18.5-23.9之间；饮食调理，低盐低脂低糖饮食，每日盐摄入量＜5g，增加全谷物、新鲜蔬菜、优质蛋白摄入，减少肥肉、油炸食品、腌制食品摄入；调节情绪，避免长期焦虑、情绪剧烈波动，通过冥想、散步等方式缓解压力。

- **高危人群防控**：心房颤动患者，遵医嘱服用抗凝药物（华法林、利伐沙班），预防血栓形成；有脑血管病家族史者，定期进行血管超声、血压、血糖、血脂筛查，早发现异常；长期服用避孕药、抗凝药者，严格遵医嘱用药，定期监测凝血功能；头部外伤、血管炎患者，积极治疗原发病，定期复查血管功能。

## 4.2 二级预防（早筛查、早干预）

- **定期筛查**：40岁以上人群每年进行血压、血糖、血脂检测，每2-3年进行颈动脉超声检查，评估血管狭窄程度；高血压、糖尿病、高血脂患者每6个月复查相关指标，每年进行颈动脉超声筛查；心房颤动患者每3-6个月复查心电图或动态心电图，监测心律变化；有短暂性脑缺血发作病史者，每月复查凝血功能、血脂，每半年进行脑血管影像学检查。

- **早期信号识别与干预**：警惕脑血管病早期信号（FAST原则）：面部不对称、肢体无力、言语不清、视物模糊，出现任一症状立即就医；短暂性脑缺血发作患者，及时启动抗血小板、调脂治疗，避免进展为脑梗死；发现颈动脉狭窄≥50%且有症状者，或狭窄≥70%无症状者，评估手术指征，必要时行支架植入或内膜剥脱术。

- **慢病管理前置**：缺血性脑血管病高危人群，遵医嘱规律服用阿司匹林、氯吡格雷等抗血小板药物，避免自行停药；高血压患者选用长效降压药，平稳控制血压，避免血压波动；糖尿病患者严格控制血糖，预防周围血管及脑血管病变进展。

## 4.3 三级预防（慢病管理与并发症防治）

- **规范治疗与用药**：脑血管病患者急性期严格遵医嘱治疗，缺血性患者在溶栓窗口期内评估溶栓/取栓指征，恢复期规律服用抗血小板、调脂、降压药物；出血性患者急性期控制出血、降低颅内压，恢复期控制血压，避免再次出血；定期复查脑血管影像学、凝血功能、肝肾功能，根据结果调整用药方案，避免药物不良反应。

- **长期康复与监测**：遗留神经功能缺损者，发病后3-6个月内启动康复训练（黄金恢复期），包括肢体功能训练、言语训练、认知训练，结合针灸、理疗、作业治疗，最大限度恢复功能；慢性期每3-6个月复查脑血管超声、血压、血糖、血脂，评估病情变化，调整康复方案；避免劳累、情绪激动、用力排便等诱发因素，预防疾病复发。

- **并发症防治**：长期卧床患者，定期翻身、拍背，预防压疮、肺部感染；肢体瘫痪患者，进行肢体按摩、被动运动，预防深静脉血栓；认知障碍患者，加强护理，避免走失、跌倒、误吸；言语障碍患者，坚持言语训练，必要时借助辅助沟通工具；癫痫发作患者，遵医嘱服用抗癫痫药物，避免发作时意外受伤。

# 五、脑血管疾病诊断方法

## 5.1 临床诊断（初步筛查）

结合患者病史、症状、体格检查及发病特点进行初步判断，快速区分缺血性与出血性脑血管病，为紧急救治提供方向，适用于首诊及急诊场景。

诊断要点：① 病史采集：重点询问既往病史（高血压、糖尿病、房颤等）、用药史、吸烟饮酒史，以及发病时间、诱因、症状进展情况；② 体格检查：评估意识状态、瞳孔大小、肢体肌力、言语功能、病理征，结合血压、心率，初步判断病变类型及严重程度。

## 5.2 实验室与特殊检查（辅助确诊与病情评估）

|诊断方法|检测目标/用途|结果解读与诊断价值|注意事项|
|---|---|---|---|
|头颅CT|快速区分缺血性与出血性脑血管病，评估病变范围|脑出血表现为高密度出血灶，蛛网膜下腔出血可见脑沟、脑池高密度影；脑梗死发病6小时内可能无明显异常，后期表现为低密度梗死灶|急诊首选，检查快速，有辐射，孕妇、儿童慎用；对脑干、小脑微小病变分辨率较低|
|头颅MRI|明确缺血性梗死灶位置、范围，发现微小出血及血管病变|DWI序列可发现发病2小时内急性脑梗死；对脑干、小脑病变分辨率高于CT，可清晰显示血管畸形|检查时间较长，幽闭恐惧症患者需镇静；体内有金属植入物者禁忌；出血急性期不如CT敏感|
|脑血管造影（DSA）|评估脑血管狭窄、闭塞、畸形、动脉瘤，为介入治疗提供依据|可清晰显示血管形态，明确狭窄部位及程度、动脉瘤大小及位置，是脑血管病介入治疗的“金标准”|属有创检查，可能引发血管损伤、造影剂过敏，术后需压迫止血，监测生命体征|
|血液/凝血功能检查|排查感染、代谢异常、凝血功能障碍，评估用药安全性|血常规提示感染或贫血；血糖、血脂评估代谢危险因素；凝血功能异常提示出血风险，指导抗凝药物使用|需空腹采血，采血前避免高脂饮食、饮酒；遵医嘱停用影响凝血功能的药物|
|颈动脉超声|评估颈动脉粥样硬化程度、斑块性质、血管狭窄情况|可发现颈动脉斑块（软斑、硬斑、混合斑），判断狭窄程度，预测脑梗死风险|无创、便捷，可重复检查；检查前无需特殊准备，避免颈部压迫|
# 六、脑血管疾病治疗方案

## 6.1 治疗原则

脑血管疾病治疗遵循“分型施治、急慢有别、综合干预”原则。急性期：缺血性脑血管病优先恢复脑部血供（溶栓、取栓、抗血小板），出血性脑血管病重点控制出血、降低颅内压、挽救生命；恢复期：以神经功能康复为核心，结合药物治疗、康复训练、危险因素管控，预防复发；介入与手术治疗需严格把握指征，精准干预病变血管，减少创伤。

## 6.2 药物治疗（核心干预手段）

药物治疗需根据疾病类型、病情阶段及患者耐受性选择，核心药物类别及应用如下：

|药物类别|代表药物|适用场景|用法要点与不良反应|
|---|---|---|---|
|溶栓药物|阿替普酶、尿激酶|急性缺血性脑梗死（阿替普酶窗口期＜4.5小时，尿激酶＜6小时）|静脉输注，严格把控适应症，避免出血风险；不良反应为颅内出血、牙龈出血，用药后密切监测意识及凝血功能。|

## 6.3 介入与手术治疗（复杂病例干预）

- **介入治疗**：适用于急性缺血性脑梗死血栓取栓、脑血管狭窄支架植入、动脉瘤栓塞。急性取栓术可快速清除血管内血栓，恢复脑部血供，窗口期＜8小时；支架植入术用于脑血管狭窄≥70%且有症状者，或狭窄≥50%但症状频繁发作者；动脉瘤栓塞术通过介入手段堵塞动脉瘤，预防破裂出血。术后需抗凝、抗血小板治疗，监测血管通畅情况，预防支架内血栓。

- **手术治疗**：适用于脑出血血肿清除、脑动脉瘤夹闭、颈动脉内膜剥脱术。脑出血血肿较大（幕上＞30ml，幕下＞10ml）或脑疝风险者，行血肿清除术，减轻脑组织压迫；颈动脉内膜剥脱术用于颈动脉狭窄≥70%，切除粥样硬化斑块，预防脑梗死；动脉瘤夹闭术用于复杂动脉瘤，通过手术夹闭动脉瘤颈，阻止出血。术后重点监测意识、神经功能，预防颅内感染、出血、脑脊液漏等并发症。

## 6.4 康复与对症治疗

- **康复治疗**：急性期病情稳定后尽早启动，肢体功能障碍者进行关节活动度训练、肌力训练、平衡训练，结合物理治疗（针灸、电疗、按摩）；言语障碍者进行发音训练、理解训练、吞咽训练；认知障碍者进行记忆力、注意力训练；康复训练需循序渐进，个体化制定方案，避免过度训练导致损伤，坚持长期康复。

- **对症治疗**：颅内压升高者除脱水药物外，必要时行去骨瓣减压术，挽救生命；癫痫发作患者，遵医嘱服用抗癫痫药物（卡马西平、丙戊酸钠），控制发作；感染患者选用敏感抗生素，加强护理，促进排痰；应激性溃疡患者，使用抑酸药物（奥美拉唑），预防消化道出血。

- **支持治疗**：意识障碍、吞咽困难者，给予鼻饲饮食或静脉营养，保证营养摄入；长期卧床者，使用气垫床、定期翻身，预防压疮及深静脉血栓；焦虑、抑郁患者，给予抗焦虑、抗抑郁药物，同时结合心理干预，改善情绪状态，提高康复积极性。

# 七、鉴别诊断与转诊指征

## 7.1 常见疾病鉴别诊断

## 7.2 转诊指征

出现以下情况，需及时转诊至上级医院神经内科、神经外科或卒中中心治疗：

- 突发脑血管病症状（肢体偏瘫、言语不清、剧烈头痛呕吐、意识障碍），怀疑急性脑梗死、脑出血，需紧急评估溶栓/取栓、手术指征；

- 脑血管影像学检查提示脑血管狭窄≥70%、动脉瘤、血管畸形，需介入或手术治疗，基层医疗机构无法开展；

- 药物治疗效果不佳，如短暂性脑缺血发作频繁发作、血压血糖控制不佳，或出现严重药物不良反应（颅内出血、肝肾功能损伤）；

- 遗留严重神经功能障碍（肢体完全瘫痪、重度认知障碍、吞咽困难），需上级医院制定个体化康复方案；

- 病因不明的脑血管病症状，需进一步完善DSA、基因检测等检查明确诊断；

- 脑血管病术后出现并发症（颅内感染、出血、支架内血栓），需进一步诊疗。
