# 慢性呼吸系统疾病医疗知识库

# 一、慢性呼吸系统疾病概述

慢性呼吸系统疾病（chronic respiratory diseases，CRDs）是一组以气道和肺组织慢性炎症、结构重塑、功能不可逆损伤为核心特征的疾病总称，病程迁延反复，需长期规范管理，是全球重大公共卫生问题。本病主要包括慢性阻塞性肺疾病（COPD）、支气管哮喘、支气管扩张症、间质性肺病、慢性肺源性心脏病等，其中慢性阻塞性肺疾病和支气管哮喘发病率最高，占慢性呼吸系统疾病的80%以上。

慢性呼吸系统疾病具有“高发病率、高致残率、高病死率、高疾病负担”的特点，全球每年因本病死亡人数超400万，我国是高发国家，尤其在中老年人群中患病率显著升高。本病发病与长期吸烟、空气污染、过敏因素、遗传背景及慢性感染密切相关，临床主要表现为慢性咳嗽、咳痰、喘息、进行性呼吸困难，病情进展可累及心脏、骨骼等多个器官，严重影响患者生活质量，甚至导致呼吸衰竭、心力衰竭等重症结局。早期筛查、长期规律干预及健康管理是延缓病情进展、降低病死率的关键。

# 二、慢性呼吸系统疾病基础常识

## 2.1 病原学与病理特征

慢性呼吸系统疾病多为非感染性疾病，但慢性感染是重要诱因和病情加重因素。病原学方面，急性加重期常见致病菌包括肺炎链球菌、流感嗜血杆菌、铜绿假单胞菌、卡他莫拉菌等，病毒感染以流感病毒、副流感病毒、呼吸道合胞病毒为主，支原体、衣原体感染也可诱发急性发作。非感染性致病因素主要包括烟草烟雾、粉尘、有害气体、过敏原、遗传变异等，其中烟草烟雾是慢性阻塞性肺疾病最主要的致病因素。

病理特征上，不同疾病存在特异性改变：慢性阻塞性肺疾病表现为气道慢性炎症、黏膜充血水肿、腺体增生肥大、气道狭窄及肺泡过度充气、间隔破坏（肺气肿改变），形成不可逆气流受限；支气管哮喘以气道慢性炎症、气道高反应性、平滑肌增生肥厚、黏液分泌增多为核心，气流受限呈可逆性；支气管扩张症可见支气管结构破坏、管腔扩张变形，伴黏膜慢性炎症及分泌物潴留；间质性肺病以肺间质纤维化、肺泡结构破坏为主要表现，导致限制性通气功能障碍；慢性肺源性心脏病则伴随肺动脉高压、右心室肥厚扩张，最终引发右心衰竭。

## 2.2 流行病学特征

- **高发人群**：40岁以上中老年人为主要高发群体，65岁以上老年人患病率可达15%以上；长期吸烟者（包括二手烟、三手烟暴露者）、职业暴露人群（长期接触粉尘、有害气体、化学物质者，如建筑工人、矿山工人、环卫工人）；过敏体质者、有慢性呼吸系统疾病家族史者；患有慢性感染性呼吸道疾病（如肺结核、慢性支气管炎）且治疗不规范者；长期暴露于空气污染、室内油烟环境的人群。

- **地域与季节因素**：我国北方地区因冬季寒冷、空气污染严重，患病率高于南方；农村地区因取暖方式、职业特点及医疗资源相对匮乏，发病率略高于城市。秋冬季节气温骤变、雾霾天气增多，易诱发疾病急性发作；春季花粉、尘螨增多，是支气管哮喘急性发作的高发季节。

- **疾病负担特点**：慢性呼吸系统疾病病程长、反复急性发作，需长期用药、定期复查及反复住院，给患者家庭和社会带来沉重经济负担；疾病后期因肺功能严重受损，患者生活自理能力下降，致残率高，同时易合并心血管疾病、骨质疏松、焦虑抑郁等并发症，进一步加重病情负担。

# 三、慢性呼吸系统疾病病因与发病机制

## 3.1 核心病因

慢性呼吸系统疾病的发生是环境因素与个体因素共同作用的结果，长期慢性刺激导致气道和肺组织持续损伤，最终形成不可逆病变，具体病因如下：

- **环境危险因素**：长期吸烟是首要危险因素，烟草烟雾中的尼古丁、焦油等有害物质可损伤气道黏膜纤毛功能，诱发慢性炎症，加速气道结构重塑；空气污染（雾霾、沙尘、工业废气）、室内油烟、生物燃料燃烧产生的有害气体，可刺激气道黏膜，加重炎症反应；职业暴露（粉尘、化学物质、放射性物质）可直接损伤肺组织，诱发特异性慢性肺部疾病；反复呼吸道感染（病毒、细菌、支原体）可导致气道黏膜反复受损，慢性炎症迁延不愈，最终发展为慢性疾病。

- **个体与遗传因素**：遗传变异与部分疾病密切相关，如α₁-抗胰蛋白酶缺乏是慢性阻塞性肺疾病的重要遗传诱因，支气管哮喘与多个易感基因相关；过敏体质者接触过敏原（花粉、尘螨、宠物毛发、食物过敏原）后，易引发气道免疫反应，诱发哮喘发作；年龄增长导致肺功能自然衰退，气道防御能力下降，也会增加患病风险；免疫功能低下者因对病原体抵抗力较弱，慢性感染发生率更高，病情更易进展。

- **其他诱发因素**：长期过度劳累、熬夜、营养不良可导致机体免疫功能下降，加重气道炎症；气候变化（寒冷、潮湿）可刺激气道平滑肌收缩，诱发喘息症状；精神心理因素（焦虑、抑郁）可通过神经-体液调节影响气道功能，增加急性发作风险。

## 3.2 发病机制

慢性呼吸系统疾病的核心发病机制是气道和肺组织的慢性炎症、氧化应激与气道结构重塑，不同疾病机制各有侧重但相互关联。慢性阻塞性肺疾病的发病机制为：有害因素刺激引发气道上皮损伤，激活中性粒细胞、巨噬细胞等炎症细胞，释放大量炎症因子，导致气道慢性炎症持续存在，同时蛋白酶-抗蛋白酶失衡、氧化应激加剧，引发气道黏膜增生、腺体分泌增多、气道狭窄及肺泡结构破坏，形成不可逆气流受限。

支气管哮喘的发病机制以免疫介导的气道慢性炎症为核心，过敏原或刺激物激活嗜酸性粒细胞、肥大细胞、T淋巴细胞等炎症细胞，释放炎症介质，导致气道高反应性，接触诱因后气道平滑肌痉挛、黏膜水肿、黏液分泌增多，引发可逆性气流受限，长期反复炎症刺激可导致气道结构重塑，使气流受限逐渐转为不可逆。支气管扩张症则因气道反复感染和炎症，导致支气管壁平滑肌、弹性纤维破坏，管腔扩张变形，分泌物潴留，进一步加重感染与炎症恶性循环。慢性肺源性心脏病则由慢性肺部疾病导致肺循环阻力增加、肺动脉高压，长期负荷过重引发右心室肥厚扩张，最终导致右心衰竭。

# 四、慢性呼吸系统疾病预防与健康管理

## 4.1 一级预防（病因预防）

- **戒烟与避害**：严格戒烟是预防慢性阻塞性肺疾病最关键的措施，同时避免接触二手烟、三手烟；职业暴露人群需做好防护措施（佩戴防尘口罩、防护服），定期进行职业健康体检，必要时调整工作岗位；雾霾、沙尘天气减少外出，外出时佩戴防雾霾口罩；室内保持通风换气，减少油烟、生物燃料燃烧产生的有害气体暴露。

- **增强体质与免疫调节**：合理饮食，保证蛋白质、维生素、矿物质摄入，多吃新鲜蔬菜、水果、优质蛋白，避免挑食偏食；规律作息，保证充足睡眠，避免熬夜；适度进行体育锻炼，如快走、慢跑、游泳、呼吸操、太极拳等，增强体质，改善肺功能；过敏体质者避免接触过敏原，家中定期清洁除螨，春季减少外出或佩戴口罩，必要时进行过敏原特异性免疫治疗。

- **预防慢性感染**：按免疫规划接种流感疫苗、肺炎球菌疫苗，高危人群每年定期接种，减少呼吸道感染风险；出现急性呼吸道感染症状时，及时规范治疗，避免病情迁延转为慢性感染；长期卧床、老年患者定期翻身、拍背，促进痰液排出，预防坠积性肺炎。

## 4.2 二级预防（早筛查、早诊断、早干预）

- **定期筛查**：40岁以上人群、长期吸烟者、职业暴露人群、有慢性呼吸系统疾病家族史者，每年进行1次肺功能检查（FEV₁/FVC、FEV₁占预计值百分比），早期发现气流受限及肺功能异常；同时结合胸部CT检查，排查支气管扩张、间质性肺病等病变，实现早期诊断。

- **症状监测与早期干预**：警惕慢性呼吸系统疾病早期信号，如慢性咳嗽、咳痰持续超过2周、反复喘息、活动后气短、夜间憋醒等，及时就医排查，明确诊断后尽早启动规范治疗，延缓病情进展；支气管哮喘患者需早期识别过敏诱因，避免接触并规范用药，控制炎症。

- **慢病筛查管理**：医疗机构建立慢性呼吸系统疾病筛查登记制度，对确诊患者进行分级管理，定期随访评估病情，调整治疗方案；社区卫生服务中心开展家庭医生签约服务，为患者提供个性化健康指导，提高治疗依从性。

## 4.3 三级预防（慢病管理与并发症防治）

- **规范用药管理**：患者需严格遵医嘱规律用药，避免自行停药、减药或换药，掌握吸入装置（气雾剂、干粉吸入剂）的正确使用方法；慢性阻塞性肺疾病稳定期长期使用支气管舒张剂、吸入性糖皮质激素，急性加重期及时就医调整治疗；支气管哮喘患者区分控制药物与缓解药物，随身携带缓解药物，以备急性发作时使用。

- **生活方式干预**：患者需戒烟限酒，避免辛辣、刺激性食物，饮食清淡易消化；适度进行呼吸功能锻炼（腹式呼吸、缩唇呼吸），每日2-3次，每次5-10分钟，增强呼吸肌力量，改善肺功能；避免过度劳累、情绪激动，预防急性发作；长期氧疗患者严格遵循氧疗方案（低流量吸氧1-2L/min，每日不少于15小时），定期清洁消毒吸氧装置。

- **并发症防治**：定期监测肺功能、心电图、血常规等指标，早期发现肺动脉高压、心力衰竭、骨质疏松、焦虑抑郁等并发症；积极治疗慢性感染，避免炎症扩散加重病情；出现急性发作症状（高热、咳脓痰、呼吸困难加重）时，及时就医，避免延误重症救治。

# 五、慢性呼吸系统疾病诊断方法

## 5.1 临床诊断（初步筛查）

结合患者病史、症状、体格检查及流行病学史进行初步判断，为后续辅助检查提供方向，适用于基层医疗机构首诊及社区筛查。

诊断要点：① 病史采集：重点询问吸烟史、职业暴露史、过敏史、家族病史、既往感染史，以及症状特点（咳嗽性质、咳痰性状、喘息频率、呼吸困难程度、急性发作次数）；② 体格检查：观察呼吸频率、节律、口唇发绀情况，听诊肺部呼吸音、干湿性啰音、哮鸣音，慢性重症患者可见杵状指、桶状胸，慢性肺源性心脏病患者可闻及肺动脉瓣第二心音亢进、颈静脉怒张、下肢水肿。

## 5.2 实验室与影像学诊断（辅助确诊与病情评估）

# 六、慢性呼吸系统疾病治疗方案

## 6.1 治疗原则

慢性呼吸系统疾病治疗以“长期控制、延缓进展、预防急性发作、改善肺功能、提高生活质量”为核心原则。稳定期以长期规律用药、生活方式干预、呼吸功能锻炼为主，控制慢性炎症，维持肺功能；急性加重期以控制感染、缓解症状、纠正呼吸衰竭为重点，及时调整治疗方案；慢性肺源性心脏病患者需同时兼顾肺部疾病与心脏功能保护，防治心力衰竭。治疗过程中需重视个体化干预，根据患者年龄、病情严重程度、合并症等调整方案，提高治疗依从性。

## 6.2 稳定期治疗

- **慢性阻塞性肺疾病**：长期使用支气管舒张剂（沙丁胺醇、异丙托溴铵、噻托溴铵等），改善气流受限；中重度患者联合吸入性糖皮质激素（布地奈德、氟替卡松），控制慢性炎症；合并咳痰困难者，使用祛痰药（氨溴索、乙酰半胱氨酸）；长期氧疗适用于静息状态下PaO₂≤55mmHg或SaO₂≤88%的患者，低流量吸氧改善缺氧。

- **支气管哮喘**：规律使用控制类药物（吸入性糖皮质激素+长效支气管舒张剂，如沙美特罗替卡松、布地奈德福莫特罗），控制气道慢性炎症；轻度患者可单用吸入性糖皮质激素或白三烯调节剂（孟鲁司特）；避免接触过敏原，必要时进行过敏原特异性免疫治疗，减少急性发作风险。

- **支气管扩张症**：长期使用祛痰药，促进痰液排出；合并气流受限者使用支气管舒张剂；反复感染患者可在医生指导下进行长期口服抗生素预防感染，避免滥用；必要时进行体位引流，帮助分泌物排出。

- **间质性肺病**：根据病理类型选用药物，如糖皮质激素、免疫抑制剂（环磷酰胺、硫唑嘌呤），抑制肺间质纤维化进展；合并缺氧者给予长期氧疗，晚期患者可考虑肺移植治疗。

## 6.3 急性加重期治疗

- **抗感染治疗**：合并细菌感染时，经验性选用抗生素（青霉素类、头孢菌素类、呼吸氟喹诺酮类、大环内酯类），后续根据痰培养药敏结果调整；病毒感染时，流感病毒感染者早期使用奥司他韦，其他病毒感染以对症支持治疗为主；避免广谱抗生素滥用，预防多重耐药菌产生。

- **对症支持治疗**：喘息、气道痉挛者，使用速效支气管舒张剂（沙丁胺醇、异丙托溴铵）雾化吸入，重症患者联合静脉糖皮质激素；痰多者加强祛痰治疗，必要时雾化吸入祛痰药或吸痰；低氧血症患者给予鼻导管或面罩吸氧，维持血氧饱和度≥92%；呼吸衰竭患者及时给予无创通气，必要时气管插管、有创机械通气。

- **慢性肺源性心脏病急性加重期**：控制肺部感染（核心措施），改善呼吸功能，纠正缺氧和二氧化碳潴留；使用利尿剂、血管扩张剂减轻心脏负荷，纠正心力衰竭；避免使用强效洋地黄类药物，防止心律失常。

## 6.4 康复治疗与护理

- **呼吸功能锻炼**：腹式呼吸（吸气时腹部鼓起，呼气时腹部收缩，吸气:呼气=1:2-3）、缩唇呼吸（呼气时缩唇呈吹口哨状，缓慢呼气），每日练习2-3次，每次5-10分钟；配合呼吸操、八段锦，增强呼吸肌力量，改善肺功能及运动耐力。

- **体位引流与排痰**：支气管扩张症、痰多患者，根据病变部位采取合适体位，借助重力促进痰液排出，每日1-2次，每次15-20分钟；咳痰无力者可借助手动或电动排痰仪，震动胸壁帮助痰液松动。

- **心理与营养护理**：关注患者心理状态，焦虑、抑郁者及时给予心理疏导，必要时药物干预；保证营养摄入，给予高蛋白、高维生素、易消化食物，避免辛辣、油腻食物，增强机体抵抗力；戒烟限酒，避免诱发病情加重。

## 7.2 转诊指征

出现以下情况，需及时转诊至上级医院或ICU治疗：

- 稳定期治疗后病情仍进展，肺功能持续下降，活动后呼吸困难加重，影响生活自理能力；

- 急性发作期经常规治疗后症状无缓解，出现呼吸困难加重、口唇发绀、血氧饱和度持续低于92%，或意识模糊、嗜睡、烦躁不安；

- 合并严重并发症，如呼吸衰竭、心力衰竭、肺栓塞、肺脓肿、大咯血等；

- 疑似合并肺部肿瘤、肺结核等疾病，需进一步检查明确诊断；

- 长期用药后出现严重不良反应，或多重耐药菌感染，常规治疗无效；

- 间质性肺病、晚期慢阻肺患者，需评估肺移植手术指征。
