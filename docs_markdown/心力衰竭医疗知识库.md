# 心力衰竭医疗知识库

# 一、心力衰竭概述

心力衰竭（Heart Failure，HF）简称心衰，是各种心脏疾病进展至严重阶段的临床综合征，由心肌收缩力减弱、舒张功能障碍或负荷过重导致心脏泵血能力下降，无法满足机体代谢需求，进而引发肺循环淤血、体循环淤血及组织灌注不足的一系列症状和体征。作为心血管系统终末期疾病，心衰发病率、再住院率及病死率均较高，我国患病人数超1.3亿，且随人口老龄化、基础心血管病发病率上升呈持续增长趋势，严重影响患者生活质量，给医疗体系带来沉重负担。

心衰病程多为慢性进展性，早期可无明显症状，随病情加重逐渐出现呼吸困难、水肿、乏力等表现，急性发作时可危及生命。本病需结合病因、心功能分级制定个体化治疗方案，通过药物、器械、康复干预等综合手段，控制症状、延缓病程进展、降低急性发作风险，改善患者预后。

# 二、心力衰竭基础常识

## 2.1 疾病分型与核心特征

临床根据发病急缓、心功能受损类型、病因等对心衰进行分类，不同分型的临床表现、治疗重点差异显著，核心分型及特征如下：

|分型方式|疾病亚型|核心表现与特征|
|---|---|---|
|按发病急缓|急性心力衰竭（AHF）|突发呼吸困难、端坐呼吸、咳粉红色泡沫样痰，伴烦躁、大汗、血压异常，病情进展迅速，需紧急抢救|
||慢性心力衰竭（CHF）|病程迁延，表现为活动后气短、乏力、下肢水肿、食欲减退，症状反复，需长期慢病管理|
|按心功能受损类型|射血分数降低型心衰（HFrEF）|左室射血分数（LVEF）＜40%，核心为心肌收缩功能减退，常见于冠心病、心肌病|
||射血分数保留型心衰（HFpEF）|左室射血分数（LVEF）≥50%，核心为心肌舒张功能障碍，常见于高血压性心脏病、老年患者|
|按病变部位|左心衰竭、右心衰竭、全心衰竭|左心衰以肺淤血为主（呼吸困难），右心衰以体循环淤血为主（下肢水肿、腹水），全心衰为两者症状叠加|
## 2.2 流行病学与高危人群

- **高发人群**：65岁以上老年人，80岁以上人群患病率超10%；冠心病、高血压性心脏病、心肌病、心律失常、心脏瓣膜病患者；糖尿病、慢性肾病、肥胖、长期饮酒者；有心力衰竭家族史者；既往发生过心肌梗死、急性心衰的患者。

- **人群与地域特征**：我国北方心衰患病率高于南方，农村地区因医疗资源不足、基础病控制欠佳，重症率及病死率更高；男性患病率略高于女性，但女性HFpEF亚型占比更高，且随年龄增长性别差异逐渐缩小。

- **疾病负担**：心衰患者反复住院率高，5年生存率较低，仅相当于恶性肿瘤水平；长期用药、康复及住院费用高昂，不仅降低患者生活自理能力，还占用大量医疗资源，是心血管疾病领域的重要公共卫生问题。

# 三、心力衰竭病因与发病机制

## 3.1 核心病因

心力衰竭的发生是心脏结构或功能受损、负荷异常及神经体液调节紊乱共同作用的结果，核心病因可分为基础病因和诱发因素，具体如下：

- **基础病因（心脏自身病变）**：冠心病、心肌梗死是最常见病因，心肌缺血坏死导致收缩功能下降；高血压性心脏病长期负荷过重，引发心肌肥厚、舒张功能障碍；心肌病（扩张型、肥厚型）、心脏瓣膜病（狭窄、关闭不全）、先天性心脏病直接损伤心脏结构与功能；心律失常（房颤、室速）导致心脏泵血效率下降，长期可诱发心衰。

- **诱发因素（加重心衰发作）**：急性感染（尤其是肺部感染）是最常见诱因，增加心脏负荷；过度劳累、情绪剧烈波动、过量饮水或钠摄入过多，加重心脏负担；电解质紊乱、贫血、甲状腺功能亢进，增加心肌耗氧量或影响心肌功能；药物滥用（如非甾体抗炎药、某些抗心律失常药）、停药不当，干扰心功能调节；气温骤变、暴饮暴食等也可诱发急性发作。

- **全身性疾病影响**：糖尿病、慢性肾病可损伤血管及心肌，加速心衰进展；肥胖、代谢综合征通过引发血脂异常、高血压，间接增加心衰风险；长期饮酒、吸烟可直接损伤心肌，诱发酒精性心肌病，加重心衰。

## 3.2 核心发病机制

心力衰竭的核心发病机制是心脏泵血功能减退，伴随神经体液调节紊乱形成“损伤-代偿-进一步损伤”的恶性循环。当心脏受到缺血、负荷过重等损伤时，机体启动代偿机制：交感神经兴奋导致心率加快、心肌收缩力增强，肾素-血管紧张素-醛固酮系统（RAAS）激活引发水钠潴留、血管收缩，短期可维持泵血功能，但长期过度激活会加重心肌耗氧量、导致心肌重构（心肌肥厚、心腔扩大），进一步损害心功能。

心肌重构是心衰进展的关键病理过程，表现为心肌细胞肥大、凋亡，心肌纤维化，导致心脏结构异常、收缩/舒张功能持续下降。同时，心衰患者存在氧化应激、炎症反应增强，加重心肌损伤；血管内皮功能障碍、血栓形成风险升高，进一步诱发并发症，推动病程进展。不同亚型机制各有侧重，HFrEF以心肌收缩力减弱为主，HFpEF则以心室舒张受限、顺应性下降为核心。

# 四、心力衰竭预防与健康管理

## 4.1 一级预防（病因预防）

针对普通人群及高危群体，通过综合干预减少心脏损伤、控制危险因素，降低心衰发病风险，核心措施如下：

- **基础疾病控制**：积极治疗冠心病、高血压、糖尿病，严格控制血压（目标＜130/80mmHg）、血糖、血脂达标；定期监测心脏功能，冠心病患者尽早开通闭塞血管，高血压患者长期规律降压，避免心肌肥厚。

- **生活方式调理**：低盐饮食，每日盐摄入≤5克，避免隐形盐（加工肉类、咸菜），控制饮水量；戒烟限酒，彻底戒烟，避免过量饮酒，酒精性心肌病患者需严格戒酒；控制体重，BMI目标＜24，腰围男性＜90cm、女性＜85cm；适度运动，每周150分钟中等强度有氧运动（快走、太极拳），避免剧烈运动，高危人群需在医生指导下运动。

- **诱因预防与体检**：预防呼吸道感染，冬季注意保暖，流感季节接种疫苗；避免过度劳累、情绪激动，保证每日7-8小时睡眠；40岁以上人群每年进行心脏超声、心电图检查，高危人群每6个月复查，早期发现心功能异常。

## 4.2 二级预防（早筛查、早干预）

- **定期筛查**：心脏超声是评估心功能的核心检查，可检测左室射血分数、心室大小及心肌结构，明确是否存在早期心功能损伤；BNP（脑钠肽）或NT-proBNP检测可辅助筛查，数值升高提示心功能异常，适用于高危人群筛查及病情预警。

- **症状监测与早期干预**：警惕心衰早期信号，如活动后气短、乏力、夜间憋醒、下肢轻度水肿等，及时就医排查；确诊早期心衰患者，尽早启动药物治疗+生活方式干预，延缓心肌重构，阻止病情进展。

- **慢病管理体系**：医疗机构建立心衰患者分级管理档案，定期随访评估心功能、药物疗效及不良反应；社区卫生服务中心提供健康指导，帮助患者掌握饮食、用药及症状监测要点，提高治疗依从性。

## 4.3 三级预防（慢病管理与并发症防治）

- **规范用药与治疗**：患者严格遵医嘱规律服药，不可自行停药、减药，掌握利尿剂、RAAS抑制剂等药物的用法及不良反应（如低钾、低血压）；定期复查心功能、电解质、肾功能，根据病情调整治疗方案。

- **康复与生活管理**：在医生指导下进行心脏康复训练，从轻度活动逐步过渡到规律运动，改善心功能及运动耐量；严格控制钠摄入和饮水量，监测体重变化（每日晨起空腹称重，若短期内增重＞2kg提示水钠潴留）；避免感染、劳累、情绪波动等诱发因素，预防急性心衰发作。

- **并发症防治**：早期发现心律失常、血栓栓塞、肺部感染等并发症，房颤合并心衰患者需遵医嘱抗凝治疗，预防脑栓塞；长期卧床患者定期翻身、活动肢体，预防压疮及下肢静脉血栓；出现发热、咳嗽、呼吸困难加重等症状时，立即就医。

# 五、心力衰竭诊断方法

## 5.1 临床诊断（初步筛查）

结合患者病史、症状、体格检查及心功能分级标准进行初步判断，为后续辅助检查提供方向，适用于基层医疗机构首诊及社区筛查。

诊断要点：① 病史采集：重点询问基础心脏病史、心衰家族史、用药史、诱发因素，以及症状特点（呼吸困难、水肿、乏力的出现时间、加重/缓解方式）；② 体格检查：听诊肺部啰音（肺淤血表现）、心脏杂音，触诊肝脾肿大，检查下肢水肿、颈静脉怒张（体循环淤血表现），测量心率、血压，评估心功能状态。③ 心功能分级（NYHA分级）：Ⅰ级无症状，Ⅱ级活动轻度受限，Ⅲ级活动明显受限，Ⅳ级静息状态下也有症状。

## 5.2 实验室与影像学诊断（辅助确诊与病情评估）

|诊断方法|检测目标/用途|结果解读与适用场景|注意事项|
|---|---|---|---|
|BNP/NT-proBNP检测|评估心功能，辅助诊断及预后判断|数值升高提示心功能异常，升高程度与心衰严重程度正相关；治疗后数值下降提示病情改善，可用于监测疗效及急性发作预警。|需空腹采血，肾功能不全、老年患者数值可能假性升高，需结合临床症状判断。|
|心脏超声|评估心脏结构与功能，明确心衰亚型|可检测LVEF、心室大小、心肌肥厚、瓣膜功能，区分HFrEF与HFpEF，明确病因及病情严重程度，是心衰诊断核心检查。|检查前无需特殊准备，配合医生调整体位，女性避开月经期。|
|心电图|排查基础心脏病及心律失常|可发现心肌缺血、心肌梗死、心律失常等基础病因，辅助判断心衰诱因，适用于所有患者首诊筛查。|检查前避免剧烈运动、情绪激动，保持安静状态。|
|血生化检查|排查诱因，评估脏器功能|检测电解质、肾功能、肝功能、血糖、血脂，明确是否存在低钾、肾损伤等诱因，为治疗方案制定提供依据。|需空腹采血，采血前避免高脂饮食、饮酒，停用影响指标药物（遵医嘱）。|
# 六、心力衰竭治疗方案

## 6.1 治疗原则

心力衰竭治疗以“去除诱因、缓解症状、逆转心肌重构、控制心率血压、预防急性发作、改善预后”为核心。急性心力衰竭需紧急抢救，快速减轻心脏负荷、纠正缺氧、稳定血流动力学；慢性心力衰竭以药物治疗为基础，结合生活方式干预、器械治疗（必要时），实现长期病情控制；同时积极治疗基础心脏病，纠正电解质紊乱、感染等诱因，个体化调整方案，兼顾合并症治疗安全性。

## 6.2 药物治疗（核心干预手段）

药物治疗适用于所有慢性心衰患者，需根据心衰亚型、心功能分级及合并症选择，核心药物类别及应用如下：

|药物类别|代表药物|适用场景|不良反应与注意事项|
|---|---|---|---|
|利尿剂|呋塞米、螺内酯、托拉塞米|所有有水肿、淤血症状的心衰患者，缓解水钠潴留|呋塞米可能导致低钾、低血压；螺内酯可能引发高钾，用药期间定期监测电解质、肾功能。|
|RAAS抑制剂|缬沙坦、依那普利、沙库巴曲缬沙坦|慢性心衰患者基础用药，逆转心肌重构，降低病死率|可能出现低血压、肾功能异常、干咳（普利类），用药初期监测血压、肾功能，逐步调整剂量。|
|β受体阻滞剂|美托洛尔、比索洛尔、卡维地洛|慢性稳定型心衰，控制心率，改善心肌重构|可能导致心率减慢、乏力，不可突然停药，需逐步减量，急性心衰发作期禁用。|
|正性肌力药物|地高辛、多巴酚丁胺|地高辛用于慢性心衰合并房颤；多巴酚丁胺用于急性心衰或重症慢性心衰|地高辛治疗窗窄，过量易中毒（恶心、心律失常），需监测血药浓度；多巴酚丁胺需静脉输注，监测血压心率。|
## 6.3 器械与手术治疗（复杂病例干预）

- **器械治疗**：植入式心脏复律除颤器（ICD）适用于有猝死风险的重度心衰患者，可自动监测心律并电击除颤；心脏再同步化治疗（CRT）适用于心室收缩不同步的心衰患者，通过同步起搏改善心功能，缓解症状。

- **手术治疗**：针对基础病因治疗，如冠心病患者行冠脉支架植入术或搭桥手术，改善心肌供血；心脏瓣膜病患者行瓣膜置换/修复术，纠正瓣膜异常；终末期心衰患者可考虑心脏移植，是最终治疗手段。

## 6.4 急性心力衰竭急救

- **紧急处理**：立即让患者取端坐位，双下肢下垂，减少回心血量；高流量吸氧，必要时给予无创呼吸机辅助通气，纠正缺氧；监测心率、血压、血氧饱和度，建立静脉通路。

- **药物急救**：静脉输注利尿剂（呋塞米）快速利尿，减轻心脏负荷；静脉应用硝酸甘油、硝普钠等血管扩张剂，降低外周阻力；必要时使用正性肌力药物（多巴酚丁胺），改善心肌收缩力，稳定血流动力学。

- **诱因纠正与监护**：快速排查并纠正诱因（如感染、心律失常、高血压急症）；病情危重者转入ICU监护治疗，必要时进行主动脉内球囊反搏（IABP），辅助心脏泵血，挽救生命。

# 七、鉴别诊断与转诊指征

## 7.1 常见疾病鉴别诊断

|鉴别疾病组合|核心鉴别点|
|---|---|
|心力衰竭与肺部疾病（肺炎、慢阻肺急性发作）|心衰：呼吸困难与体位相关（端坐位缓解），伴下肢水肿、颈静脉怒张，BNP升高、心脏超声提示心功能异常；肺部疾病：伴发热、咳痰，呼吸困难与呼吸相关，胸部影像学可见肺部病变，BNP正常。|
|心力衰竭与肝硬化腹水|心衰：腹水伴下肢水肿、颈静脉怒张，呼吸困难明显，心脏超声异常；肝硬化：腹水伴脾大、黄疸、肝功能异常，无明显呼吸困难，BNP正常，有肝炎、饮酒史。|
|急性心衰与急性心肌梗死|急性心梗：突发胸痛，心肌酶、肌钙蛋白升高，心电图可见ST段改变；急性心衰：以呼吸困难为核心，可由心梗诱发，需通过心电图、心肌酶检查明确是否合并心梗。|
## 7.2 转诊指征

出现以下情况，需及时转诊至上级医院心血管内科或ICU治疗：

- 突发呼吸困难、端坐呼吸、咳粉红色泡沫样痰，疑似急性心力衰竭，常规治疗无法缓解；

- 慢性心衰患者症状加重，水肿、呼吸困难明显，药物调整后疗效不佳，需进一步评估心功能；

- 心功能分级Ⅳ级，静息状态下有症状，或合并严重心律失常、血栓栓塞等并发症；

- 需要器械治疗（ICD、CRT）或手术治疗（瓣膜置换、心脏移植），基层医疗机构无法开展；
