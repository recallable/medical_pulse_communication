# ⼄型流感（⼄流）医疗知识库

## ⼄流简介

⼄型流感（简称⼄流）是由⼄型流感病毒引起的急性呼吸道传染病，属于丙类传染病，其传染性和传播速度略低于甲型流感，但仍易在秋冬季节引发聚集性疫情，尤其多见于学校、托幼机构等集体场所。⼄流病毒不具备跨物种传播能力，仅在人群中流行，变异速度较慢，流行毒株相对稳定，主要分为Victoria系和Yamagata系两大亚型，且两大亚型可交替流行。⼄流临床症状与甲流相近，但全身症状稍缓，重症发生率低于甲流，不过老年人、婴幼儿、孕妇及基础疾病患者感染后仍需警惕重症风险。

与甲流相比，⼄流的季节性更集中，多数地区仅在冬季形成单一流行高峰，病程相对较短，多数轻症患者可在1周左右自愈，对人群健康的整体影响较甲流温和，但每年仍会造成一定的就诊量和误工、缺勤负担。

## ⼄流基础常识

### 2.1 病原学特征

⼄型流感病毒属于正黏病毒科流感病毒属，为单股负链RNA病毒，病毒颗粒呈球形，直径80-120nm，表面同样含血凝素（HA）和神经氨酸酶（NA）两种关键糖蛋白，但其基因序列与甲型流感病毒存在明显差异，无亚型划分的多样性，仅分为Victoria系和Yamagata系两个抗原分支。

⼄流病毒变异能力显著弱于甲流，仅发生缓慢的抗原漂移，无抗原转变现象，因此流行毒株相对稳定，疫苗交叉保护效果较好，无需每年频繁更新毒株类型，且不会引发全球性大流行，多以区域性、季节性流行为主。

### 2.2 流行病学特征

- **传染源**：主要为⼄流患者和无症状感染者，患者发病后1-7天具有传染性，发病后24-48小时传染性最强，无症状感染者排毒量较低，传播风险弱于显性感染者，但仍是重要的隐性传染源。
- **传播途径**：与甲流一致，以呼吸道飞沫传播为主要途径，患者咳嗽、打喷嚏产生的飞沫被他人吸入后引发感染；同时可通过接触被病毒污染的物品间接传播，密闭空间内气溶胶传播风险较低，仅在极端密集环境下可能发生。
- **易感人群**：人群普遍易感，其中5-15岁青少年儿童为高发人群，感染后症状相对明显；老年人、孕妇、免疫功能低下者及患有慢性心肺疾病、糖尿病等基础疾病的人群，感染后易发展为重症，需重点防护。

## ⼄流病因与发病机制



# 一、甲流简介

## 3.1 核心病因

乙流的直接病因是感染乙型流感病毒，病毒通过呼吸道侵入人体后，借助表面血凝素（HA）与呼吸道上皮细胞表面的唾液酸受体结合，进入细胞内完成复制，产生子代病毒后释放，进一步感染周围呼吸道上皮细胞，引发局部黏膜炎症，并通过血液循环扩散至全身，导致相应症状。

间接诱发因素包括：秋冬季节气温骤降，呼吸道黏膜屏障功能减弱；集体场所人群密集，增加病毒传播机会；个体过度劳累、受凉、营养不良导致免疫功能下降；未接种流感疫苗，缺乏特异性免疫力，易受病毒侵袭。

## 3.2 发病机制

乙流病毒侵入人体后，首先在鼻咽部、咽喉部的上皮细胞内复制，导致局部黏膜充血、水肿、分泌物增多，出现流涕、咽痛、咳嗽等上呼吸道症状；随着病毒复制扩散，进入血液循环引发全身性炎症反应，刺激机体释放白细胞介素、肿瘤坏死因子等炎症因子，导致发热、头痛、肌肉酸痛、乏力等全身症状。

乙流重症病例的发病机制与甲流类似，均与免疫反应过度激活引发的“细胞因子风暴”相关，但发生率更低。重症患者体内大量炎症因子损伤自身组织器官，可引发急性呼吸窘迫综合征、心肌损伤、感染性休克等并发症，严重时危及生命，多发生于高危人群中。

# 四、乙流预防措施

## 4.1 疫苗预防（最有效手段）

接种乙型流感疫苗是预防乙流最经济有效的措施，可刺激机体产生针对乙流病毒的特异性抗体，显著降低感染风险及感染后重症发生率，尤其对青少年儿童和高危人群保护效果明显。

- 接种人群：优先推荐5-15岁青少年儿童、65岁以上老年人、孕妇、免疫功能低下者、医务人员及慢性病患者；普通健康人群也可接种，提升群体免疫力，减少流行扩散。
- 接种时间：建议在每年流感流行季前（9-11月）完成接种，接种后2-4周产生有效抗体，保护期约6-8个月，因病毒存在抗原漂移，建议每年定期接种。
- 疫苗类型：目前常用的流感疫苗多为甲、乙流联合疫苗（三价或四价），其中四价疫苗同时覆盖乙流Victoria系和Yamagata系，防护范围更全面，可根据需求选择接种。

## 4.2 日常防护措施

- 呼吸道防护：流行季减少前往人群密集、通风不良的场所，必要时佩戴医用外科口罩；咳嗽、打喷嚏时用纸巾或肘部遮挡口鼻，避免飞沫传播，养成良好卫生习惯。
- 手部卫生：勤用流动水和洗手液（或肥皂）规范洗手，揉搓时间不少于20秒；接触公共物品后避免直接接触口、鼻、眼，无洗手条件时可使用含酒精（75%）的免洗洗手液。
- 环境与物品消毒：定期对居住、工作、学习场所通风换气，每日至少2次，每次不少于30分钟；对门把手、餐具、玩具、手机等高频接触物品，用含氯消毒剂或75%酒精擦拭消毒。



# 一、生活习惯调整

保证充足睡眠、均衡饮食、适度体育锻炼，增强机体免疫力；根据气温变化及时增减衣物，避免受凉，减少病毒感染机会。

## 4.3 暴露后预防

与乙流患者密切接触后，需及时进行居家隔离观察，观察期为7天，期间每日监测体温及呼吸道症状，避免接触他人。可在医生指导下，于接触后48小时内服用奥司他韦等抗病毒药物进行预防，降低感染概率，尤其适用于高危人群。

# 五、乙流诊断方法

## 5.1 临床诊断（初步筛查）

结合流行病学史、临床表现进行初步判断，适用于基层医疗机构快速识别，为后续诊疗提供参考。

🔍 诊断要点：
1. 流行病学史：发病前7天内接触过乙流患者，或处于乙流流行区域、集体单位聚集性发病环境；
2. 临床表现：突发低热至中度发热（部分可达39℃）、头痛、肌肉酸痛、乏力、咳嗽、咽痛，少数患者伴有恶心、呕吐、腹泻，症状整体较甲流缓和。

## 5.2 实验室诊断（确诊依据）

| 诊断方法         | 检测目标               | 优势                                   | 局限性                     |
|------------------|------------------------|----------------------------------------|----------------------------|
| 核酸检测（RT-PCR） | 乙流病毒RNA            | 特异性高（95%以上）、灵敏度高，可区分乙流Victoria系和Yamagata系 | 需专业设备和技术，1-4小时，基层医疗度有限 |
| 抗原检测         | 乙流病毒抗原           | 操作简便、检测快速（15-30分钟），可居家或基层快速筛查 | 灵敏度较低，发病病毒载量低时易出现问题，无法区分亚型 |
| 血清学检测       | 乙流特异性抗体（IgM、IgG） | 可判断近期感染（IgM阳性）或既往感染（IgG阳性），用于流行病学调查 | 抗体产生需1-2周，早期诊断，不能区分既往感染 |

# 六、甲型H1N1流感的治疗

## 6.1 对症支持治疗

- **隔离休息**：居家或住院隔离，避免传播他人，保证充足休息，减少体力消耗，促进机体恢复，隔离至体温正常后24小时且症状明显缓解。
- **对症处理**：发热、头痛、肌肉酸痛者，可服用对乙酰氨基酚、布洛芬等非甾体抗炎药（儿童禁用阿司匹林，以防瑞氏综合征）；咳嗽、咳痰者，给予氨溴索、对乙酰半胱氨酸等止咳祛痰药物；咽痛者可使用含漱液、润喉片缓解症状。
- **补液支持**：鼓励多饮水，多摄入清淡易消化的食物，维持水电解质平衡；呕吐、腹泻严重者，给予口服补液盐，必要时静脉补液，避免脱水。

## 6.2 抗病毒治疗（核心治疗）

甲流抗病毒治疗药物与甲流通用，早期使用可缩短病程、减轻症状、降低重症风险，建议在发病48小时内启动治疗，具体药物如下：

| 药物名称   | 作用机制                       | 用法要点                                   | 注意事项                       |
|------------|--------------------------------|--------------------------------------------|--------------------------------|
| 奥司他韦   | 抑制神经氨酸酶，阻止病毒从感染细胞释放，阻断病毒传播 | 口服，成人每次75mg，每日2次，疗程5天；儿童根据体重调整剂量，发病48小时内服用效果最佳 | 不良反应轻微，偶吐，饭后服用可减激，肝肾功能不全量 |
| 扎那米韦   | 作用于神经氨酸酶，抑制病毒释放，发挥抗病毒作用 | 吸入给药，成人及12岁以上儿童，每次10mg，每日2次，疗程5天 | 不适用于哮喘、慢性疾病患者，可能诱发，吸入后需观察 |
| 帕拉米韦   | 新型神经氨酸酶抑制剂，抗病毒作用强效、持久 | 静脉滴注，成人单次300mg，重症患者可间隔12-24小时重复给药1次，疗程根据病情调整 | 仅用于重症病例或无口服的患者，需在医生用，监测肝肾功能 |

## 6.3 重症病例治疗

甲流重症病例少见，多发生于高危人群，一旦出现呼吸困难、低氧血症、意识障碍等症状，需立即住院治疗，采取综合救治措施：

- **呼吸支持**：给予吸氧、无创通气，病情进展者及时行气管插管、有创机械通气，严重呼吸衰竭者可采用体外膜肺氧合（ECMO）支持治疗，维持呼吸功能。
- **抗炎治疗**：针对“细胞因子风暴”，在医生指导下合理使用糖皮质激素（如甲泼尼龙），避免滥用导致感染加重、骨质疏松等不良反应。
- **并发症治疗**：积极治疗肺部继发感染、心肌损伤、肾功能衰竭等并发症，加强营养支持，保护重要器官功能，降低病死率。

# 七、甲流临床特征与诊疗路径



# 7.1 临床分型与表现

- **轻症型**：占绝大多数，病程5-7天，主要表现为低热至中度发热（37.5-39℃）、头痛、肌肉酸痛、乏力、咳嗽、咽痛、流涕，无明显呼吸困难，症状持续3-5天后逐渐缓解，预后良好，可自愈或经对症治疗康复。
- **重症型**：发病后病情进展较快，出现持续高热（≥39℃，持续3天以上）、呼吸困难、胸闷、喘憋、口唇发绀，可伴有意识模糊、嗜睡、烦躁不安、呕吐、腹泻等症状，严重者引发急性呼吸窘迫综合征、感染性休克、多器官功能衰竭，需紧急救治。
- **特殊人群表现**：青少年儿童感染后发热症状较明显，易出现高热惊厥；老年人感染后全身症状重，合并基础疾病者易发生重症，症状不典型者可能仅表现为低热、乏力；孕妇感染后可能影响胎儿发育，增加早产风险，自身重症率高于普通人群。

# 7.2 临床诊疗路径

乙流临床诊疗遵循“筛查-诊断-分型治疗-随访”标准化路径，确保诊疗规范、高效，具体流程如下：

1. **初步筛查**：基层医疗机构结合流行病学史、临床表现，对发热患者进行乙流抗原检测，快速区分乙流与普通感冒、甲流、新冠病毒感染等。

2. **确诊与分型**：抗原检测阳性者可初步诊断，核酸检测阳性为确诊依据，结合症状、体征及血常规、胸片/CT、血氧饱和度检查，判断轻症或重症。

3. **分层治疗**：轻症患者居家隔离，给予对症治疗，必要时早期服用抗病毒药物；重症患者立即转至定点医院，启动重症救治流程，加强器官功能支持。

4. **随访与康复**：轻症患者治疗后3-5天随访，监测症状消退及体温恢复情况；重症患者康复后定期复查肺功能、心功能，避免后遗症，同时指导康复训练。

# 7.3 临床鉴别诊断

乙流需与普通感冒、甲型流感、新冠病毒感染、支原体肺炎等疾病相鉴别，核心鉴别要点如下：
