# 肾脏疾病医疗知识库

# 一、肾脏疾病概述

肾脏疾病是一组累及肾脏实质、肾小管、肾血管及肾间质的疾病总称，涵盖感染性、免疫性、代谢性、遗传性、梗阻性等多种类型。其核心病理改变为肾脏结构损伤、滤过与排泄功能异常，临床常表现为蛋白尿、血尿、水肿、高血压、肾功能减退等，晚期可进展为尿毒症，需依赖肾脏替代治疗维持生命，严重威胁患者健康。

该类疾病发病隐匿，早期症状不典型，易被忽视，且病程多呈慢性进展，需长期监测与管理。不同年龄段发病特点各异，如急性肾小球肾炎多见于儿童青少年，慢性肾脏病以中老年人群高发，多囊肾等遗传性疾病则多在青中年期显现症状。病因与感染、免疫紊乱、代谢异常、遗传、药物毒性及尿路梗阻等密切相关，早期干预并保护肾功能，可显著改善预后，降低疾病负担。

# 二、肾脏疾病基础常识

## 2.1 疾病分型与核心特征

根据发病急缓、病理性质及病因，肾脏疾病可分为常见亚型，各亚型核心特征如下：

|分型类别|典型疾病|核心病因|核心临床特征|
|---|---|---|---|
|感染性肾脏疾病|急性肾盂肾炎、慢性肾盂肾炎|病原体上行感染（大肠埃希菌为主）、血行感染，慢性者多伴尿路梗阻|急性者伴发热、腰痛、肾叩击痛、尿路刺激征；慢性者病程迁延，可伴肾功能渐进性减退|
|免疫性肾脏疾病|急性肾小球肾炎、慢性肾小球肾炎、肾病综合征、狼疮性肾炎|感染后免疫反应、自身免疫紊乱、免疫复合物沉积|蛋白尿、血尿、水肿、高血压为共性表现，肾病综合征伴大量蛋白尿（>3.5g/d）、低蛋白血症|
|代谢性肾脏疾病|糖尿病肾病、高血压肾损害、尿酸肾病|血糖/血压长期控制不佳、高尿酸血症，损伤肾血管及肾小管|早期伴微量白蛋白尿，渐进性出现肾功能减退，晚期进展为尿毒症|
|遗传性与梗阻性肾脏疾病|多囊肾、遗传性肾炎、肾结石、肾积水|遗传变异、尿液代谢异常、尿路梗阻压迫肾实质|多囊肾伴双侧肾脏多发囊肿；肾结石突发肾绞痛、血尿；肾积水伴腰腹部胀痛，长期可致肾衰|
## 2.2 流行病学与高危人群

- **高发人群**：糖尿病、高血压患者为核心高危群体，病程超过10年者肾脏受累风险达30%-50%；有肾脏疾病家族史者，尤其是多囊肾、遗传性肾炎家族史，患病概率显著升高；长期服用肾毒性药物（非甾体抗炎药、造影剂）、接触重金属者，易引发药物性/中毒性肾病；反复尿路感染、尿路梗阻患者，可进展为慢性肾盂肾炎、梗阻性肾病；老年人因肾脏退行性变，肾功能储备下降，患病风险随年龄增长递增。

- **人群与地域特征**：慢性肾脏病全球患病率约13%，我国患病率达10.8%，且城乡差异不显著；糖尿病肾病在发达国家及我国经济发达地区发病率更高，与饮食结构西化、代谢病高发相关；多囊肾发病率无地域差异，仅与遗传背景相关；肾结石发病率与气候相关，干旱炎热地区因尿量减少，患病率显著高于湿润地区。

- **疾病负担**：慢性肾脏病病程长，中晚期患者需长期用药、定期监测，治疗成本较高；进展至尿毒症后，依赖血液透析、腹膜透析或肾移植维持生命，透析患者年治疗费用高昂，且生活质量严重下降；肾移植受供体短缺、术后排斥反应等因素限制，多数患者需长期透析，给家庭和社会带来沉重医疗负担。

# 三、肾脏疾病病因与发病机制

## 3.1 核心病因

肾脏疾病病因复杂，多为单一因素或多因素协同作用导致，核心病因可分为以下几类：

- **感染因素**：病原体感染是重要诱因，大肠埃希菌等细菌经尿道上行至肾脏，引发肾盂肾炎，长期反复感染可导致肾实质纤维化；链球菌等感染后，机体产生免疫复合物，诱发急性肾小球肾炎，属于感染后免疫介导性损伤；病毒感染（如乙肝病毒、丙肝病毒）可引发病毒相关性肾炎，损伤肾脏功能。

- **免疫与炎症因素**：自身免疫功能紊乱时，机体产生抗肾组织抗体，攻击肾小球、肾小管等固有细胞，导致免疫复合物沉积于肾脏，激活补体系统，引发慢性肾小球肾炎、狼疮性肾炎等；肾脏局部炎症长期不愈，可促进肾间质纤维化、肾小球硬化，加速肾功能衰退。

- **代谢与血管因素**：糖尿病患者血糖控制不佳，可损伤肾脏微血管及肾小管上皮细胞，引发糖尿病肾病；高血压长期高压状态可导致肾小动脉硬化、管腔狭窄，肾脏灌注不足，逐步发展为高血压肾损害；高尿酸血症时，尿酸结晶沉积于肾小管，引发尿酸肾病，严重者可致肾小管坏死。

- **遗传与环境因素**：多囊肾由PKD1、PKD2基因变异导致，呈常染色体显性遗传；遗传性肾炎与COL4A3、COL4A4等基因相关，表现为血尿、进行性肾功能减退；长期吸烟、饮酒可加重肾脏血管损伤，降低肾功能储备；饮水不足、长期憋尿可诱发肾结石，进而导致梗阻性肾损伤。

- **其他诱因**：药物与毒物因素，如长期服用布洛芬、萘普生等非甾体抗炎药，或使用含马兜铃酸的中药，可直接损伤肾小管；造影剂可引发造影剂肾病，尤其在肾功能不全患者中风险更高；尿路梗阻（结石、肿瘤、先天性畸形）可导致尿液淤积，肾实质受压缺血，逐步发展为梗阻性肾病；外伤可直接破坏肾脏结构，引发出血、肾挫伤，严重者致急性肾损伤。

## 3.2 核心发病机制

肾脏疾病核心发病机制围绕肾脏结构损伤、功能紊乱及病理代偿展开，不同亚型机制各有侧重，共同病理过程如下：

免疫介导性损伤：感染后或自身免疫性疾病中，免疫复合物沉积于肾小球基底膜或系膜区，激活补体系统，招募炎性细胞浸润，损伤肾小球滤过屏障，导致蛋白尿、血尿等症状。长期免疫损伤可引发肾小球硬化、肾小管萎缩，逐步丧失滤过与排泄功能，进展为慢性肾脏病。

肾血管与灌注异常：高血压、糖尿病等导致肾小动脉硬化、管腔狭窄，肾脏灌注不足，肾小球内压力升高，引发肾小球高滤过、高灌注状态，长期可导致肾小球硬化；肾动脉狭窄、血栓形成等可引发肾脏缺血缺氧，急性者致急性肾损伤，慢性者逐步进展为肾功能衰竭。

肾小管损伤与间质纤维化：药物、毒物、感染等因素可直接损伤肾小管上皮细胞，导致肾小管重吸收、排泄功能异常，出现尿糖、尿蛋白、电解质紊乱等；肾小管损伤后引发局部炎症反应，促进成纤维细胞增殖，肾间质纤维化，进而压迫肾小球及肾血管，加速肾功能衰退，这是多数慢性肾脏病进展的共同路径。

梗阻性损伤机制：尿路梗阻导致尿液排出受阻，尿液淤积使肾盂、肾盏压力升高，反向传导至肾小球，导致肾小球滤过功能下降；同时淤积尿液滋生细菌，诱发感染，感染与梗阻相互加重，导致肾实质受压、缺血、纤维化，逐步发展为梗阻性肾病，最终致肾功能丧失。

# 四、肾脏疾病预防与健康管理

## 4.1 一级预防（病因预防）

针对普通人群及高危群体，通过综合干预减少致病因素，降低肾脏疾病发病风险，核心措施如下：

- **危险因素管控**：严格控制糖尿病、高血压，糖尿病患者空腹血糖控制在4.4-7.0mmol/L，高血压患者目标血压＜130/80mmHg，规律服药并定期监测，避免肾血管损伤；避免长期使用肾毒性药物，服用非甾体抗炎药、抗生素、造影剂等需遵医嘱，用药前评估肾功能；远离重金属、化学毒物，做好职业防护；积极治疗尿路感染、尿路梗阻，避免病情迁延损伤肾脏。

- **生活方式干预**：保证每日饮水量1500-2000ml，促进代谢废物排出，稀释尿液中有害物质浓度，预防肾结石及感染；养成规律排尿习惯，避免长期憋尿；饮食均衡，低盐（每日＜5g）、低脂、优质蛋白饮食，糖尿病患者严格控糖，高尿酸患者限制高嘌呤食物（动物内脏、海鲜）摄入；戒烟限酒，减少烟草有害物质及酒精对肾脏血管的刺激；适度运动，每周150分钟中等强度有氧运动，控制体重，避免肥胖加重代谢负担。

- **高危人群防控**：有肾脏疾病家族史者，定期进行尿常规、肾功能及泌尿系超声筛查，多囊肾患者需早期监测囊肿大小及肾功能；糖尿病、高血压患者，每6个月复查尿微量白蛋白、肾功能，早期发现肾脏受累；长期用药人群，定期监测肾功能，及时调整用药方案；反复尿路感染患者，明确诱因并针对性治疗，避免转为慢性肾盂肾炎。

## 4.2 二级预防（早筛查、早干预）

- **定期筛查**：普通人群每年进行尿常规、肾功能（血肌酐、尿素氮、尿酸）及泌尿系超声检查；糖尿病、高血压患者每3-6个月复查尿微量白蛋白定量、肾功能，每年行泌尿系超声；高危人群（家族史、长期用药者）每6个月全面筛查，必要时加做肾脏ECT评估肾小球滤过率。

- **早期信号识别与干预**：警惕肾脏疾病早期信号，包括泡沫尿、血尿、眼睑及下肢水肿、不明原因高血压、夜尿增多、乏力、食欲减退等；发现异常及时就医，明确诊断后尽早干预，如早期糖尿病肾病使用RAAS抑制剂减少蛋白尿，急性肾小球肾炎及时抗感染、对症治疗，避免病情进展为慢性。

- **慢病管理前置**：确诊早期慢性肾脏病患者，遵医嘱规律用药，控制蛋白尿、血压血糖，延缓肾功能进展；肾结石患者治愈后，坚持多喝水、调整饮食，预防复发；药物性肾病患者立即停用肾毒性药物，改用替代药物，同时进行保肾治疗。

## 4.3 三级预防（慢病管理与并发症防治）

- **规范治疗与用药**：中晚期慢性肾脏病患者，严格遵循个体化治疗方案，控制血压血糖、减少蛋白尿，必要时使用激素、免疫抑制剂（针对免疫性肾病）；避免使用肾毒性药物，用药剂量根据肾功能调整；尿毒症患者及时启动肾脏替代治疗（血液透析、腹膜透析），符合条件者评估肾移植指征。定期复查肾功能、电解质、血常规，调整治疗方案，避免并发症。

- **长期康复与护理**：透析患者规律治疗，血液透析每周3次，腹膜透析每日规范操作，注意无菌防护，避免感染；饮食严格遵循低盐、低钾、低磷、优质低蛋白原则，预防贫血、肾性骨病等并发症；慢性肾脏病患者避免劳累、受凉，预防感染（感冒、尿路感染），减少肾功能急性加重诱因；术后患者（肾移植、结石手术）做好伤口护理，定期复查，遵医嘱用药及康复训练。

- **并发症防治**：慢性肾脏病患者定期监测电解质，预防高钾血症、代谢性酸中毒；补充促红细胞生成素及铁剂，预防肾性贫血；补充维生素D及钙剂，预防肾性骨病；透析患者做好血管通路护理，预防通路感染、血栓形成；肾移植患者长期服用免疫抑制剂，监测药物浓度，预防排斥反应及感染。

# 五、肾脏疾病诊断方法

## 5.1 临床诊断（初步筛查）

结合患者病史、症状、体格检查及发病特点进行初步判断，区分肾脏疾病亚型，为进一步检查提供方向，适用于首诊及社区筛查。

诊断要点：① 病史采集：重点询问既往病史（糖尿病、高血压、感染史）、用药史、家族史、毒物接触史，以及症状起病时间、进展速度、伴随症状；② 体格检查：包括血压测量、水肿评估、肾脏叩击痛、心肺查体，初步判断病变严重程度及是否合并并发症。

## 5.2 实验室与特殊检查（辅助确诊与病情评估）

|诊断方法|检测目标/用途|结果解读与诊断价值|注意事项|
|---|---|---|---|
|尿常规|筛查蛋白尿、血尿、管型尿、白细胞尿|蛋白尿提示肾小球损伤，血尿可见于肾炎、结石，管型尿提示肾小管/肾小球病变，白细胞尿提示感染|留取中段尿，避免污染；女性避开月经期，保证标本准确性|
|肾功能检查|评估肾小球滤过功能、肾小管功能|血肌酐、尿素氮、尿酸升高提示肾功能减退；内生肌酐清除率（Ccr）可量化肾功能损伤程度，指导治疗|需空腹采血；肌肉量多、剧烈运动后可能影响血肌酐结果，需综合判断|
|尿微量白蛋白定量|早期发现肾小球损伤，尤其糖尿病、高血压肾病|尿微量白蛋白30-300mg/24h为微量白蛋白尿，提示早期肾损伤；>300mg/24h为临床蛋白尿|留取24小时尿液，避免剧烈运动、感染，严格遵医嘱准备|
|泌尿系超声|评估肾脏大小、形态、结构，筛查结石、囊肿、积水、占位|慢性肾脏病可见肾脏缩小、皮质变薄；多囊肾可见双侧多发囊肿；结石、积水可明确梗阻部位|无创便捷，检查前需憋尿；对早期微小病变、免疫性肾病病理诊断价值有限|
|肾穿刺活检|明确肾脏病理类型，指导免疫性肾病治疗|可区分肾小球肾炎、肾病综合征亚型，明确病理损伤程度，为激素、免疫抑制剂使用提供依据|有创检查，术后卧床24小时，监测血压及尿色，预防出血、感染等并发症|
# 六、肾脏疾病治疗方案

## 6.1 治疗原则

肾脏疾病治疗遵循“病因治疗、对症施策、保护肾功能、预防并发症”原则。急性肾脏疾病以去除诱因、快速控制病情、逆转肾功能损伤为核心；慢性肾脏疾病以控制进展、减少蛋白尿、控制血压血糖、延缓尿毒症为目标；免疫性肾病需结合病理类型制定个体化方案，合理使用激素及免疫抑制剂；终末期肾病以肾脏替代治疗为主，提高患者生存质量。

## 6.2 药物治疗（核心干预手段）

药物治疗以病因治疗、对症支持、保护肾功能为主，不同亚型用药重点不同，核心药物类别及应用如下：

|药物类别|代表药物|适用场景|用法要点与不良反应|

## 6.3 手术与肾脏替代治疗

- **手术治疗**：肾结石直径＞0.6cm者，可行体外冲击波碎石、输尿管镜取石术、经皮肾镜取石术，解除梗阻保护肾功能；多囊肾囊肿较大、压迫肾实质者，可行囊肿去顶减压术，缓解压迫；尿路梗阻（肿瘤、畸形）引发的梗阻性肾病，需手术解除梗阻，如肿瘤切除术、畸形矫正术；肾移植为终末期肾病根治性治疗手段，适用于符合移植指征、找到合适供体者。

- **肾脏替代治疗**：尿毒症患者需长期替代治疗，血液透析通过机器替代肾脏排泄功能，每周3次，每次4小时；腹膜透析利用自身腹膜作为透析膜，每日居家操作，灵活性较高；两种透析方式均需配合饮食管理，预防并发症，延长患者生存期。肾移植术后需长期服用免疫抑制剂，预防排斥反应，提高移植肾存活率。

- **康复与饮食管理**：术后患者卧床休息，逐步进行康复训练，避免剧烈运动；慢性肾脏病患者严格遵循低盐、低钾、低磷、优质低蛋白饮食，控制液体摄入量；透析患者保证营养摄入，补充优质蛋白，预防营养不良；同时给予心理支持，缓解患者焦虑、抑郁情绪，提高治疗依从性。

## 7.2 转诊指征

出现以下情况，需及时转诊至上级医院肾内科、泌尿外科治疗：

- 疑似免疫性肾脏疾病（如肾病综合征、狼疮性肾炎），需行肾穿刺活检明确病理类型，指导激素及免疫抑制剂使用；

- 慢性肾脏病进展至肾功能不全中期以上，或出现高钾血症、代谢性酸中毒等并发症，需制定个体化治疗方案；

- 急性肾损伤（如药物性、缺血性），肾功能快速恶化，需紧急查找诱因并干预，避免进展为尿毒症；

- 肾结石直径＞1cm、出现肾积水或感染，体外碎石无效，需手术治疗（经皮肾镜、输尿管镜）；

- 终末期肾病，需评估肾脏替代治疗（透析、肾移植）指征，制定长期治疗方案；

- 药物治疗效果不佳，或出现严重药物不良反应（如激素导致的严重感染、免疫抑制剂导致的骨髓抑制）。
