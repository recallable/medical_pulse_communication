# 儿童发热家庭护理手册

# 一、手册概述

本手册专为0-14岁儿童家长定制，聚焦发热这一儿科最常见症状，系统梳理发热的基础认知、家庭护理方法、用药指南、危险信号及误区规避。手册遵循“科学护理、安全优先、精准判断”原则，以通俗语言拆解专业知识，提供可落地的实操步骤，帮助家长理性应对儿童发热，避免盲目用药与过度医疗，同时明确就医指征，保障儿童健康安全。

发热是儿童免疫系统对抗感染、异物入侵的正常生理反应，并非独立疾病。家庭护理的核心是缓解不适、监测病情，而非单纯追求“快速降温”，需结合儿童年龄、精神状态综合判断，实现“护理+监测+就医”的有序衔接。

# 二、发热基础认知

## 2.1 发热判断标准

儿童正常体温因测量部位不同存在差异，以腋温为常用参考标准，具体划分如下：低热（37.3-38℃）、中度发热（38.1-39℃）、高热（39.1-41℃）、超高热（≥41.1℃）。需注意，儿童体温波动较成人明显，哭闹、活动、进食后体温可能暂时升高0.3-0.5℃，属于生理性波动，休息30分钟后复测可恢复正常。

测量体温时需选择合适工具与方法：0-3岁婴幼儿推荐腋温测量（安全无创），学龄期儿童可选择腋温或口温（需避免咬碎体温计），禁止给婴幼儿测量肛温（易损伤肠道黏膜）；测量前需将体温计归零，确保测量时间足够（腋温5-10分钟），排除环境温度干扰。

## 2.2 发热常见原因

- **感染性因素**：占儿童发热的90%以上，以病毒感染为主，如急性上呼吸道感染、手足口病、疱疹性咽峡炎、流感等；细菌感染如化脓性扁桃体炎、细菌性肺炎、腹泻病等，多在病毒感染基础上继发；其他病原体感染如支原体、衣原体，多见于学龄期儿童，常伴刺激性干咳。

- **非感染性因素**：少见，包括接种疫苗后不良反应（接种后24-48小时内出现低热，持续1-2天可自愈）、穿盖过多导致的“捂热综合征”、出牙期低热、免疫性疾病及肿瘤等（多表现为长期低热或反复发热）。

- **年龄相关特点**：婴幼儿因免疫系统尚未成熟，易因病毒感染引发高热，且可能伴随高热惊厥；学龄期儿童发热多与呼吸道感染、支原体感染相关，症状相对典型，精神状态多较好。

# 三、家庭护理核心方法

## 3.1 物理降温（低热首选）

物理降温适用于低热（<38.5℃）、精神状态良好的儿童，核心是通过散热降低体温，避免使用刺激性方式，具体操作如下：

- **温水擦拭**：准备32-34℃温水，用柔软毛巾擦拭儿童颈部、腋下、腹股沟、大腿根部等大血管集中部位，每次擦拭10-15分钟，每1-2小时重复1次；避免擦拭胸前、腹部、足底（易引发不适或受凉），禁止使用酒精擦浴（儿童皮肤娇嫩，酒精易吸收导致中毒）。

- **环境与穿着调整**：保持室内温度22-26℃，定时开窗通风（每次30分钟，避免直吹儿童）；给儿童穿宽松、透气的棉质衣物，利于散热，避免穿盖过多（“捂汗”会导致体温升高，增加捂热综合征风险）。

- **补充水分**：发热时代谢加快，水分流失多，需少量多次饮用温水、口服补液盐或清淡汤羹（如米汤、面汤），避免脱水；婴幼儿保证奶量摄入，拒奶时可少量多次喂水，防止因脱水导致体温持续升高。

## 3.2 生活与饮食护理

- **休息保障**：保证儿童充足睡眠，减少活动量，避免劳累，利于免疫系统对抗病原体；精神状态良好时可适当活动，无需强制卧床。

- **饮食调理**：给予清淡、易消化、营养丰富的食物，如粥、烂面条、蒸蛋羹等，避免辛辣、油腻、生冷食物及甜食（加重肠胃负担）；食欲不佳时不强迫进食，待体温恢复后食欲会逐渐好转。

- **口腔与皮肤护理**：发热时口腔黏膜干燥，可少量多次喂水清洁口腔，年长儿可用淡盐水漱口；保持皮肤清洁，出汗后及时更换衣物，避免受凉加重病情。

# 四、退热用药指南

## 4.1 用药原则

儿童退热用药需严格遵循“按需使用、按龄给药、避免滥用”原则，仅在高热（≥38.5℃）或低热但精神状态差、伴随明显不适时使用，优先选择儿童专用剂型，禁止使用成人药物及复方感冒药滥用。

用药警示：① 避免同时使用两种及以上退热药物（易导致肝肾损伤）；② 不擅自增减剂量，严格按体重计算用量；③ 退热药物仅缓解症状，不能治疗病因，体温反复是正常现象，无需频繁用药。

## 4.2 常用退热药物及用法

|药物名称|适用年龄|用法用量（参考）|注意事项|
|---|---|---|---|
|对乙酰氨基酚|3个月以上儿童|每次15mg/kg，间隔4-6小时重复，24小时不超过4次；混悬液、分散片按需选择剂型。|不良反应轻微，偶有胃肠道不适，肝肾功能不全者慎用。|
|布洛芬|6个月以上儿童|每次10mg/kg，间隔6-8小时重复，24小时不超过4次；避免空腹服用。|可能引发胃肠道不适，有胃溃疡病史者慎用；脱水儿童禁用。|
## 4.3 特殊人群用药

- **婴幼儿（&lt;6个月）**：3个月以下婴儿发热，首选物理降温和就医排查，不擅自用药；3-6个月婴儿如需用药，优先选择对乙酰氨基酚，严格按体重计算剂量。

- **有基础疾病儿童**：肝肾功能不全、心脏病、癫痫病史儿童，用药前需咨询医生，避免药物不良反应加重病情。

- **接种疫苗后发热**：低热无需用药，物理降温即可；高热可按常规使用退热药物，不影响疫苗效果。

# 五、病情监测与就医指征

## 5.1 日常监测要点

- **体温监测**：低热每4小时测量1次体温，高热每1-2小时测量1次，记录体温变化趋势（如发热高峰时间、间隔时间），为就医提供参考。

- **精神状态评估**：精神状态是判断病情轻重的核心，若儿童发热时哭闹但易安抚、能正常进食饮水，提示病情较轻；若出现精神萎靡、嗜睡、烦躁不安、眼神呆滞，需警惕重症。

- **伴随症状观察**：密切关注是否出现咳嗽喘息、呼吸困难、口唇发绀、呕吐腹泻、皮疹、抽搐、意识模糊等症状，记录症状出现时间及特点，就医时告知医生。

## 5.2 必须就医的危险信号

出现以下情况，无论体温高低，均需立即就医，避免延误重症疾病诊治：

- **年龄因素**：3个月以下婴儿发热（腋温≥37.3℃），新生儿发热需紧急就医排查败血症。

- **体温与症状**：高热持续超过24小时无缓解，或反复发热超过3天；超高热（≥41℃）；发热伴随抽搐、意识模糊、精神萎靡。

- **呼吸与循环**：呼吸急促（婴幼儿呼吸≥50次/分钟，学龄儿童≥30次/分钟）、呼吸困难、口唇发绀、四肢冰凉、皮肤花纹。

- **其他异常**：呕吐频繁、无法进食饮水、尿量显著减少（脱水迹象）；皮肤出现皮疹、瘀斑；剧烈头痛、颈部僵硬；接种疫苗后发热超过48小时不缓解。

# 六、常见护理误区与正确做法

|常见误区|正确做法|
|---|---|
|发热即“捂汗”，追求快速降温|穿宽松衣物、保持环境通风，以缓解不适为目标，而非强制降温；捂汗易导致体温升高，引发捂热综合征。|
|用酒精擦浴、冰水降温|禁止酒精擦浴（儿童易酒精中毒）；冰水降温易引发寒战，导致体温反弹，仅可用32-34℃温水擦拭。|
|同时服用多种退热药物或复方感冒药|仅选用一种退热药物，按剂量使用；复方感冒药多含退热成分，易与单一退热药物叠加导致过量，不建议儿童常规使用。|
|发热时禁食禁水，避免“加重病情”|发热时需少量多次补充水分及营养，防止脱水；食欲不佳时不强迫进食，选择清淡易消化食物即可。|
|低热也必须用药，避免“烧出肺炎”|低热是免疫系统正常反应，无需用药，物理降温即可；肺炎并非“烧出来的”，是病原体感染导致，需通过检查确诊后治疗。|
# 七、特殊情况护理

## 7.1 高热惊厥护理

高热惊厥多见于6个月-5岁儿童，发作时需立即采取急救措施，避免窒息，同时及时就医：

- 立即将儿童侧卧，解开衣领、腰带，保持呼吸道通畅，避免呕吐物堵塞气道；

- 不要强行按压肢体（防止骨折），不要往口中塞任何物品（避免咬伤）；

- 惊厥持续超过5分钟、反复惊厥或发作后精神萎靡，需立即拨打急救电话。

## 7.2 脱水护理

发热伴随呕吐、腹泻时易发生脱水，需及时补液，出现以下脱水迹象需就医：

- 轻度脱水：尿量减少、口唇干燥、哭时泪少，可口服补液盐Ⅲ少量多次饮用；

- 中度至重度脱水：无尿、皮肤弹性差、眼窝凹陷、精神萎靡，需立即静脉补液治疗。

## 7.3 反复发热护理

反复发热多为病原体未清除所致，护理时需：① 按医嘱规律用药，不擅自停药；② 加强营养与休息，增强免疫力；③ 记录发热规律，就医时提供给医生，帮助明确病因（如支原体感染、EB病毒感染等）。
