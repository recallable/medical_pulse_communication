# 儿科常见病诊疗指南

# 一、指南概述

本指南聚焦儿科临床高发常见病，涵盖呼吸道、消化道、皮肤及营养代谢性疾病，针对0-14岁儿童生理特点与疾病诊疗规律，明确诊断标准、治疗方案及护理要点。指南遵循“安全优先、精准诊疗、兼顾预后”原则，适用于基层医疗机构儿科医师、全科医师及儿童保健人员，为临床诊疗提供标准化参考，同时指导家长科学应对儿童常见病症，减少误诊误治及并发症发生。

儿科疾病具有起病急、变化快、症状不典型等特点，诊疗中需充分考虑儿童各年龄段生长发育差异，严格把控用药剂量与治疗安全性，重视家庭护理与随访干预，实现“诊疗+护理+康复”一体化管理。

# 二、儿科常见病基础常识

## 2.1 儿科疾病特点

儿童免疫系统尚未发育成熟，对病原体抵抗力较弱，易发生呼吸道、消化道感染性疾病，且感染后易扩散蔓延；疾病症状多不典型，婴幼儿常以发热、哭闹、拒奶为主要表现，缺乏特异性体征，需结合年龄、病史及辅助检查综合判断。此外，儿童肝肾功能代谢能力较弱，用药后不良反应发生率高于成人，需严格遵循“按年龄、体重给药”原则，避免滥用药物。

儿科疾病预后与诊疗及时性密切相关，多数常见病经规范治疗可快速痊愈，少数重症病例（如重症肺炎、中毒性痢疾）若延误治疗，可能引发呼吸衰竭、心力衰竭等严重并发症，甚至危及生命，需重点警惕。

## 2.2 流行病学特征

- **高发疾病类型**：感染性疾病占儿科门诊病例的70%以上，其中急性上呼吸道感染、支气管炎、肺炎、腹泻病最为常见；其次为皮肤疾病（湿疹、荨麻疹）、营养缺乏症（维生素D缺乏性佝偻病、缺铁性贫血）及意外伤害（烫伤、误服）。

- **年龄分布特点**：婴幼儿（0-3岁）以急性上呼吸道感染、腹泻病、维生素D缺乏性佝偻病高发；学龄前儿童（3-6岁）易发生手足口病、疱疹性咽峡炎、急性胃肠炎；学龄期儿童（6-14岁）则以支气管炎、肺炎、过敏性疾病（哮喘、荨麻疹）多见。

- **季节相关性**：呼吸道感染性疾病多见于冬春季；腹泻病、手足口病、疱疹性咽峡炎高发于夏秋季；过敏性疾病（哮喘、荨麻疹）在春秋季花粉、尘螨增多时易发作。

# 三、常见疾病病因与发病机制

## 3.1 感染性疾病（呼吸道、消化道）

儿科感染性疾病主要由病毒、细菌、支原体等病原体引起，与儿童免疫力低下、环境因素及接触传播密切相关，具体病因如下：

- **病毒感染**：急性上呼吸道感染、支气管炎、手足口病、疱疹性咽峡炎多由病毒引起，常见病毒包括鼻病毒、流感病毒、副流感病毒、肠道病毒（EV71、柯萨奇病毒）等，主要通过呼吸道飞沫、接触被污染物品传播，冬春季或人群密集场所易暴发流行。

- **细菌感染**：细菌性肺炎、细菌性腹泻、化脓性扁桃体炎多由细菌感染所致，常见致病菌为肺炎链球菌、金黄色葡萄球菌、大肠杆菌、痢疾杆菌等，多在病毒感染基础上继发，或因饮食不洁、皮肤破损侵入引发。

- **其他病原体感染**：支原体肺炎由肺炎支原体引起，多见于学龄期儿童，通过呼吸道飞沫传播，症状以刺激性干咳为特点；真菌性肠炎多因滥用抗生素导致肠道菌群失调，或免疫力低下时继发念珠菌感染。

## 3.2 非感染性疾病（皮肤、营养代谢）

非感染性疾病多与遗传、环境、饮食、过敏等因素相关，发病机制复杂，常见类型及病因如下：

- **皮肤疾病**：湿疹与遗传过敏体质、皮肤屏障功能受损、接触过敏原（牛奶、鸡蛋、尘螨）相关，表现为皮肤红斑、丘疹、瘙痒；荨麻疹多由食物过敏、药物过敏、感染诱发，属于变态反应性疾病，皮肤出现风团、瘙痒，可自行消退。

- **营养代谢性疾病**：维生素D缺乏性佝偻病因儿童日照不足、维生素D摄入缺乏，导致钙磷代谢紊乱，骨骼矿化不足；缺铁性贫血多由婴幼儿辅食添加不当、生长发育过快、慢性失血（如肠道寄生虫感染）引起，铁元素摄入不足或丢失过多导致血红蛋白合成减少。

# 四、预防措施

## 4.1 一级预防（病因预防）

- **增强免疫力**：合理喂养，婴幼儿坚持母乳喂养至6个月，按时添加辅食，保证蛋白质、维生素、矿物质摄入；规律作息，保证充足睡眠，适度进行户外活动（每日1-2小时），增强体质；按国家免疫规划程序完成疫苗接种（如流感疫苗、手足口病疫苗、肺炎疫苗），提高特异性免疫力。

- **做好防护措施**：传染病流行季减少前往人群密集、通风不良场所，佩戴儿童专用口罩；勤洗手，尤其是饭前便后、接触公共物品后，用流动水和儿童专用洗手液清洁；保持居住环境通风换气，定期消毒玩具、餐具，避免接触过敏原（如花粉、宠物毛发）。

- **养成良好习惯**：培养儿童规律作息、不挑食偏食的习惯，避免营养不良；注意个人卫生，勤剪指甲，避免搔抓皮肤；夏季注意饮食卫生，不吃生冷、不洁食物，预防腹泻病；冬季做好保暖，避免受凉诱发呼吸道感染。

## 4.2 二级预防（早发现、早诊断、早治疗）

- **定期健康体检**：0-3岁婴幼儿每3个月体检1次，3-6岁每6个月体检1次，6-14岁每年体检1次，重点监测生长发育指标（身高、体重、头围）、血常规、维生素D水平，早期发现营养缺乏症、贫血等疾病，及时干预。

- **警惕早期信号**：家长密切关注儿童身体变化，出现以下症状及时就医：持续发热超过24小时、精神萎靡、拒奶呕吐；咳嗽喘息、呼吸困难、口唇发绀；腹泻频繁、大便带血、尿量减少；皮肤出现皮疹、瘙痒、破溃；生长发育迟缓、多汗易惊、面色苍白。

- **规范家庭护理**：儿童出现轻微不适（如低热、轻度咳嗽）时，家长避免盲目用药，可通过物理降温、补充水分、休息等方式护理，症状无缓解或加重时及时就医，避免延误病情。

# 五、常见疾病诊断方法

## 5.1 临床诊断（初步筛查）

结合儿童年龄、病史、临床表现及体格检查进行初步判断，是儿科疾病诊断的基础，适用于基层医疗机构快速识别病症。

诊断要点：① 病史采集：重点询问发病时间、症状特点（发热、咳嗽、腹泻等）、饮食睡眠情况、既往病史、疫苗接种史、接触史；② 体格检查：全面检查精神状态、体温、呼吸、心率，重点排查呼吸道（肺部啰音）、消化道（腹部压痛、肠鸣音）、皮肤黏膜（皮疹、口腔疱疹），结合年龄特点判断病情轻重。

## 5.2 实验室与影像学诊断（辅助确诊）

|诊断方法|检测目标/用途|适用疾病|注意事项|
|---|---|---|---|
|血常规|判断感染类型（细菌/病毒），排查贫血|呼吸道感染、腹泻病、贫血|婴幼儿采血可采用指尖血，避免空腹，减少哭闹应激|
|粪便常规+培养|排查腹泻病因，明确致病菌|腹泻病、细菌性痢疾|采集新鲜粪便样本，避免污染，及时送检|
|胸部X线/CT|评估肺部炎症范围，排查肺炎、支气管炎|肺炎、支气管炎、喘息性疾病|控制辐射剂量，婴幼儿需家长陪同，做好防护|
|维生素D+微量元素检测|排查维生素D缺乏、缺铁、缺锌|佝偻病、贫血、生长发育迟缓|无需空腹，检测前避免大量补充营养素|
|病毒核酸检测|明确病毒感染类型|手足口病、疱疹性咽峡炎、流感|采集咽拭子或粪便样本，采样时动作轻柔，减少儿童不适|
# 六、常见疾病治疗方案

## 6.1 急性上呼吸道感染（感冒）

多由病毒感染引起，具有自限性，治疗以对症支持为主，避免滥用抗生素。

- **对症治疗**：低热（<38.5℃）采用物理降温（温水擦拭颈部、腋下、腹股沟）；高热（≥38.5℃）给予对乙酰氨基酚（每次15mg/kg）或布洛芬（每次10mg/kg），间隔4-6小时重复，24小时不超过4次；鼻塞流涕者用生理盐水喷鼻剂清洗鼻腔，缓解症状。

- **抗病毒治疗**：病毒感染早期（48小时内）可酌情使用小儿豉翘清热颗粒、小儿氨酚黄那敏颗粒等中成药，无需常规使用抗病毒西药；合并细菌感染（如化脓性扁桃体炎）时，遵医嘱使用青霉素类、头孢类抗生素，疗程3-5天。

- **护理要点**：保证休息，多饮水，清淡饮食；保持室内通风，避免受凉；密切监测体温及精神状态，警惕高热惊厥。

## 6.2 小儿腹泻病

分为感染性（病毒/细菌）和非感染性（饮食不当、腹部受凉），治疗核心是预防脱水、纠正电解质紊乱。

- **补液治疗**：轻度脱水者口服补液盐Ⅲ，每次50-100ml，少量多次饮用；中度至重度脱水需静脉补液，纠正脱水及电解质紊乱，维持酸碱平衡。

- **对症治疗**：病毒感染性腹泻可使用蒙脱石散（保护肠黏膜，与其他药物间隔1-2小时）、益生菌（调节肠道菌群）；细菌性腹泻需遵医嘱使用抗生素，避免滥用；腹泻缓解后逐渐恢复饮食，从清淡流质（米汤、面汤）过渡至正常饮食。

- **护理要点**：注意腹部保暖，避免受凉；勤换尿布，清洗臀部，预防红臀；记录排便次数、性状及尿量，评估脱水情况。

## 6.3 维生素D缺乏性佝偻病

多见于婴幼儿，因维生素D缺乏导致钙磷代谢紊乱，治疗以补充维生素D、钙剂为主，结合户外活动。

- **药物治疗**：活动期给予维生素D制剂，每日400-800IU，口服为主，严重者可肌内注射维生素D₃，每月1次，连用2-3次；同时补充钙剂（如葡萄糖酸钙、碳酸钙），避免与牛奶同服，影响吸收。

- **非药物治疗**：每日保证1-2小时户外活动，多晒太阳，促进皮肤合成维生素D；合理喂养，母乳喂养者母亲补充维生素D，人工喂养者选择维生素D强化奶粉，及时添加辅食（蛋黄、深海鱼）。

- **随访监测**：每3个月复查维生素D及血钙水平，调整用药剂量，避免维生素D过量中毒。

## 6.4 手足口病

由肠道病毒引起，多见于5岁以下儿童，多数轻症可自愈，重症需及时干预。

- **对症治疗**：发热者给予退热药物，口腔疱疹疼痛者用康复新液含漱或涂抹，缓解疼痛；皮肤疱疹保持清洁干燥，避免搔抓，防止感染。

- **重症治疗**：出现持续高热、精神萎靡、呕吐、肢体抖动、呼吸困难等重症表现时，立即住院治疗，给予抗病毒、脱水降颅压、呼吸支持等综合救治，避免引发脑炎、肺水肿。

- **隔离与护理**：患儿居家隔离至症状消失后1周，避免传播；清淡饮食，多喝水，保证休息；对玩具、餐具、衣物彻底消毒，避免交叉感染。

# 七、诊疗路径与护理要点

## 7.1 通用诊疗路径

1. 初步筛查：结合年龄、病史、临床表现及体格检查，判断疾病类型，必要时完善辅助检查（血常规、粪便常规等）。

2. 明确诊断：根据检查结果确诊疾病，区分轻症与重症，评估是否存在并发症。

3. 分层治疗：轻症病例居家治疗，遵医嘱用药及护理；重症病例住院治疗，制定个性化诊疗方案，加强病情监测。

4. 随访康复：轻症患儿治疗后3-5天随访，评估症状缓解情况；重症患儿康复后定期复查，指导后续护理及预防措施。

## 7.2 家庭护理核心要点

- **饮食护理**：生病期间给予清淡、易消化、营养丰富的食物，避免辛辣、油腻、生冷食物；婴幼儿保证奶量摄入，辅食添加循序渐进，避免加重肠胃负担。

- **皮肤护理**：保持皮肤清洁干燥，衣物选择宽松、透气的棉质材料；腹泻患儿及时清洗臀部，涂抹护臀膏；皮疹患儿避免搔抓，防止破溃感染。

- **病情监测**：密切监测体温、呼吸、心率及精神状态，记录症状变化（如咳嗽、腹泻次数）；出现异常情况（高热不退、精神萎靡、呼吸困难）及时就医。

- **用药护理**：严格遵医嘱用药，按时、按量服用，不擅自增减剂量或停药；注意观察用药后反应，如出现皮疹、呕吐、腹泻等不良反应，立即停药并就医。

## 7.3 常见误区提醒

|常见误区|正确做法|
|---|---|
|儿童发热立即用退烧药|低热（<38.5℃）优先物理降温，高热时再用退烧药，避免频繁用药掩盖病情|
|感冒后滥用抗生素|多数感冒为病毒感染，无需使用抗生素，合并细菌感染时遵医嘱使用|
|腹泻后禁食禁水|腹泻时需保证水分及营养摄入，轻度脱水口服补液盐，避免禁食导致脱水加重|
|自行给儿童服用成人药物|儿童用药需选择专用剂型，按年龄、体重给药，严禁使用成人药物|
