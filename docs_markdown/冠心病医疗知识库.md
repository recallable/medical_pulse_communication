# 冠心病医疗知识库

# 一、冠心病概述

冠心病（Coronary Heart Disease，CHD）全称冠状动脉粥样硬化性心脏病，是由于冠状动脉粥样硬化使血管腔狭窄、闭塞，或（和）因冠状动脉功能性痉挛（血管收缩异常）导致心肌缺血缺氧或坏死而引起的心脏病，简称冠心病，亦称缺血性心脏病。作为全球高发的心血管疾病，冠心病是中老年人致死、致残的主要原因之一，我国患病群体已超1.1亿，且中青年患病率呈逐年上升趋势，严重威胁居民健康。

冠心病病程进展隐匿，早期可无明显症状，随血管狭窄程度加重，逐渐出现胸痛、胸闷等表现，重症可引发急性心肌梗死、心源性猝死等严重结局。本病可通过早期筛查、规范干预控制病情进展，降低急性发作风险，其治疗需结合生活方式调整、药物治疗、介入治疗或外科手术，实现个体化精准诊疗。

# 二、冠心病基础常识

## 2.1 疾病分型与核心特征

临床根据症状表现、病情急缓及病理改变，将冠心病分为以下亚型，各亚型特征及预后差异显著：

|疾病亚型|核心表现|病情特点|
|---|---|---|
|慢性稳定型心绞痛|劳累、情绪激动时出现胸骨后压榨性胸痛、胸闷，休息或含服硝酸甘油后3-5分钟缓解|病情稳定，血管狭窄程度相对固定，发作频率、诱因较规律|
|急性冠状动脉综合征（ACS）|突发剧烈胸痛，持续不缓解，伴大汗、呼吸困难、恶心呕吐，部分患者疼痛放射至左肩、后背|病情危急，含斑块破裂、血栓形成，包括不稳定型心绞痛、非ST段抬高型心肌梗死、ST段抬高型心肌梗死|
|无症状性心肌缺血|无明显胸痛、胸闷症状，仅在体检、心电图检查时发现心肌缺血证据|易被忽视，仍存在心肌损伤及进展风险，需定期监测|
|缺血性心肌病|长期心肌缺血导致心脏扩大、心功能减退，表现为呼吸困难、下肢水肿、乏力|病程迁延，易合并心律失常、心力衰竭，预后较差|
## 2.2 流行病学与高危人群

- **高发人群**：45岁以上男性、55岁以上女性及绝经后女性；高血压、糖尿病、高脂血症（“三高”）患者；长期吸烟者及二手烟、三手烟暴露者；超重肥胖（BMI≥24）、缺乏运动者；有冠心病家族史者（直系亲属55岁前患病）；长期熬夜、压力过大、过量饮酒者。

- **人群与地域特征**：我国北方患病率高于南方，城市略高于农村，与饮食结构（高油高盐）、气候寒冷、医疗筛查覆盖率相关；中青年群体中，男性患病率显著高于女性，且因不良生活方式，发病年龄持续提前。

- **疾病负担**：冠心病急性发作需紧急救治及长期康复，医疗费用高昂，且易导致劳动力丧失、生活自理能力下降，给家庭和社会带来沉重经济负担，是我国居民心血管疾病死亡的主要诱因之一。

# 三、冠心病病因与发病机制

## 3.1 核心病因

冠心病的核心病因是冠状动脉粥样硬化，其发生是遗传因素、环境因素与代谢异常共同作用的结果，核心诱因如下：

- **代谢与生活方式因素**：高脂血症是首要危险因素，低密度脂蛋白胆固醇（LDL-C）升高可直接沉积于血管壁，形成粥样斑块；高血压损伤血管内皮，加速脂质沉积；糖尿病导致血管内皮功能紊乱，增加斑块形成及破裂风险；长期吸烟、过量饮酒、高油高盐饮食、久坐不动等，进一步加重血管损伤。

- **遗传与个体因素**：家族聚集性明显，直系亲属患冠心病，子女患病风险比常人高2-3倍；年龄增长导致血管弹性自然减退，内皮功能下降，患病风险升高；绝经后女性因雌激素保护作用消失，风险显著高于绝经前。

- **其他诱发因素**：长期焦虑、情绪剧烈波动、寒冷刺激、过度劳累等，可诱发冠状动脉痉挛，加重心肌缺血；肥胖、高同型半胱氨酸血症等，也会增加患病风险。

## 3.2 核心发病机制

冠心病的核心发病机制是冠状动脉粥样硬化形成与进展，以及斑块破裂引发的血栓栓塞。高LDL-C、高血压、吸烟等因素导致冠状动脉内皮受损，脂质成分沉积于血管内皮下，形成脂质条纹，逐渐发展为粥样斑块。斑块由核心脂质区、纤维帽组成，纤维帽完整时，斑块可缓慢增大，导致血管腔狭窄，引起心肌缺血（如稳定型心绞痛）。

当纤维帽变薄、破裂，斑块内脂质成分暴露，可激活凝血系统，快速形成血栓，阻塞冠状动脉管腔。若血栓完全阻塞血管，且持续时间较长，可导致心肌坏死（心肌梗死）；若血栓不完全阻塞或短暂阻塞，可引发不稳定型心绞痛。此外，冠状动脉功能性痉挛可导致血管暂时性狭窄，即使无明显斑块，也可诱发心肌缺血。

# 四、冠心病预防与健康管理

## 4.1 一级预防（病因预防）

针对普通人群及高危群体，通过综合干预降低粥样斑块形成风险，是冠心病预防的核心，均为A级证据强推荐：

- **膳食与生活方式调理**：遵循“低盐、低脂、低糖、高纤维”原则，每日盐摄入≤5克，减少动物脂肪、反式脂肪摄入，多吃新鲜蔬菜、水果、全谷物、优质蛋白（鱼、禽、豆制品）；彻底戒烟，避免二手烟暴露，严格限酒，男性每日酒精摄入≤25克，女性≤15克，最好不饮酒；规律有氧运动，每周≥150分钟中等强度运动（快走、游泳、骑自行车），控制体重，BMI目标＜24，腰围男性＜90cm、女性＜85cm。

- **高危因素控制**：定期监测血压、血糖、血脂，“三高”人群严格遵医嘱控制指标达标（血压＜130/80mmHg，空腹血糖＜7.0mmol/L，LDL-C＜1.8mmol/L）；有家族史、肥胖等高危因素者，每年进行心血管专项筛查，早期发现异常。

- **情绪与作息管理**：保证每日7小时高质量睡眠，避免熬夜；积极调节情绪，缓解焦虑压力，避免长期精神紧张。

## 4.2 二级预防（早筛查、早干预）

- **定期筛查**：40岁以上人群每年进行1次心电图、血脂、血糖、血压检查；高危人群（“三高”、家族史、吸烟者）每6个月复查1次，必要时进行冠脉CT、运动负荷试验，早期发现心肌缺血及血管狭窄。

- **症状监测与早期干预**：警惕冠心病早期信号，如劳累后胸痛、胸闷、心悸、活动后气短等，及时就医排查；确诊轻症患者，尽早启动生活方式干预+药物治疗，延缓斑块进展。

- **慢病登记管理**：医疗机构建立冠心病患者分级管理体系，对确诊患者进行定期随访，评估病情变化，调整治疗方案；社区卫生服务中心提供健康指导，提高患者治疗依从性。

## 4.3 三级预防（慢病管理与并发症防治）

- **规范用药与治疗**：患者严格遵医嘱规律服药，不可自行停药、减药，掌握药物不良反应及应对方法；术后患者（支架、搭桥）需长期服用抗血小板药物，预防血栓形成。

- **康复与生活管理**：心肌梗死、介入/手术治疗后患者，在医生指导下进行康复训练（从轻度活动逐步过渡到中等强度运动），避免剧烈运动、情绪剧烈波动、寒冷刺激等急性诱因；定期复查，评估心功能及斑块稳定性。

- **并发症防治**：早期发现心律失常、心力衰竭等并发症，及时干预；长期卧床患者定期翻身、活动肢体，预防血栓栓塞；控制感染，避免诱发病情加重。

# 五、冠心病诊断方法

## 5.1 临床诊断（初步筛查）

结合患者病史、症状、体格检查及流行病学史进行初步判断，为后续辅助检查提供方向，适用于基层医疗机构首诊及社区筛查。

诊断要点：① 病史采集：重点询问冠心病家族史、既往病史（“三高”、吸烟史）、用药史，以及胸痛症状特点（发作诱因、持续时间、缓解方式、放射部位）；② 体格检查：测量血压、心率、心律，听诊心脏杂音、肺部啰音，评估心功能状态，急性发作期可能出现心率加快、血压下降、面色苍白等体征。

## 5.2 实验室与影像学诊断（辅助确诊与病情评估）

|诊断方法|检测目标/用途|结果解读与适用场景|注意事项|
|---|---|---|---|
|血生化检查（心肌酶、血脂、血糖）|评估心肌损伤、代谢危险因素|肌钙蛋白、肌酸激酶同工酶升高提示心肌损伤/梗死；LDL-C升高为核心危险因素，用于病情评估及预后判断。|需空腹采血，急性发作期可随时采血，采血前避免高脂饮食、饮酒。|
|心电图（常规+动态）|评估心肌缺血、心律失常|常规心电图可见ST段压低、T波倒置（心肌缺血），ST段抬高（急性心肌梗死）；动态心电图可捕捉阵发性心肌缺血及心律失常。|检查前避免剧烈运动、情绪激动，动态心电图监测期间保持正常生活节奏。|
|心脏超声|评估心脏结构与功能，排查心肌缺血范围|可显示心室壁运动异常（心肌缺血/梗死表现）、心腔大小、心功能指标，用于病情评估及并发症排查。|检查前无需特殊准备，配合医生调整体位即可。|
|冠状动脉造影（CAG）|评估冠状动脉狭窄部位、程度，确诊冠心病|冠心病诊断“金标准”，可精准判断血管狭窄程度（轻度＜50%、中度50%-70%、重度≥70%），为介入/手术治疗提供依据。|有创检查，需严格评估适应症，术后卧床休息，观察穿刺部位出血情况。|
# 六、冠心病治疗方案

## 6.1 治疗原则

冠心病治疗以“缓解症状、改善心肌供血、稳定斑块、预防急性发作、降低病死率”为核心。慢性稳定期以药物治疗+生活方式干预为主，控制病情进展；急性冠状动脉综合征需紧急救治，快速开通阻塞血管，挽救缺血心肌；复杂病例（严重血管狭窄、多支病变）需结合介入或外科手术治疗，重建冠状动脉供血，同时兼顾合并症治疗，实现个体化方案。

## 6.2 药物治疗（基础干预手段）

药物治疗适用于所有冠心病患者，需根据病情类型、合并症选择，核心药物类别及应用如下：

|药物类别|代表药物|适用场景|不良反应与注意事项|
|---|---|---|---|
|抗血小板药|阿司匹林、氯吡格雷、替格瑞洛|所有冠心病患者，预防血栓形成|增加出血风险，观察皮肤瘀斑、牙龈出血，避免与抗凝药滥用。|
|调脂药（他汀类）|阿托伐他汀、瑞舒伐他汀|稳定斑块，降低急性发作风险|可能损伤肝功能、引发肌痛，用药期间定期复查肝功能、肌酸激酶。|
|硝酸酯类|硝酸甘油、单硝酸异山梨酯|缓解心绞痛症状，扩张冠状动脉|可能出现头痛、面部潮红、低血压，舌下含服硝酸甘油时需坐位，避免晕厥。|
|β受体阻滞剂|美托洛尔、比索洛尔|稳定型心绞痛、心肌梗死后，降低心率及心肌耗氧量|可能导致心率减慢、乏力，不可突然停药，需逐步减量。|
## 6.3 介入与手术治疗（复杂病例干预）

- **介入治疗（PCI）**：微创治疗，核心为冠状动脉支架植入术。通过导管将支架送至血管狭窄部位，扩张后支撑血管壁，恢复血流通畅，适用于急性心肌梗死、不稳定型心绞痛、重度血管狭窄患者。具有创伤小、恢复快、起效迅速的优势，术后需长期服用抗血小板药物。

- **外科手术治疗**：核心为冠状动脉旁路移植术（冠脉搭桥术）。取患者自身血管（大隐静脉、乳内动脉），搭建旁路血管，绕过狭窄部位，为心肌供血，适用于多支血管病变、严重血管狭窄、介入治疗效果不佳的患者。需开胸操作，创伤较大，术后需长期康复训练。

## 6.4 急性冠状动脉综合征急救

- **紧急处理**：立即卧床休息、吸氧，保持呼吸道通畅，监测心率、血压、血氧饱和度；舌下含服硝酸甘油缓解胸痛，剧烈疼痛者可使用吗啡镇痛，同时给予抗血小板药物（阿司匹林+氯吡格雷/替格瑞洛）嚼服。

- **再灌注治疗**：ST段抬高型心肌梗死需在发病12小时内尽快开通阻塞血管，首选介入治疗（支架植入），无条件者可进行静脉溶栓治疗（如尿激酶、替奈普酶），挽救缺血心肌。

- **并发症防治**：预防心力衰竭、心源性休克、心律失常等并发症，必要时使用血管活性药物、利尿剂，转入ICU监护治疗。

# 七、鉴别诊断与转诊指征

## 7.1 常见疾病鉴别诊断

|鉴别疾病组合|核心鉴别点|
|---|---|
|冠心病心绞痛与肺部疾病（肺炎、胸膜炎）|心绞痛：胸痛与劳累、情绪相关，休息后缓解，无发热、咳痰，心电图可见心肌缺血；肺部疾病：胸痛与呼吸、咳嗽相关，伴发热、咳痰，胸部影像学可见肺部病变。|
|冠心病与主动脉夹层|冠心病：胸痛多为压榨性，局限于胸骨后，硝酸甘油可缓解；主动脉夹层：突发剧烈撕裂样胸痛，放射至后背，血压显著升高，CT血管造影可明确诊断。|
|冠心病与胃食管反流病|冠心病：胸痛与进食无关，劳累后发作；胃食管反流病：胸痛伴反酸、烧心，进食后加重，抑酸药物治疗有效，心电图无心肌缺血表现。|
## 7.2 转诊指征

出现以下情况，需及时转诊至上级医院心血管内科或ICU治疗：

- 突发剧烈胸痛，持续不缓解，伴大汗、呼吸困难、恶心呕吐，疑似急性心肌梗死；

- 稳定型心绞痛患者症状加重，发作频率、持续时间增加，药物治疗效果不佳；

- 冠状动脉造影提示重度血管狭窄（≥70%）、多支血管病变，需介入或手术治疗；

- 出现心力衰竭、严重心律失常（室速、房颤）等并发症，常规治疗无法控制；

- 介入/手术治疗后出现出血、支架内血栓等不良反应，需进一步处理；

- 疑似冠心病但诊断不明确，需进一步进行深度检查（如冠脉造影）明确病情。
