# 高血压医疗知识库

# 一、高血压概述

高血压（hypertension，arterial hypertension）是一种以体循环动脉压升高、周围小动脉阻力增高为核心特征，伴或不伴心排血量和血容量增加的临床综合征，是全球高发的心血管疾病，也是脑卒中、心力衰竭、肾衰竭等严重并发症的重要危险因素。我国成人高血压患病率已达27.9%，每4个成年人中就有1人患病，其中65岁以上人群发病率近50%，中青年群体患病率增速显著加快，成为重大公共卫生问题。

高血压多起病隐匿，早期常无明显症状，被称为“沉默的杀手”。临床诊断需结合诊室、家庭及动态血压监测结果综合判断，而非单次血压升高。本病分为原发性高血压（占比90%-95%，病因与遗传、环境因素相关）和继发性高血压（由肾脏疾病、内分泌疾病等明确病因引发），原发性高血压虽无法根治，但通过长期规范干预可有效控制血压，显著降低并发症风险；部分轻度高血压患者通过生活方式调整可实现血压逆转，无需药物治疗。

# 二、高血压基础常识

## 2.1 诊断标准与分级

2024年修订版《中国高血压防治指南》明确采用“三位一体”诊断体系，以诊室血压为核心，结合家庭和动态血压监测，避免单一测量导致的误判。正常血压及高血压分级标准如下，适用于18岁以上成年人，当收缩压（SBP）和舒张压（DBP）分属不同级别时，以较高分级为准。

|分类|收缩压（mmHg）|舒张压（mmHg）|补充说明|
|---|---|---|---|
|正常血压|<120|<80|心血管疾病风险较低|
|正常高值|120-139|80-89|10年后心血管风险翻倍，64%可能进展为高血压|
|高血压|≥140|≥90|需非同日3次测量确认|
|1级高血压（轻度）|140-159|90-99|低危人群可先干预生活方式|
|2级高血压（中度）|160-179|100-109|建议立即启动药物治疗+生活干预|
|3级高血压（重度）|≥180|≥110|极高危，需紧急医疗干预|
|单纯收缩期高血压|≥140|<90|多见于老年人|
特殊测量标准：家庭血压连续5-7天监测，平均值≥135/85mmHg可确诊；24小时动态血压平均≥130/80mmHg、白天≥135/85mmHg、夜间≥120/70mmHg也可确诊，可有效识别“白大衣高血压”（诊室高、日常正常）和“隐匿性高血压”（诊室正常、日常升高）。

## 2.2 流行病学与高危人群

- **高发人群**：有高血压家族史者；65岁以上老年人；情绪易激动、工作生活压力大者；长期吸烟者及二手烟、三手烟暴露者；每日盐摄入超标者；超重肥胖者（BMI≥24）；过量饮酒者；合并糖尿病、慢性肾病者。

- **人群特征**：我国北方患病率高于南方，农村略高于城市；中青年（20-39岁）发病率增速最快，较往年上升144.4%；34%中青年患者表现为“单纯舒张期高血压”（收缩压<140mmHg、舒张压≥90mmHg），与熬夜、高盐饮食、压力大密切相关。

- **疾病负担**：高血压病程长、控制不佳易引发多器官损害，其导致的脑卒中、心力衰竭等并发症，不仅降低患者生活质量，还带来沉重的家庭和社会经济负担，是我国居民心血管疾病死亡的主要诱因之一。

# 三、高血压病因与发病机制

## 3.1 核心病因

高血压的发生是遗传因素与环境因素共同作用的结果，原发性高血压病因复杂，继发性高血压则有明确致病源头，具体如下：

- **遗传因素**：家族聚集性明显，父母均患高血压，子女患病风险比常人高2-3倍，遗传因素在高血压发病中占比约40%，与多个易感基因变异相关。

- **环境因素**：高盐饮食是重要诱因，每日盐摄入＞5克会显著升高血压，隐形盐（酱油、火腿肠、挂面）危害突出；超重肥胖、缺乏运动、过量饮酒、长期吸烟可通过影响血管弹性和神经调节诱发高血压；长期焦虑、睡眠不足（＜7小时）激活肾素-血管紧张素系统，导致血压持续升高。

- **继发性病因**：由其他疾病引发，常见于肾脏疾病（肾小球肾炎、肾动脉狭窄）、内分泌疾病（甲亢、原发性醛固酮增多症）、睡眠呼吸暂停综合征及药物（避孕药、糖皮质激素）等，去除病因后部分患者血压可恢复正常。

## 3.2 发病机制

高血压的核心发病机制是血管调节功能紊乱，导致外周血管阻力增加、血容量异常，不同病因引发的机制各有侧重。遗传与环境因素共同作用下，血管平滑肌细胞钙通道异常激活，导致血管收缩增强；肾素-血管紧张素-醛固酮系统过度激活，促进水钠潴留和血管收缩，升高血容量和外周阻力；交感神经兴奋则加快心率、收缩血管，进一步升高血压。

长期高血压会引发血管内皮损伤，加速动脉粥样硬化，形成“高血压-血管损伤-血压进一步升高”的恶性循环。轻度高血压早期多为功能性调节异常，此时通过生活方式干预，可激活人体自身血压调节机制，实现血压逆转；若病情进展，血管出现器质性病变，则需长期药物维持。

# 四、高血压预防与健康管理

## 4.1 一级预防（病因预防）

针对正常人群及高危群体，通过生活方式干预降低发病风险，是高血压预防的核心，以下措施均为A级证据强推荐：

- **膳食调理**：严格限盐，每日摄入量≤5克（约一啤酒瓶盖），减少隐形盐摄入；多吃富含钾、钙的食物（新鲜蔬菜、水果、奶制品），限制高脂、高糖、高胆固醇食物；戒烟限酒，男性每日酒精摄入≤25克（约1两白酒），女性≤15克，避免空腹饮酒。

- **生活方式干预**：控制体重，BMI目标＜24，腰围男性＜90cm、女性＜85cm，每减重1公斤可使血压下降1-2mmHg；规律有氧运动，每周≥150分钟中等强度运动（快走、游泳、骑自行车），促进血管舒张；保证每日7小时高质量睡眠，避免熬夜，积极管理情绪，缓解焦虑压力。

- **高危人群筛查**：有家族史、肥胖、高盐饮食等高危因素者，每年至少测量2次血压；40岁以上人群体检需常规检测血压，早期发现血压异常。

## 4.2 二级预防（早发现、早干预）

- **规范诊断**：单次诊室血压≥140/90mmHg不确诊，需非同日3次测量，或通过家庭自测、24小时动态血压监测确认，排除白大衣高血压等假性升高情况。

- **分层干预**：低危人群（1级高血压+无危险因素）先进行3-6个月生活方式干预，定期监测血压；中高危人群（1级高血压+吸烟/肥胖，或2级及以上高血压）立即启动药物治疗+强化生活干预；轻度高血压患者（140-159/90-99mmHg）通过减重、限盐等干预，42.3%可降至理想水平（＜130/80mmHg）。

- **定期复查**：确诊患者每1-3个月复查血压，同时检测血糖、血脂、肾功能，评估靶器官损害情况，调整干预方案。

## 4.3 三级预防（慢病管理与并发症防治）

- **规范用药管理**：遵医嘱规律服药，不可自行停药、减药，避免血压反跳性升高诱发急症；根据血压特点选择服药时间，如利尿剂晨起服用，避免夜尿过多。

- **靶器官监测**：定期进行心电图、心脏超声、尿常规、眼底检查，早期发现左心室肥厚、蛋白尿、眼底病变等器官损害，及时调整治疗方案。

- **并发症预防**：严格控制血压达标，收缩压从140mmHg降至130mmHg，可使脑卒中风险降低14%；避免情绪剧烈波动、过度劳累，预防高血压危象、脑病等急性并发症。

# 五、高血压诊断方法

## 5.1 临床诊断（初步筛查）

结合病史、症状及体格检查初步判断，为辅助检查提供方向，适用于基层医疗机构首诊。

诊断要点：① 病史采集：重点询问高血压家族史、生活习惯（盐摄入、饮酒、运动）、既往病史及用药史，有无头晕、头痛等症状；② 体格检查：规范测量右上臂血压（测前安静休息5分钟），检查心率、心律，评估有无颈静脉怒张、下肢水肿等靶器官损害体征。

## 5.2 实验室与辅助诊断（确诊与风险评估）

|诊断方法|检测目标/用途|结果解读与适用场景|注意事项|
|---|---|---|---|
|24小时动态血压监测|精准评估血压波动规律，排除假性高血压|可识别夜间高血压、隐匿性高血压，为个体化治疗提供依据，尤其适用于诊室血压波动大的患者。|监测期间保持正常生活节奏，避免剧烈运动，准确记录监测时间及活动情况。|
|血常规+血生化|评估肝肾功能、电解质，排查并发症及继发性病因|肌酐、尿素氮升高提示肾功能损害；低钾血症可能与利尿剂使用或原发性醛固酮增多症相关；血糖、血脂异常增加心血管风险。|需空腹采血，采血前避免高脂饮食、饮酒，停用影响指标的药物（遵医嘱）。|

# 六、高血压治疗方案

## 6.1 治疗原则

高血压治疗以“个体化达标、长期控制、预防并发症”为核心，根据患者年龄、病情、合并症设定血压目标：一般人群（含65-79岁老人）诊室血压＜140/90mmHg，能耐受者降至＜130/80mmHg；80岁以上老人收缩压控制在130-139mmHg，避免过低导致头晕跌倒；糖尿病、冠心病、肾病患者严格控制在＜130/80mmHg。多数患者需在4-12周内逐步降压，年轻、病程短者可稍快，老年、合并症多者需放缓速度。

## 6.2 非药物治疗（基础干预）

适用于所有高血压患者，低危1级高血压患者可作为首选，需长期坚持，具体措施如下：

- 限盐减重：每日盐摄入≤5克，减重5%-10%可显著改善血管功能，降低血压1-2mmHg/公斤体重。

- 运动与作息：每周150分钟中等强度运动，配合每日7小时睡眠，避免熬夜和过度劳累，缓解精神压力。

- 戒烟限酒：彻底戒烟，避免二手烟暴露；严格控制饮酒量，最好不饮酒，饮酒者需减量并避免空腹饮用。

- 定期监测：家庭自测血压（早晚各1次，每次2遍取平均），记录数值，为医生调整方案提供依据。

## 6.3 药物治疗（规范用药）

2024年指南推荐六大类降压药作为初始和维持治疗药物，需根据患者合并症、耐受性选择，优先单药治疗，必要时联合用药，具体分类及特点如下：

|药物类别|代表药物|强适应症|不良反应与注意事项|
|---|---|---|---|
|噻嗪类利尿剂|氢氯噻嗪、吲达帕胺|老年高血压、单纯收缩期高血压、盐敏感性高血压|低钾血症、高尿酸血症，晨起服用避免夜尿；血尿酸≥480μmol/L需换药。|
|二氢吡啶类CCB|氨氯地平、硝苯地平控释片|老年高血压、伴心绞痛、颈动脉粥样硬化|脚踝水肿、牙龈增生，联用普利类/沙坦类可减轻水肿；保持口腔卫生。|
|ACEI（普利类）|依那普利、培哚普利|心力衰竭、心肌梗死后、糖尿病肾病、蛋白尿|干咳、高钾血症；干咳不耐受者换沙坦类，定期监测血钾、肌酐。|

|ARNI|沙库巴曲缬沙坦|高血压合并心力衰竭、左心室肥厚、难治性高血压|血管性水肿、高钾血症；禁与ACEI合用，停用ACEI后需间隔36小时再使用。|
## 6.4 并发症治疗

- **高血压危象/脑病**：立即静脉输注降压药（硝普钠、硝酸甘油），1小时内将收缩压降至160mmHg左右，避免血压骤降；控制脑水肿，预防抽搐和昏迷，转入ICU监护。

- **靶器官损害**：合并脑卒中者选用利尿剂、ACEI；心力衰竭者优先ARNI或ACEI+β受体阻滞剂；肾功能损害者避免使用肾毒性药物，控制血压＜130/80mmHg，必要时透析治疗。

- **主动脉夹层**：紧急降压至收缩压100-120mmHg，镇痛镇静，尽快手术治疗，降低病死率。

# 七、鉴别诊断与转诊指征

## 7.1 常见疾病鉴别诊断

|鉴别疾病|核心鉴别点|
|---|---|
|白大衣高血压|诊室血压≥140/90mmHg，家庭及动态血压正常，无靶器官损害，无需药物治疗，定期监测即可。|
|体位性高血压|卧位转直立位后收缩压升高≥20mmHg，多无明显症状，体检时发现，以生活方式干预为主。|
|继发性高血压|血压骤升（≥180/110mmHg）、药物治疗效果差，伴低钾、血尿、心悸等，排查肾脏、内分泌疾病可确诊。|
## 7.2 转诊指征

出现以下情况，需及时转诊至上级医院心血管内科治疗：

- 3级高血压（≥180/110mmHg），或伴剧烈头痛、呕吐、视力模糊等急性并发症表现；

- 血压控制不佳，多种药物联合治疗后仍不达标，或出现严重药物不良反应；

- 疑似继发性高血压，需进一步排查病因（肾动脉狭窄、内分泌疾病等）；

- 合并严重靶器官损害，如心力衰竭、心肌梗死、脑卒中、肾功能衰竭；

- 妊娠期高血压、青少年高血压，或合并糖尿病、冠心病等复杂基础疾病。
