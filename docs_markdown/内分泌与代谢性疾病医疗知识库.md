# 内分泌与代谢性疾病医疗知识库

# 一、内分泌与代谢性疾病概述

内分泌与代谢性疾病是累及内分泌腺体（甲状腺、胰腺、肾上腺、垂体等）及代谢系统的一组疾病总称，病因涵盖激素分泌异常、受体敏感性障碍、代谢酶缺陷、遗传因素、环境诱因等，临床症状复杂多样，涉及代谢、生长发育、生殖、免疫等多个系统。此类疾病发病率呈逐年上升趋势，我国成人患病率超20%，其中糖尿病、甲状腺疾病占比最高，部分疾病（如严重甲亢危象、糖尿病酮症酸中毒）发病急骤，若救治不及时可危及生命。

多数内分泌与代谢性疾病呈慢性病程，早期症状隐匿，易被忽视，长期激素紊乱或代谢异常可引发心脑血管疾病、肾脏损伤、骨骼病变、神经病变等严重并发症。诊断需结合症状体征、激素水平检测、代谢指标评估、影像学检查及基因检测等手段，治疗以调节激素平衡、纠正代谢紊乱、控制病情进展、预防并发症为核心，同步配合饮食调理、运动干预及生活方式管理，实现个体化、长期化诊疗。

# 二、内分泌与代谢性疾病基础常识

## 2.1 疾病分类与核心特征

临床根据病变腺体及代谢异常类型对内分泌与代谢性疾病分类，核心类别及典型疾病特征如下，不同类型疾病的诊疗重点差异显著：

|病变类型/腺体|典型疾病|核心表现与特征|
|---|---|---|
|甲状腺疾病|甲状腺功能亢进症、甲状腺功能减退症、甲状腺结节、桥本甲状腺炎|甲亢伴心慌、手抖、消瘦、怕热多汗；甲减伴乏力、怕冷、体重增加、记忆力减退；甲状腺结节多无症状，部分需排查恶性风险|
|糖代谢疾病|1型糖尿病、2型糖尿病、糖尿病前期、糖尿病酮症酸中毒|典型症状为多饮、多食、多尿、体重减轻；长期血糖控制不佳引发周围神经、肾脏、眼底并发症；酮症酸中毒为急性并发症，伴昏迷、呕吐|
|脂代谢与骨代谢疾病|高脂血症、骨质疏松症、痛风、代谢综合征|高脂血症多无症状，易诱发心脑血管疾病；骨质疏松症伴骨痛、易骨折；痛风表现为关节突发红肿热痛，与尿酸代谢异常相关|
|其他内分泌疾病|垂体瘤、肾上腺疾病、多囊卵巢综合征|垂体瘤影响激素分泌，引发生长发育异常、月经紊乱；肾上腺疾病伴血压波动、电解质紊乱；多囊卵巢综合征表现为月经不调、痤疮、肥胖|
## 2.2 流行病学与高危人群

- **高发人群**：40岁以上中老年人，代谢功能衰退，患病风险显著升高；有内分泌与代谢性疾病家族史者（尤其是糖尿病、甲状腺疾病、痛风家族史）；肥胖、长期久坐、缺乏运动、饮食不规律者；长期精神压力大、熬夜、情绪波动剧烈者；女性更年期、妊娠期人群，激素水平波动易诱发疾病；长期服用激素药物、影响代谢的药物者。

- **人群与地域特征**：糖尿病、高脂血症在经济发达地区发病率较高，与饮食精细化、肥胖率高、运动不足相关；甲状腺疾病在女性中发病率高于男性，桥本甲状腺炎、甲状腺结节多见于中青年女性；痛风在沿海地区发病率较高，与高嘌呤饮食（海鲜、啤酒）摄入较多相关。

- **疾病负担**：此类疾病多为慢性疾病，需长期用药、监测指标，医疗费用较高；并发症可导致患者生活质量下降，甚至残疾、死亡，占用大量医疗资源，是我国重要的公共卫生问题之一。

# 三、内分泌与代谢性疾病病因与发病机制

## 3.1 核心病因

内分泌与代谢性疾病的发生是遗传因素、环境因素、激素调节异常及机体功能紊乱共同作用的结果，核心病因分类如下：

- **遗传因素**：糖尿病、甲状腺疾病、痛风、多囊卵巢综合征等均有明显家族遗传倾向，直系亲属患病者风险显著升高；部分罕见代谢病（如苯丙酮尿症、糖原累积病）由单一基因缺陷导致，属遗传性疾病；基因变异可影响激素分泌、受体功能或代谢酶活性，诱发疾病。

- **环境与生活方式因素**：肥胖、高油高脂高糖饮食、长期暴饮暴食，可诱发胰岛素抵抗，进而发展为糖尿病、代谢综合征；长期高嘌呤饮食、过量饮酒，可导致尿酸代谢异常，引发痛风；精神压力过大、熬夜、情绪波动，可影响下丘脑-垂体-靶腺轴功能，导致激素分泌紊乱，诱发甲状腺疾病、月经紊乱；长期缺乏运动，可加重代谢迟缓，增加患病风险。

- **激素与代谢调节异常**：内分泌腺体增生、肿瘤、炎症等，可导致激素分泌过多或不足（如甲亢为甲状腺激素分泌过多，甲减为分泌不足）；激素受体敏感性下降（如胰岛素抵抗），即使激素水平正常，也无法发挥正常生理作用；代谢酶缺陷、转运蛋白异常，可导致代谢底物堆积或产物缺乏，引发代谢紊乱。

- **其他诱因**：长期服用糖皮质激素、避孕药等药物，可干扰激素平衡，诱发血糖升高、血压波动；甲状腺、垂体等腺体手术切除后，激素分泌减少；感染、创伤、应激反应等，可诱发激素分泌紊乱，加重病情；女性妊娠期、更年期激素水平剧烈波动，易诱发甲状腺疾病、糖代谢异常。

## 3.2 核心发病机制

内分泌与代谢性疾病的核心发病机制可概括为激素分泌失衡、信号传导障碍、代谢途径异常三大类，不同疾病机制各有侧重，且相互关联。

激素分泌失衡是核心机制之一：内分泌腺体受下丘脑-垂体-靶腺轴调控，任何环节异常均可导致激素分泌紊乱。如甲亢由自身免疫因素导致甲状腺激素合成释放过多，作用于全身组织，加快代谢速率；垂体瘤可压迫腺体或异常分泌促激素，导致靶腺激素分泌异常；肾上腺肿瘤可过量分泌儿茶酚胺，引发血压骤升。

代谢异常机制以底物代谢障碍、胰岛素抵抗为核心：糖尿病的核心是胰岛素分泌不足或胰岛素抵抗，导致葡萄糖无法有效被组织利用，血糖升高，长期可损伤血管、神经；高脂血症是由于脂质合成过多、分解减少，导致血清胆固醇、甘油三酯升高，沉积于血管壁引发动脉粥样硬化；痛风是尿酸生成过多或排泄减少，尿酸盐结晶沉积于关节、肾脏，引发炎症反应。

此外，自身免疫功能紊乱在部分疾病中发挥关键作用：桥本甲状腺炎由自身抗体攻击甲状腺组织，导致甲状腺功能逐渐衰退；1型糖尿病由自身免疫损伤胰岛β细胞，导致胰岛素分泌绝对不足；自身免疫性肾上腺炎可导致肾上腺皮质激素分泌减少，引发乏力、电解质紊乱。

# 四、内分泌与代谢性疾病预防与健康管理

## 4.1 一级预防（病因预防）

针对普通人群及高危群体，通过综合干预减少致病因素，降低内分泌与代谢性疾病发病风险，核心措施如下：

- **生活方式干预**：保持规律作息，避免熬夜，保证每日7-8小时睡眠，维持激素分泌节律；适度运动，每周进行150分钟中等强度有氧运动（快走、慢跑、游泳），搭配力量训练，控制体重，改善代谢功能；调节情绪，缓解精神压力，避免长期焦虑、抑郁，减少激素紊乱诱因；戒烟限酒，避免过量饮酒诱发痛风、血糖波动，彻底戒烟减少血管损伤。

- **饮食调理**：规律饮食，三餐定时定量，避免暴饮暴食；均衡膳食，减少高油高脂、高糖、高嘌呤食物摄入，增加新鲜蔬菜、水果、全谷物、优质蛋白摄入；控制总热量摄入，BMI维持在18.5-23.9之间，预防肥胖；痛风患者严格限制海鲜、动物内脏、啤酒等高嘌呤食物，多喝水促进尿酸排泄。

- **病因防控**：有家族病史者，提前建立健康档案，加强定期筛查，早发现早干预；避免长期滥用激素药物，用药需遵医嘱，定期监测激素水平；女性妊娠期、更年期定期体检，监测甲状腺功能、血糖等指标，及时发现异常；积极治疗感染、腺体炎症等基础疾病，避免诱发激素紊乱。

## 4.2 二级预防（早筛查、早干预）

- **定期筛查**：40岁以上人群每年进行血糖、血脂、甲状腺功能（TSH、T3、T4）、尿酸筛查；有家族病史、肥胖、高血压等高危因素者，每6个月复查一次血糖、甲状腺功能；女性多囊卵巢综合征高危人群，筛查性激素六项、血糖血脂；甲状腺结节患者定期复查甲状腺超声，评估结节性质。

- **症状监测与早期干预**：警惕疾病早期信号，如心慌手抖、怕热怕冷、体重异常变化、多饮多食多尿、关节红肿热痛、月经紊乱、乏力嗜睡等，及时就医排查；确诊糖尿病前期、甲状腺功能亚临床异常、高脂血症等早期疾病，立即通过饮食、运动干预，必要时药物治疗，阻断病情进展。

- **慢病管理前置**：糖尿病前期人群通过控制饮食、增加运动，定期监测血糖，逆转或延缓发展为糖尿病；高脂血症患者通过生活方式干预，必要时服用调脂药，控制血脂水平；甲状腺结节良性者定期随访，避免过度治疗，恶性者及时手术。

## 4.3 三级预防（慢病管理与并发症防治）

- **规范治疗**：慢性内分泌与代谢性疾病患者严格遵医嘱规律用药，不可自行停药、减药，掌握药物用法及不良反应（如降糖药可能引发低血糖，甲状腺素片需逐步调整剂量）；定期监测相关指标（血糖、血脂、甲状腺功能、尿酸），根据结果调整治疗方案，维持指标稳定。

- **康复与生活管理**：糖尿病患者严格控制饮食，规律运动，做好血糖自我监测，预防低血糖；痛风患者避免诱发因素，急性发作期卧床休息，缓解期适度运动；骨质疏松症患者补充钙及维生素D，避免剧烈运动，预防骨折；术后患者遵医嘱复查激素水平，逐步调整药物剂量，恢复腺体功能。

- **并发症防治**：早期筛查并发症，糖尿病患者定期检查眼底、肾功能、神经功能，预防眼底病变、肾病、神经病变；高脂血症患者监测血压、血管超声，预防动脉粥样硬化、冠心病；甲亢患者控制心率，避免劳累，预防甲亢危象；出现并发症及时干预，延缓病情进展，降低残疾风险。

# 五、内分泌与代谢性疾病诊断方法

## 5.1 临床诊断（初步筛查）

结合患者病史、症状、体格检查及流行病学史进行初步判断，为后续辅助检查提供方向，适用于基层医疗机构首诊及社区筛查。

诊断要点：① 病史采集：重点询问家族病史、饮食史、运动史、用药史、月经史（女性），以及症状特点（症状出现时间、持续规律、伴随症状如体重变化、心慌、关节痛等）；② 体格检查：测量身高、体重、血压、心率，检查甲状腺是否肿大、有无结节，评估关节红肿情况、精神状态，初步判断病变类型及严重程度。

## 5.2 实验室与影像学诊断（辅助确诊与病情评估）

|诊断方法|检测目标/用途|结果解读与适用场景|注意事项|
|---|---|---|---|
|激素与代谢指标检测|评估激素水平、代谢功能|甲状腺功能（TSH、T3、T4）诊断甲亢、甲减；血糖（空腹、餐后2小时）、糖化血红蛋白诊断糖尿病；血脂四项（总胆固醇、甘油三酯等）诊断高脂血症；尿酸检测诊断痛风；性激素六项评估生殖内分泌功能。|需空腹采血（血糖、血脂、甲状腺功能）；糖化血红蛋白不受进食影响；激素检测需避免剧烈运动、情绪激动，部分需经期后采集。|
|自身抗体检测|排查自身免疫性内分泌疾病|甲状腺自身抗体（TPOAb、TgAb）诊断桥本甲状腺炎；胰岛自身抗体（GADAb）鉴别1型与2型糖尿病；肾上腺自身抗体诊断自身免疫性肾上腺炎。|采血前无需特殊准备，抗体阳性需结合激素水平、症状综合判断，不可单独作为诊断依据。|
|影像学检查|评估内分泌腺体形态、结构|甲状腺超声筛查甲状腺结节、肿大；垂体MRI排查垂体瘤；肾上腺CT评估肾上腺肿瘤、增生；骨密度检测诊断骨质疏松症。|甲状腺超声无需空腹；增强MRI/CT需告知过敏史，检查后多喝水促进造影剂排泄；骨密度检测无创伤，无需特殊准备。|
|其他特殊检查|明确诊断与病情分期|葡萄糖耐量试验诊断糖尿病前期；甲状腺细针穿刺活检明确甲状腺结节良恶性；基因检测诊断遗传性代谢病。|葡萄糖耐量试验需空腹，服糖后定时采血；穿刺活检后按压止血，预防出血；基因检测需结合家族史，结果需专业解读。|
# 六、内分泌与代谢性疾病治疗方案

## 6.1 治疗原则

内分泌与代谢性疾病治疗以“调节激素平衡、纠正代谢紊乱、去除病因、控制症状、预防并发症、改善预后”为核心。激素分泌过多者以抑制分泌、切除病灶为主；激素分泌不足者以替代治疗为主；代谢异常者以饮食、运动干预为基础，结合药物调节代谢指标；自身免疫性疾病以控制免疫反应、保护腺体功能为主；同时重视长期管理，根据病情变化调整治疗方案，兼顾患者生活质量。

## 6.2 药物治疗（基础干预手段）

药物治疗适用于多数内分泌与代谢性疾病，需根据疾病类型、激素水平及合并症选择，核心药物类别及应用如下：

|药物类别|代表药物|适用场景|不良反应与注意事项|
|---|---|---|---|
|降糖药物|二甲双胍、格列齐特、胰岛素、SGLT-2抑制剂|1型、2型糖尿病，控制血糖水平，延缓并发症|二甲双胍可能引起胃肠道反应；格列齐特、胰岛素可能诱发低血糖；SGLT-2抑制剂需监测泌尿感染、脱水风险。|
|甲状腺疾病药物|甲巯咪唑、丙硫氧嘧啶、左甲状腺素钠|甲亢（抑制甲状腺激素合成）、甲减（激素替代治疗）|甲巯咪唑可能导致粒细胞减少；左甲状腺素钠需逐步调量，过量引发甲亢症状，定期监测甲状腺功能。|
|调脂与降尿酸药物|阿托伐他汀、瑞舒伐他汀、别嘌醇、非布司他|高脂血症（调节血脂）、痛风（降低尿酸水平）|他汀类药物需监测肝功能、肌酸激酶；别嘌醇可能引发过敏反应；非布司他需监测肝功能，痛风急性发作期禁用。|
|激素替代药物|糖皮质激素、雌激素、生长激素|肾上腺皮质功能减退、更年期综合征、生长激素缺乏症|长期服用糖皮质激素可能引发骨质疏松、血糖升高；雌激素需排除乳腺、子宫肿瘤风险，定期复查。|
## 6.3 手术与介入治疗（复杂病例干预）

- **手术治疗**：适用于内分泌腺体肿瘤、结节恶变、腺体增生过度等。甲状腺结节恶性者行甲状腺切除术；垂体瘤压迫症状明显或激素分泌异常者行垂体瘤切除术；肾上腺肿瘤、胰岛细胞瘤行肿瘤切除术；甲状腺功能亢进药物治疗无效者，可行甲状腺次全切除术。

- **介入与其他治疗**：甲状腺结节良性但体积较大者，可行射频消融术；糖尿病患者血糖控制不佳，可行胰岛素泵持续输注治疗；骨质疏松症患者可行椎体成形术，治疗椎体骨折；痛风石较大者，可行手术剔除，缓解关节压迫。

## 6.4 急性并发症急救与对症支持治疗

- **糖尿病酮症酸中毒急救**：立即补液，纠正脱水与电解质紊乱；小剂量胰岛素静脉输注，降低血糖；纠正酸中毒，监测血气分析；抗感染治疗，去除诱因（如感染）；密切监测血糖、电解质、尿量，预防休克。

- **甲亢危象急救**：卧床休息，物理降温；静脉输注丙硫氧嘧啶，抑制甲状腺激素合成；使用β受体阻滞剂，控制心率；补充液体与电解质，纠正紊乱；去除诱因（如感染、手术应激），转入ICU监护。

- **对症支持治疗**：痛风急性发作期，给予秋水仙碱、非甾体抗炎药，缓解关节疼痛；低血糖发作时，立即补充糖分（口服葡萄糖或静脉输注），纠正低血糖；骨质疏松症患者补充钙及维生素D，配合抗骨质疏松药物，预防骨折；代谢综合征患者综合控制血压、血糖、血脂，改善代谢紊乱。

## 7.2 转诊指征

出现以下情况，需及时转诊至上级医院内分泌科或ICU治疗：

- 突发糖尿病酮症酸中毒、甲亢危象、严重低血糖昏迷，病情危急，需紧急抢救；

- 影像学检查提示内分泌腺体占位性病变（如垂体瘤、肾上腺肿瘤），疑似恶性，需进一步病理活检、分期检查明确治疗方案；

- 慢性疾病药物治疗效果不佳，指标持续异常（如血糖、甲状腺功能、尿酸控制不佳），或出现严重并发症（如糖尿病肾病、眼底病变、甲状腺危象前兆）；

- 不明原因体重骤降、激素水平显著异常、关节反复红肿热痛，基层医疗机构无法明确诊断；

- 需行手术治疗（如甲状腺切除术、垂体瘤切除术）、介入治疗（如射频消融），基层医疗机构无法开展；

- 儿童、青少年生长发育异常（如矮小、性早熟），疑似内分泌紊乱，需专科评估病因。
