# 上呼吸道感染医疗知识库

# 一、上呼吸道感染概述

上呼吸道感染（upper respiratory tract infection，URTI）简称上感，是鼻腔、咽或喉部急性炎症的总称，并非单一疾病，而是一组疾病的统称，包括普通感冒、病毒性咽炎、喉炎、疱疹性咽峡炎、咽结膜热、细菌性咽-扁桃体炎等。本病为临床最常见的呼吸系统感染性疾病，发病率高，全年均可发病，冬春季更为高发，具有自限性，多数患者预后良好，少数年老体弱、免疫功能低下者可能出现严重并发症，危及健康。

上感的核心致病原为病毒，占比70%-80%，少数由细菌感染引起，传播途径以呼吸道飞沫传播为主，接触被病原体污染的物品也可间接传播。儿童、老年人、免疫功能低下者及过度疲劳、受凉人群为高发群体，临床症状多表现为鼻塞、流涕、咽痛、咳嗽、低热等，因疾病类型和个体差异有所不同。

# 二、上呼吸道感染基础常识

## 2.1 病原学与病理特征

病原学方面，病毒感染是上感的主要诱因，常见病毒包括鼻病毒、流感病毒、副流感病毒、呼吸道合胞病毒、腺病毒、柯萨奇病毒等，其中鼻病毒是普通感冒的最主要致病菌。细菌感染占比20%-30%，以溶血性链球菌最为常见，其次为流感嗜血杆菌、肺炎球菌、葡萄球菌等，多在病毒感染后因机体抵抗力下降而继发感染。

病理变化主要表现为病变部位黏膜充血、水肿、渗出，上皮细胞变性、坏死、脱落，伴随淋巴细胞、中性粒细胞浸润等炎症反应。不同类型上感的病变部位有所侧重，如普通感冒以鼻腔、鼻咽部黏膜炎症为主，细菌性咽-扁桃体炎则聚焦于咽部及扁桃体的化脓性炎症。

## 2.2 流行病学特征

- **高发人群**：人群普遍易感，其中5岁以下儿童、65岁以上老年人、免疫功能低下者（如艾滋病患者、长期使用免疫抑制剂者）、患有慢性基础疾病者（如慢性阻塞性肺疾病、糖尿病、心血管疾病）及过度疲劳、受凉人群为高危群体，感染风险显著高于普通人群。

- **传播途径**：主要通过呼吸道飞沫传播，患者咳嗽、打喷嚏时产生的带菌飞沫被他人吸入后引发感染；同时可通过接触传播，如触摸被病原体污染的门把手、桌面、餐具等，再接触口鼻导致感染。

- **季节与环境因素**：冬春季气温变化大、室内通风不良，利于病原体滋生传播，为高发季节；夏季可因游泳等接触传播引发咽结膜热、疱疹性咽峡炎等类型上感，儿童多见。人员密集场所（如学校、托幼机构、养老院）易发生聚集性疫情。

# 三、上呼吸道感染病因与发病机制

## 3.1 核心病因

上感的发生是病原体入侵与机体防御功能下降共同作用的结果，具体病因如下：

- **病原体入侵**：病毒或细菌通过呼吸道飞沫、接触传播等途径侵入上呼吸道黏膜，在局部繁殖生长，引发炎症反应。病毒感染多为原发性，细菌感染则多为继发性，常发生在病毒感染后，此时黏膜屏障受损，细菌易趁虚而入。

- **机体防御功能下降**：受凉、淋雨、过度疲劳、熬夜、营养不良等因素，会导致全身或呼吸道局部防御功能降低，使原本存在于上呼吸道的条件致病菌大量繁殖，或对外部入侵病原体的抵抗力减弱，从而诱发感染。

- **环境与行为因素**：长期处于空气污染、粉尘、有害气体环境中，会损伤呼吸道黏膜，增加感染风险；人群密集、通风不良的场所，会提高病原体传播效率，加速疫情扩散。

## 3.2 发病机制

当机体防御功能下降时，病毒或细菌侵入上呼吸道后，迅速黏附于黏膜上皮细胞表面并繁殖，引发局部炎症反应。病毒感染可直接损伤上皮细胞，导致黏膜充血水肿、分泌物增多，出现鼻塞、流涕等卡他症状；细菌感染则通过释放毒素刺激黏膜，引发化脓性炎症，表现为咽痛、扁桃体肿大伴脓性分泌物。

同时，机体免疫系统被激活，淋巴细胞、中性粒细胞等炎症细胞聚集于病变部位，清除病原体的过程中释放炎症因子，可引起发热、头痛、全身酸痛等全身症状。部分患者因炎症扩散或免疫反应异常，可能引发鼻窦炎、中耳炎、心肌炎等并发症。

# 四、上呼吸道感染预防措施

## 4.1 一级预防（病因预防）

- **增强机体抵抗力**：合理饮食，保证蛋白质、维生素摄入，多吃新鲜蔬菜、水果及优质蛋白，避免挑食偏食；规律作息，保证充足睡眠，避免熬夜；适度进行体育锻炼，如快走、慢跑、游泳等，增强体质，提高呼吸道防御能力。

- **做好个人防护**：勤洗手，采用七步洗手法，用肥皂或洗手液配合流动水清洁，持续不少于20秒，无洗手条件时使用含75%酒精的免洗洗手液；咳嗽、打喷嚏时用纸巾或肘部遮挡口鼻，避免飞沫传播；高发季节减少前往人员密集场所，必要时佩戴医用外科口罩。

- **优化环境与生活习惯**：保持室内通风换气，每日至少2-3次，每次15-30分钟，维持室内温度22-26℃、湿度40%-60%；定期清洁消毒门把手、桌面、手机等高频接触物品，可用含氯消毒剂或75%酒精擦拭；避免受凉、淋雨、过度疲劳，根据气温变化及时增减衣物。

## 4.2 二级预防（早发现、早诊断、早治疗）

- **症状监测**：密切关注自身及家人健康状况，出现鼻塞、流涕、咽痛、低热等症状时，及时居家休息，避免接触他人，减少传播风险；症状持续不缓解或加重时，及时就医排查，避免延误并发症诊治。

- **高危人群防护**：儿童、老年人、免疫功能低下者等高危群体，高发季节可接种流感疫苗，减少感染风险；经常反复感染者，可在医生指导下酌情应用卡介苗素细菌溶解物等，增强呼吸道抵抗力。

- **聚集性疫情防控**：学校、托幼机构等集体场所，加强晨午检制度，及时发现发热、咽痛症状者并隔离；定期开展环境消毒，避免聚集性疫情发生。

# 五、上呼吸道感染诊断方法

## 5.1 临床诊断（初步筛查）

结合病史、流行病学史、临床表现及体格检查进行初步判断，为临床诊疗提供方向，适用于基层医疗机构首诊。

诊断要点：① 病史采集：重点询问发病诱因（受凉、疲劳、接触史）、症状特点（鼻塞、流涕、咽痛、发热等出现时间及程度）、既往病史及基础疾病；② 体格检查：观察鼻腔黏膜充血水肿及分泌物情况，检查咽部充血程度、扁桃体是否肿大化脓，触摸颌下淋巴结有无肿大压痛，肺部听诊多无异常体征，若存在上气道梗阻可闻及喉部喘鸣音。

# 六、上呼吸道感染治疗方案

## 6.1 治疗原则

上感治疗以对症治疗为主，无需常规使用抗病毒药物，合并细菌感染时酌情使用抗生素；同时注重休息、多饮水、保持室内空气流通，避免并发症发生。治疗核心是缓解症状、缩短病程、增强机体抵抗力，促进自愈。

## 6.2 对症治疗

- **解热镇痛**：发热、头痛、全身肌肉酸痛者，可选用对乙酰氨基酚、布洛芬等药物，儿童及哮喘病史者忌用阿司匹林；低热者优先物理降温，用32-34℃温水擦拭颈部、腋下等部位，避免酒精擦浴。

- **缓解鼻喉症状**：鼻塞、鼻黏膜充血者，可用盐酸伪麻黄碱滴鼻剂收缩血管；频繁喷嚏、流涕者，可酌情选用马来酸氯苯那敏等抗过敏药物，宜睡前服用以减轻嗜睡不良反应；咽痛者可使用含漱液、润喉片缓解，咳嗽明显者给予右美沙芬、喷托维林等镇咳药。

- **补液与休息**：保证每日1500-2000毫升水分摄入，饮用温开水、米汤等，避免脱水；发热、病情较重者卧床休息，减少体力消耗，促进恢复。

## 6.3 抗感染治疗

- **抗病毒治疗**：无特效抗病毒药物，一般无需常规使用；免疫缺陷患者可早期选用利巴韦林、奥司他韦等，流感病毒感染者可早期使用奥司他韦缩短病程。

- **抗生素治疗**：普通上感无需使用抗生素，仅当明确合并细菌感染时选用，如咽部脓苔、咳黄痰、白细胞及CRP升高者。可选用青霉素类、头孢菌素类、大环内酯类抗生素，耐药病例可酌情使用呼吸氟喹诺酮类（左氧氟沙星、莫西沙星），疗程根据病情调整为3-7天。

## 6.4 中医与辅助治疗

中医治疗需辨证论治，风热袭肺者可选用复方瓜子金颗粒，清热利咽、祛痰止咳；小儿上感咳嗽者可使用小儿咳喘灵颗粒，宣肺清热、止咳祛痰。同时可配合饮食调理，选择稀软清淡、易消化食物，避免辛辣、油腻、生冷食物及浓茶、蜂蜜，减轻肠胃负担。

# 七、鉴别诊断与并发症管理

## 7.1 常见疾病鉴别诊断

|鉴别疾病|核心鉴别点|
|---|---|
|过敏性鼻炎|起病急骤，由过敏因素诱发，表现为突发喷嚏、清水样鼻涕、鼻痒，无发热，脱离过敏原后数分钟至2小时症状缓解，鼻黏膜苍白水肿，分泌物涂片可见嗜酸性粒细胞增多。|
|流行性感冒|由流感病毒引起，传染性强，全身症状重（高热、全身酸痛、眼结膜炎），鼻咽部症状轻，病毒检测可明确诊断，早期使用抗流感药物有效。|

## 7.2 常见并发症及处理

上感若治疗不及时或病情较重，可能引发以下并发症，需及时干预：

- **急性鼻窦炎**：上感3-5日后症状加重，出现脓性鼻涕、发热、头痛，鼻窦部位压痛，需选用敏感抗生素治疗，配合鼻腔冲洗缓解症状。

- **急性中耳炎**：咽鼓管蔓延感染所致，表现为耳痛、发热、耳流脓、听力减退，需抗感染治疗，局部清理脓液，避免听力损伤。

- **心肌炎、肾小球肾炎**：少数链球菌或病毒感染可诱发，心肌炎表现为低热、心悸、乏力，肾小球肾炎表现为蛋白尿、血尿、水肿，需及时就医，针对性治疗，避免病情迁延。

## 7.3 转诊指征

出现以下情况，需及时转诊至上级医院治疗：

- 持续高热（>39℃），经3天常规治疗无效，或反复发热超过1周；

- 存在上气道梗阻，出现喘息、吸气性三凹征，有窒息风险；

- 出现心肌炎、肾小球肾炎、肺炎等严重并发症；

- 年老体弱、有严重基础疾病或免疫功能低下者，病情进展迅速；

- 短时间内出现呼吸、循环系统衰竭症状及体征。
