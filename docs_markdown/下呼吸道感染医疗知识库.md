# 下呼吸道感染医疗知识库

# 一、下呼吸道感染概述

下呼吸道感染（lower respiratory tract infection，LRTI）是指累及气管、支气管、细支气管、肺泡及肺间质的感染性疾病总称，是临床常见的呼吸系统重症，发病率仅次于上呼吸道感染，但其病情较重、并发症多，对儿童、老年人及免疫功能低下者的健康威胁更大。本病全年均可发病，冬春季高发，致病原以病毒、细菌、支原体、衣原体为主，临床主要表现为咳嗽、咳痰、喘息、呼吸困难、发热等，病情严重程度因感染部位、致病原及患者体质差异较大，部分重症病例可进展为呼吸衰竭、感染性休克，甚至危及生命。

下呼吸道感染按病程可分为急性和慢性，急性下呼吸道感染包括急性支气管炎、支气管肺炎、毛细支气管炎等，病程较短，经规范治疗后多可痊愈；慢性下呼吸道感染则以慢性支气管炎、慢性阻塞性肺疾病急性加重、支气管扩张合并感染等为主，病程迁延反复，需长期管理干预。早期精准诊断、合理抗感染治疗及全程护理是改善预后的关键。

# 二、下呼吸道感染基础常识

## 2.1 病原学与病理特征

病原学方面，下呼吸道感染的致病原复杂多样，且与患者年龄、基础疾病密切相关。病毒感染常见于儿童及青壮年，主要包括呼吸道合胞病毒、流感病毒、副流感病毒、腺病毒、鼻病毒等，其中呼吸道合胞病毒是婴幼儿毛细支气管炎的首要致病菌；细菌感染以肺炎链球菌、金黄色葡萄球菌、流感嗜血杆菌、铜绿假单胞菌、大肠杆菌等为主，肺炎链球菌是成人社区获得性肺炎的最常见致病菌，铜绿假单胞菌多见于医院获得性肺炎及支气管扩张患者；非典型病原体（支原体、衣原体）感染在学龄期儿童及青壮年中高发，肺炎支原体是引起间质性肺炎的主要原因之一。

病理特征上，下呼吸道感染主要表现为病变部位黏膜充血水肿、分泌物增多、上皮细胞损伤脱落，伴随炎症细胞浸润及组织渗出。急性支气管炎以支气管黏膜炎症为主，肺泡受累较轻；肺炎则累及肺实质，肺泡内出现浆液性、纤维素性渗出物，形成实变；毛细支气管炎多见于婴幼儿，因细支气管黏膜水肿、分泌物堵塞，导致气道狭窄，引发喘息及呼吸困难；慢性感染则伴随气道结构重塑、纤维组织增生，甚至肺泡破坏，形成不可逆损伤。

## 2.2 流行病学特征

- **高发人群**：人群普遍易感，高危群体包括5岁以下婴幼儿、65岁以上老年人；免疫功能低下者（如艾滋病患者、接受化疗或器官移植者、长期使用免疫抑制剂者）；患有慢性基础疾病者（慢性阻塞性肺疾病、支气管扩张、糖尿病、心血管疾病、营养不良等）；长期吸烟、酗酒者及长期暴露于粉尘、有害气体环境的职业人群。

- **传播途径**：主要通过呼吸道飞沫传播，患者咳嗽、打喷嚏时产生的带菌飞沫被他人吸入后引发感染；部分病原体可通过接触传播，如触摸被污染的物品后接触口鼻；医院获得性肺炎还可通过医护人员手卫生不当、呼吸机管路污染等途径传播。

- **季节与环境因素**：冬春季气温骤变、室内通风不良，利于病原体滋生传播，是急性下呼吸道感染高发季节；夏季高温高湿环境下，真菌性下呼吸道感染风险增加；雾霾、沙尘天气及室内空气污染可刺激呼吸道黏膜，降低防御能力，诱发或加重感染；集体场所（学校、托幼机构、养老院、医院）易发生聚集性疫情。

# 三、下呼吸道感染病因与发病机制

## 3.1 核心病因

下呼吸道感染的发生是致病原入侵、机体防御功能下降及环境因素共同作用的结果，具体病因如下：

- **致病原入侵**：病毒、细菌、支原体等病原体通过呼吸道飞沫、接触传播等途径侵入下呼吸道，在气管、支气管、肺泡内黏附、繁殖，释放毒素或引发免疫反应，导致局部炎症病变。部分细菌可在病毒感染基础上继发感染，因病毒感染导致黏膜屏障受损，细菌易趁虚而入，加重炎症反应。

- **机体防御功能下降**：受凉、淋雨、过度疲劳、熬夜、营养不良等因素，会导致全身免疫功能降低；长期吸烟、慢性肺部疾病会损伤呼吸道黏膜纤毛运动功能，削弱分泌物清除能力，使病原体易在局部定植繁殖；免疫缺陷患者因免疫细胞功能异常，无法有效清除病原体，易发生严重感染且病情迁延。

- **环境与医疗因素**：长期暴露于粉尘、烟雾、有害气体（如甲醛、油烟）中，会损伤呼吸道黏膜，增加感染风险；医院获得性肺炎与侵入性操作（如气管插管、呼吸机辅助呼吸）、广谱抗生素滥用、医护人员手卫生不佳等因素密切相关，易引发多重耐药菌感染。

## 3.2 发病机制

当机体防御功能下降时，致病原侵入下呼吸道后，迅速黏附于呼吸道上皮细胞表面并繁殖，引发局部炎症反应。病毒感染可直接损伤上皮细胞，导致黏膜充血水肿、分泌物增多，堵塞气道，引发咳嗽、喘息；细菌感染则通过释放毒素刺激黏膜，引发化脓性炎症，肺泡内渗出物增多，影响气体交换，导致低氧血症；支原体、衣原体感染则主要侵犯肺间质，引发间质性炎症，表现为刺激性干咳。

机体免疫系统被激活后，中性粒细胞、淋巴细胞、巨噬细胞等炎症细胞聚集于病变部位，清除病原体的过程中释放大量炎症因子，可引发全身性炎症反应，出现发热、全身酸痛等症状。重症病例因炎症反应过度激活，可能引发“细胞因子风暴”，导致肺部严重损伤、急性呼吸窘迫综合征，甚至多器官功能衰竭；慢性感染则因炎症反复刺激，导致气道结构重塑、肺功能下降，形成慢性病程。

# 四、下呼吸道感染预防措施

## 4.1 一级预防（病因预防）

- **增强机体抵抗力**：合理饮食，保证蛋白质、维生素、矿物质摄入，多吃新鲜蔬菜、水果、优质蛋白及富含膳食纤维的食物，避免挑食偏食；规律作息，保证充足睡眠，避免熬夜；适度进行体育锻炼，如快走、慢跑、游泳、呼吸操等，增强体质，提高呼吸道防御能力；戒烟限酒，避免损伤呼吸道黏膜及免疫功能。

- **做好个人与环境防护**：勤洗手，采用七步洗手法，用肥皂或洗手液配合流动水清洁，持续不少于20秒，无洗手条件时使用含75%酒精的免洗洗手液；咳嗽、打喷嚏时用纸巾或肘部遮挡口鼻，避免飞沫传播；高发季节减少前往人员密集、通风不良场所，必要时佩戴医用外科口罩；保持室内通风换气，每日2-3次，每次30分钟以上，定期清洁消毒高频接触物品（门把手、桌面、手机等）。

- **高危人群专项防护**：儿童、老年人、慢性肺部疾病患者等高危群体，每年冬春季前接种流感疫苗、肺炎球菌疫苗，降低感染风险；免疫缺陷患者可在医生指导下使用免疫增强剂，增强抵抗力；长期卧床患者定期翻身、拍背，促进痰液排出，避免坠积性肺炎。

## 4.2 二级预防（早发现、早诊断、早治疗）

- **症状监测与早期干预**：密切关注自身及家人健康状况，出现持续咳嗽、咳痰、喘息、低热等症状时，及时居家休息，避免劳累，多喝水，症状无缓解或加重（如出现高热、呼吸困难）时，及时就医排查，避免延误病情；慢性肺部疾病患者需定期复诊，遵医嘱规律用药，减少急性发作风险。

- **医院感染防控**：医疗机构加强医护人员手卫生管理，严格执行无菌操作，减少侵入性操作；合理使用广谱抗生素，避免多重耐药菌产生；对呼吸机管路、吸痰装置等医疗设备严格消毒，定期更换；住院患者加强翻身、拍背、雾化吸入等护理，预防坠积性肺炎。

- **聚集性疫情防控**：学校、托幼机构、养老院等集体场所，加强晨午检制度，及时发现发热、咳嗽症状者并隔离；定期开展环境消毒与通风，避免聚集性疫情发生；教职工、护理人员做好个人防护，降低交叉感染风险。

# 五、下呼吸道感染诊断方法

## 5.1 临床诊断（初步筛查）

结合病史、流行病学史、临床表现及体格检查进行初步判断，为后续诊疗提供方向，适用于基层医疗机构首诊及门诊筛查。

诊断要点：① 病史采集：重点询问发病诱因（受凉、疲劳、接触史、住院史、侵入性操作史）、症状特点（咳嗽性质、咳痰性状、喘息程度、发热规律、呼吸困难情况）、既往病史（慢性肺部疾病、基础疾病）及用药史；② 体格检查：观察呼吸频率、节律、口唇有无发绀，听诊肺部呼吸音、干湿性啰音、哮鸣音，慢性感染患者可能出现杵状指，重症患者可见吸气性三凹征。

## 5.2 实验室与影像学诊断（辅助确诊与鉴别）

|诊断方法|检测目标/用途|结果解读与适用场景|注意事项|
|---|---|---|---|
|血常规+炎症标志物（CRP、PCT）|判断感染类型（病毒/细菌/非典型），评估感染严重程度|细菌感染：白细胞及中性粒细胞升高、CRP及PCT显著升高；病毒感染：白细胞正常或偏低、淋巴细胞升高，CRP轻度升高；非典型感染：炎症标志物多中度升高。适用于急性下呼吸道感染初步分型。|无需空腹，采血前避免剧烈运动，结果需结合临床症状综合判断，避免单一指标误判。|

# 六、下呼吸道感染治疗方案

## 6.1 治疗原则

下呼吸道感染治疗以“抗感染治疗为核心、对症支持治疗为辅助、并发症防治为重点”，遵循“早期经验性治疗+后期精准调整”原则。急性感染需及时控制致病原，缓解症状，防止病情进展；慢性感染需长期管理，控制炎症，改善肺功能，减少急性发作；重症感染需加强器官功能支持，积极防治呼吸衰竭、感染性休克等并发症。

## 6.2 抗感染治疗

- **细菌感染**：社区获得性肺炎首选青霉素类（阿莫西林）、头孢菌素类（头孢呋辛、头孢地尼）、大环内酯类（阿奇霉素）抗生素；医院获得性肺炎、多重耐药菌感染可选用碳青霉烯类（美罗培南）、呼吸氟喹诺酮类（左氧氟沙星、莫西沙星）、头孢哌酮舒巴坦等，具体根据痰培养药敏结果调整。疗程一般7-14天，重症感染可延长至21天，确保病原体彻底清除。

- **病毒感染**：无特效抗病毒药物，一般以对症支持治疗为主；流感病毒感染可早期（发病48小时内）使用奥司他韦、扎那米韦等抗流感药物；呼吸道合胞病毒感染多见于婴幼儿，可使用利巴韦林雾化吸入治疗，免疫缺陷患者可早期静脉使用抗病毒药物。

- **非典型病原体感染**：支原体、衣原体感染首选大环内酯类抗生素（阿奇霉素、红霉素），成人及18岁以上青少年可选用呼吸氟喹诺酮类药物，疗程10-14天，避免过早停药导致病情反复。

## 6.3 对症支持治疗

- **止咳祛痰**：痰多者以祛痰为主，选用氨溴索、溴己新、乙酰半胱氨酸等药物，或雾化吸入祛痰药，促进痰液排出；无痰干咳者可选用右美沙芬、喷托维林等镇咳药，避免强效镇咳药抑制排痰。

- **平喘解痉**：伴有喘息、气道痉挛者，使用支气管舒张剂（沙丁胺醇、异丙托溴铵）雾化吸入或口服，重症患者可联合吸入性糖皮质激素（布地奈德），缓解气道痉挛，改善通气。

- **氧疗与呼吸支持**：低氧血症患者给予鼻导管或面罩吸氧，维持血氧饱和度≥92%；重症患者出现呼吸衰竭时，及时给予无创通气，必要时气管插管、有创机械通气，严重者可采用体外膜肺氧合（ECMO）支持。

- **补液与营养支持**：保证水分摄入，避免脱水，呕吐、腹泻严重者给予静脉补液，维持水电解质平衡；给予清淡易消化、营养丰富的食物，重症患者无法进食时，给予肠内或肠外营养支持，增强机体抵抗力。

## 6.4 重症感染与并发症治疗

重症下呼吸道感染需转入ICU综合救治，除强化抗感染治疗外，还需采取以下措施：① 控制全身性炎症反应，必要时在医生指导下使用糖皮质激素，避免滥用；② 防治感染性休克，给予补液扩容、血管活性药物，维持血压稳定；③ 治疗急性呼吸窘迫综合征，采用肺保护性通气策略，必要时俯卧位通气；④ 积极治疗胸腔积液、肺脓肿、心肌炎等并发症，针对性干预，降低病死率。

# 七、鉴别诊断与转诊指征

## 7.1 常见疾病鉴别诊断

|鉴别疾病|核心鉴别点|
|---|---|
|上呼吸道感染|以鼻塞、流涕、咽痛等上呼吸道症状为主，咳嗽较轻，肺部无啰音，胸部影像学无肺实质病变；下呼吸道感染以咳嗽、咳痰、喘息为核心，肺部可闻及干湿性啰音，影像学可见肺实质或支气管病变。|

## 7.2 转诊指征

出现以下情况，需及时转诊至上级医院或ICU治疗：

- 持续高热（>39℃）超过3天，经经验性抗感染治疗无效，或反复发热超过1周；

- 出现呼吸困难、喘息明显，口唇发绀，呼吸频率加快（成人≥30次/分钟，儿童≥50次/分钟），血氧饱和度持续低于92%；

- 意识模糊、嗜睡、烦躁不安，或出现血压下降、四肢冰凉、脉搏微弱等感染性休克表现；

- 胸部影像学提示大面积肺实变、多肺叶受累，或合并胸腔积液、肺脓肿、呼吸衰竭；

- 年老体弱、有严重基础疾病或免疫功能低下者，病情进展迅速，常规治疗效果不佳；

- 医院获得性感染、多重耐药菌感染，或合并多器官功能损伤。
