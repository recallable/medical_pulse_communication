# 甲状腺疾病医疗知识库

# 一、甲状腺疾病概述

甲状腺疾病是一组累及甲状腺腺体的内分泌疾病总称，甲状腺作为人体重要的内分泌器官，主要分泌甲状腺激素（T3、T4），调控机体新陈代谢、生长发育、能量代谢及器官功能。此类疾病病因复杂，核心与甲状腺激素分泌异常、自身免疫功能紊乱、遗传因素、环境诱因相关，临床症状涉及全身多系统，表现多样。

甲状腺疾病发病率呈逐年上升趋势，我国成人患病率超10%，其中女性发病率显著高于男性（约3:1），且发病年龄覆盖全人群。多数甲状腺疾病呈慢性病程，早期症状隐匿，易被忽视，长期激素紊乱可引发心脏损伤、骨骼病变、神经功能异常、生殖系统紊乱等并发症，部分甲状腺结节存在恶性变风险，需及时筛查干预。甲状腺疾病需结合激素水平、影像学检查综合诊断，治疗以调节激素平衡、去除病灶、控制并发症为核心，实现个体化管理。

# 二、甲状腺疾病基础常识

## 2.1 疾病分型与核心特征

临床根据甲状腺功能状态、病因及病变性质，将甲状腺疾病主要分为功能异常型、结构异常型及自身免疫型，核心分型及特征如下：

|分型类别|典型疾病|核心病因|核心特征|
|---|---|---|---|
|甲状腺功能亢进症|格雷夫斯病、毒性结节性甲状腺肿|甲状腺激素分泌过多，格雷夫斯病为自身免疫性病因|代谢加快，表现为心慌、手抖、怕热多汗、消瘦、易怒、眼球突出|
|甲状腺功能减退症|桥本甲状腺炎、术后甲减、药物性甲减|甲状腺激素分泌不足，桥本甲状腺炎为自身免疫性损伤|代谢减慢，表现为乏力、怕冷、体重增加、记忆力减退、黏液性水肿|
|甲状腺结节/肿瘤|良性结节（增生性、囊性）、甲状腺癌（乳头状癌等）|遗传、碘摄入异常、激素紊乱、辐射暴露|多数良性结节无症状，恶性结节可能伴结节增大、声音嘶哑、颈部淋巴结肿大|
|自身免疫性甲状腺病|桥本甲状腺炎、格雷夫斯病|自身免疫功能紊乱，产生甲状腺自身抗体，攻击腺体组织|桥本氏多进展为甲减，格雷夫斯病表现为甲亢，均伴自身抗体阳性|
## 2.2 流行病学与高危人群

- **高发人群**：女性（尤其育龄期、更年期女性），激素水平波动易诱发自身免疫性甲状腺病；有甲状腺疾病家族史者，直系亲属患病风险较普通人群高2-5倍；长期碘摄入异常（过量或不足）者；长期暴露于电离辐射（如头颈部放疗史）者；长期精神压力大、熬夜、情绪波动剧烈者；长期服用含碘药物、抗甲状腺药物者。

- **人群与地域特征**：碘缺乏地区甲减、地方性甲状腺肿发病率较高，碘过量地区自身免疫性甲状腺病、甲状腺结节发病率升高；城市人群患病率高于农村，与精神压力、诊疗水平提升（检出率增加）相关；甲状腺癌在中青年女性中高发，乳头状癌占比超80%，预后较好。

- **疾病负担**：多数甲状腺疾病为慢性病程，需长期用药、定期监测，部分患者需手术治疗，医疗费用较高；甲减、甲亢若控制不佳，可影响心血管、生殖、神经等系统功能，降低生活质量；甲状腺癌虽预后较好，但术后需长期激素替代治疗，部分晚期患者预后较差，占用一定医疗资源。

# 三、甲状腺疾病病因与发病机制

## 3.1 核心病因

甲状腺疾病的发生是遗传因素、环境因素、自身免疫功能紊乱及激素调节异常共同作用的结果，不同分型病因各有侧重：

- **遗传因素**：甲状腺疾病具有明显家族聚集性，桥本甲状腺炎、格雷夫斯病、甲状腺癌均与遗传基因相关，HLA基因多态性是自身免疫性甲状腺病的重要遗传易感因素；直系亲属患有甲状腺疾病者，个体患病风险显著升高，且可能出现家族性甲状腺结节、自身抗体阳性聚集现象。

- **环境与生活方式因素**：碘摄入异常是核心诱因，碘不足导致甲状腺激素合成原料缺乏，诱发地方性甲状腺肿、甲减；碘过量可诱发自身免疫性甲状腺病、甲状腺结节；长期精神压力大、熬夜、慢性应激，可影响下丘脑-垂体-甲状腺轴功能，导致激素分泌紊乱；电离辐射暴露（如头颈部放疗、核辐射）是甲状腺癌的明确诱因，可损伤甲状腺细胞，诱发基因突变。

- **自身免疫功能紊乱**：自身免疫性甲状腺病的核心病因，机体产生甲状腺自身抗体（如甲状腺过氧化物酶抗体TPOAb、甲状腺球蛋白抗体TgAb、促甲状腺激素受体抗体TRAb），攻击甲状腺组织或受体；TPOAb、TgAb损伤甲状腺滤泡细胞，导致激素分泌不足（桥本氏）；TRAb刺激甲状腺激素过度分泌（格雷夫斯病）。

- **其他诱因**：甲状腺手术切除、放射性碘治疗后，可导致甲状腺组织减少，引发术后甲减；长期服用抗甲状腺药物（如甲巯咪唑）、锂剂、胺碘酮等，可干扰甲状腺激素合成或分泌，诱发药物性甲亢/甲减；甲状腺炎（病毒性、细菌性）可导致甲状腺滤泡细胞损伤，引发一过性甲亢，后续可能进展为甲减。

## 3.2 核心发病机制

甲状腺疾病的核心发病机制可概括为甲状腺激素分泌失衡、自身免疫性损伤、甲状腺组织异常增生三大类，最终导致甲状腺功能及结构异常，影响全身代谢。

激素分泌失衡：甲状腺激素分泌受下丘脑-垂体-甲状腺轴调控，下丘脑分泌TRH，促进垂体分泌TSH，TSH调控甲状腺合成释放T3、T4，形成负反馈调节。甲亢时，TRAb刺激甲状腺组织，或甲状腺结节自主分泌过多T3、T4，反馈抑制TSH分泌，导致血中T3、T4升高、TSH降低；甲减时，甲状腺组织损伤导致T3、T4分泌不足，反馈刺激垂体分泌TSH，导致血中TSH升高、T3、T4降低。

自身免疫性损伤：自身抗体攻击甲状腺组织引发持续性损伤，桥本甲状腺炎中，TPOAb、TgAb与甲状腺滤泡细胞结合，激活免疫反应，导致滤泡细胞破坏、凋亡，甲状腺激素合成能力逐步下降，最终进展为甲减；格雷夫斯病中，TRAb与甲状腺TSH受体结合，模拟TSH作用，刺激甲状腺激素过量分泌，同时引发眼眶后组织免疫反应，导致突眼。

甲状腺组织异常增生：碘摄入异常、激素紊乱、遗传因素可导致甲状腺组织增生，形成结节；良性结节多为滤泡细胞增生、囊性变，一般无激素分泌功能；恶性结节由甲状腺细胞基因突变引发，异常增殖形成肿瘤，部分可侵犯周围组织、发生淋巴结转移；长期炎症刺激也可导致甲状腺弥漫性增生、肿大，影响甲状腺功能。

# 四、甲状腺疾病预防与健康管理

## 4.1 一级预防（病因预防）

针对普通人群及高危群体，通过综合干预减少致病因素，降低甲状腺疾病发病风险，核心措施如下：

- **碘摄入管理**：保持适量碘摄入，避免过量或不足，成人每日碘摄入量推荐为150μg，孕妇、哺乳期女性增至200μg；日常通过加碘盐补充碘，适量食用海带、紫菜等含碘食物，避免长期大量摄入；碘缺乏地区加强碘营养补充，碘过量地区减少含碘食物及含碘药物使用。

- **生活方式干预**：保持规律作息，避免熬夜，保证每日7-8小时睡眠，维持下丘脑-垂体-甲状腺轴功能稳定；调节情绪，缓解精神压力，避免长期焦虑、抑郁，通过运动、冥想等方式调节心理状态；避免头颈部暴露于电离辐射，必要时做好防护措施；戒烟限酒，避免吸烟加重格雷夫斯病突眼症状。

- **高危人群防控**：有甲状腺疾病家族史者、自身免疫病史者，定期筛查甲状腺功能及自身抗体；长期服用含碘药物、抗甲状腺药物者，遵医嘱监测甲状腺功能，及时调整用药；女性育龄期、更年期定期体检，关注甲状腺功能变化；头颈部放疗史者，每年进行甲状腺超声及功能筛查，警惕甲状腺癌。

## 4.2 二级预防（早筛查、早干预）

- **定期筛查**：普通人群每年进行甲状腺超声筛查，40岁以上女性、高危人群每6-12个月复查；同步检测甲状腺功能（TSH、T3、T4）及自身抗体（TPOAb、TgAb），早期发现亚临床甲亢/甲减、无症状结节；甲状腺结节患者根据结节大小、形态，定期复查超声，评估结节性质。

- **症状监测与早期干预**：警惕甲状腺疾病早期信号，甲亢表现为心慌、手抖、消瘦、怕热，甲减表现为乏力、怕冷、体重增加、记忆力减退，结节增大可能伴颈部压迫感、声音嘶哑，出现症状及时就医排查；亚临床甲亢/甲减患者，虽无明显症状，仍需定期监测，必要时药物干预，避免进展为临床甲亢/甲减。

- **结节分级管理**：甲状腺超声根据结节大小、形态、边界、钙化等特征分级（TI-RADS分级），1-2级为良性，定期随访；3级良性可能性大，每6-12个月复查；4级及以上存在恶性风险，需进一步行细针穿刺活检，明确性质后干预。

## 4.3 三级预防（慢病管理与并发症防治）

- **规范治疗与用药**：甲亢患者遵医嘱使用抗甲状腺药物、放射性碘或手术治疗，定期监测甲状腺功能，调整用药剂量，避免药物过量引发甲减；甲减患者长期服用左甲状腺素钠替代治疗，从小剂量开始逐步加量，定期复查TSH，维持指标正常；甲状腺癌术后患者，根据病情进行放射性碘治疗、激素抑制治疗，定期复查Tg、超声，监测复发。

- **长期监测与随访**：甲亢/甲减患者每3-6个月复查甲状腺功能，病情稳定后可延长至6-12个月；甲状腺结节患者根据分级定期复查超声，良性结节每年复查，恶性结节术后每3-6个月复查；自身免疫性甲状腺病患者，定期监测自身抗体及甲状腺功能，及时发现功能异常。

- **并发症防治**：甲亢患者控制心率，避免劳累，预防甲亢性心脏病、骨质疏松；甲减患者及时补充甲状腺激素，预防黏液性水肿、心血管疾病、生殖系统紊乱；甲状腺癌术后患者，做好颈部护理，避免感染，遵医嘱进行功能锻炼，预防颈部粘连、声音嘶哑等并发症。

# 五、甲状腺疾病诊断方法

## 5.1 临床诊断（初步筛查）

结合患者病史、症状、体格检查及流行病学史进行初步判断，为后续辅助检查提供方向，适用于基层医疗机构首诊及社区筛查。

诊断要点：① 病史采集：重点询问家族病史、碘摄入史、用药史、放疗史、既往甲状腺病史，以及症状特点（代谢异常、颈部不适、眼部症状等出现时间、持续规律）；② 体格检查：触摸甲状腺大小、质地、有无结节、压痛，检查颈部淋巴结是否肿大，评估眼部体征（突眼、眼睑水肿），初步判断病情类型。

## 5.2 实验室与特殊检查（辅助确诊与病情评估）

|诊断方法|检测目标/用途|结果解读与诊断标准|注意事项|
|---|---|---|---|
|甲状腺功能检测|评估甲状腺激素分泌水平，诊断甲亢/甲减|甲亢：TSH降低，FT3、FT4升高；甲减：TSH升高，FT3、FT4降低；亚临床甲亢：TSH降低，FT3、FT4正常；亚临床甲减：TSH升高，FT3、FT4正常。|需空腹采血，避免剧烈运动、情绪激动，采血前停用影响甲状腺功能的药物（遵医嘱）。|
|甲状腺自身抗体检测|鉴别自身免疫性甲状腺病，评估病情|TPOAb、TgAb阳性：多见于桥本甲状腺炎；TRAb阳性：多见于格雷夫斯病，是甲亢病因鉴别核心指标。|采血前无需特殊准备，抗体滴度与病情活动度相关，需结合甲状腺功能综合判断。|
|甲状腺超声检查|评估甲状腺形态、大小、结构，筛查结节、占位|通过TI-RADS分级评估结节良恶性风险，1-2级良性，3级低风险，4级及以上高风险；可显示甲状腺弥漫性增生、囊性变、钙化等。|无需空腹，检查时充分暴露颈部，避免佩戴项链等饰品；结节较大者需结合其他检查明确性质。|
|甲状腺细针穿刺活检（FNA）|明确甲状腺结节良恶性，指导治疗方案|病理结果分为良性、可疑恶性、恶性、无法明确，恶性结果需手术治疗，良性结果定期随访。|穿刺前评估凝血功能，避免出血；术后按压穿刺部位15-20分钟，预防血肿，监测颈部不适症状。|
|甲状腺核素扫描|评估甲状腺功能及结节性质（热结节/冷结节）|热结节：结节摄取核素能力强，多为良性功能性结节；冷结节：结节无摄取能力，恶性风险略高，需进一步活检。|检查前停用含碘食物、药物2-4周；孕妇、哺乳期女性禁用；检查后避免接触婴幼儿，遵医嘱防护。|
# 六、甲状腺疾病治疗方案

## 6.1 治疗原则

甲状腺疾病治疗以“调节甲状腺激素平衡、去除病灶、控制症状、预防并发症、降低复发风险”为核心，根据疾病类型、病情严重程度、患者年龄及合并症制定个体化方案。甲亢治疗以抑制甲状腺激素分泌、缓解症状为主；甲减以激素替代治疗为主，维持甲状腺功能正常；良性结节以定期随访为主，必要时手术切除；甲状腺癌以手术治疗为主，结合放射性碘、激素抑制治疗等综合手段。

## 6.2 药物治疗（核心干预手段）

药物治疗适用于甲亢、甲减、自身免疫性甲状腺病等，需根据病情选择药物，严格遵医嘱使用，核心药物类别及应用如下：

|药物类别|代表药物|适用场景|用法要点与不良反应|
|---|---|---|---|
|抗甲状腺药物|甲巯咪唑、丙硫氧嘧啶|格雷夫斯病甲亢、轻度甲亢，术前准备、放射性碘治疗前辅助治疗|甲巯咪唑每日1次服用，丙硫氧嘧啶分3次服用；不良反应为粒细胞减少、皮疹、肝功能损伤，定期监测血常规、肝功能，出现发热、咽痛及时停药。|
|甲状腺激素替代药物|左甲状腺素钠|各种原因引起的甲减、甲状腺癌术后激素抑制治疗|晨起空腹服用，与食物间隔1小时以上；从小剂量开始逐步加量，过量引发甲亢症状；定期监测TSH，调整剂量，终身服药者不可擅自停药。|
|β受体阻滞剂|普萘洛尔、美托洛尔|甲亢辅助治疗，缓解心慌、手抖、心率过快等症状|根据心率调整剂量，甲亢症状控制后逐步停药；支气管哮喘、心动过缓者禁用，监测心率、血压，避免突然停药。|

## 6.3 手术与放射性碘治疗（复杂病例干预）

- **手术治疗**：适用于甲状腺癌、巨大甲状腺结节（有压迫症状）、毒性结节性甲状腺肿、甲亢药物治疗无效者。手术方式包括甲状腺全切术、次全切术，甲状腺癌患者需同时清扫颈部淋巴结；术后重点监测甲状腺功能、血钙（预防甲状旁腺损伤），做好伤口护理，预防感染、出血、喉返神经损伤等并发症。

- **放射性碘治疗**：适用于格雷夫斯病甲亢、甲状腺癌术后残留/复发、毒性结节性甲状腺肿。利用放射性碘破坏甲状腺滤泡细胞，减少甲状腺激素分泌；甲亢患者治疗后可能出现一过性甲亢加重，后续可能进展为甲减，需长期替代治疗；甲状腺癌术后患者需在治疗前停用左甲状腺素钠，治疗后遵医嘱恢复用药，定期监测Tg、TSH。

## 6.4 急性并发症急救与对症治疗

- **甲亢危象急救**：甲亢严重急性并发症，多由感染、手术应激、擅自停药诱发。立即卧床休息，物理降温；静脉输注丙硫氧嘧啶（抑制激素合成）、碘剂（抑制激素释放）；使用β受体阻滞剂控制心率，糖皮质激素减轻应激反应；补充液体、电解质，纠正紊乱；去除诱因，转入ICU监护，死亡率较高，需早期识别干预。

- **黏液性水肿昏迷**：甲减严重急性并发症，多见于重症甲减未治疗者，诱因包括感染、寒冷、创伤。立即静脉输注左甲状腺素钠，快速补充甲状腺激素；保暖、吸氧，纠正低血压、低血糖；抗感染治疗，去除诱因；密切监测生命体征、电解质，预防多脏器功能衰竭。

- **急性甲状腺炎对症治疗**：细菌性甲状腺炎给予足量抗生素治疗，形成脓肿时需穿刺引流或手术切开；病毒性甲状腺炎以对症治疗为主，疼痛明显者给予非甾体抗炎药，症状严重者短期使用糖皮质激素，监测甲状腺功能，必要时给予激素替代治疗。

## 7.2 转诊指征

出现以下情况，需及时转诊至上级医院内分泌科、甲状腺外科治疗：

- 突发甲亢危象、黏液性水肿昏迷，病情危急，需紧急抢救；

- 甲状腺结节TI-RADS分级4级及以上，或细针穿刺活检提示可疑恶性、恶性，需手术治疗；

- 甲亢药物治疗无效、反复发作，或出现严重药物不良反应，需调整治疗方案（放射性碘、手术）；

- 甲状腺肿大明显，出现呼吸困难、吞咽困难、声音嘶哑等压迫症状，需手术干预；

- 格雷夫斯病伴严重突眼，需专科评估，进行免疫抑制、眼部减压等治疗；

- 甲状腺癌术后复发，或需要放射性碘治疗、靶向治疗，基层医疗机构无法开展；

- 儿童、青少年甲状腺疾病，需专科制定个体化治疗方案，指导生长发育监测。
