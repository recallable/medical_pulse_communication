# 常见传染病医疗知识库

# 一、常见传染病概述

常见传染病是由细菌、病毒、真菌、寄生虫等病原体引起，可通过各种传播途径在人与人、动物与人、动物与动物之间相互传播的一组疾病。其核心病理机制为病原体侵入机体后繁殖扩散，引发组织损伤、炎症反应或毒素中毒，临床常表现为发热、乏力、皮疹、呼吸道/消化道/泌尿道症状等，部分传染病可累及多脏器，严重时导致器官衰竭、死亡。传染病具有传播性、流行性、季节性等特点，对个体健康、公共卫生安全及社会秩序造成显著影响，规范防控与早期治疗是降低危害的关键。

该类疾病人群覆盖广泛，不同年龄段、体质人群易感程度各异，如儿童易感手足口病、麻疹，老年人及免疫低下者易感肺炎、流感，青壮年高发病毒性肝炎、性传播疾病。病因主要与病原体感染、传播途径暴露、机体免疫力低下相关，多数传染病病程具有自限性，部分需针对性抗感染治疗，接种疫苗是预防多数传染病最有效的手段，早发现、早隔离、早治疗可显著降低传播风险与病死率。

# 二、常见传染病基础常识

## 2.1 疾病分型与核心特征

病毒性传染病是由各类病毒感染引发的传染性疾病，具有传播速度快、流行范围广、变异能力强等特点，是常见传染病中危害公共卫生安全的主要类别之一。根据传播途径可进一步细分亚型，各亚型核心特征及典型疾病如下：

### 2.1.1 呼吸道传播病毒性传染病

此类疾病主要通过呼吸道飞沫、气溶胶传播，冬春季高发，易在密闭环境引发聚集性疫情，人群普遍易感，儿童、老年人及免疫低下者症状更重。

|典型疾病|核心病原体|核心临床特征|传播要点|
|---|---|---|---|
|流行性感冒|甲、乙、丙型流感病毒|起病急，高热（39-40℃）、全身肌肉酸痛、头痛乏力明显，伴轻中度咳嗽、咽痛，病程约5-7天，重症可引发肺炎。|飞沫传播为主，接触被污染物品可间接传播，病毒易变异，每年需更新疫苗。|
|新型冠状病毒感染|新型冠状病毒（SARS-CoV-2）|以发热、干咳、乏力为核心症状，部分伴咽痛、嗅觉味觉减退，重症进展快，可引发急性呼吸窘迫综合征。|呼吸道飞沫、密切接触传播，气溶胶传播在密闭环境下存在风险，不同变异株传染性差异较大。|
|麻疹|麻疹病毒|前驱期有发热、咳嗽、眼结膜充血，出疹期按耳后、发际至全身顺序出疹，伴口腔麻疹黏膜斑，易并发肺炎。|飞沫传播极强，人群普遍易感，接种麻疹疫苗可有效预防，病后可获终身免疫。|
### 2.1.2 消化道传播病毒性传染病

此类疾病经粪-口途径传播，与不洁饮食、饮水及个人卫生习惯密切相关，夏秋季高发，儿童为主要易感人群，多数病程具有自限性。

|典型疾病|核心病原体|核心临床特征|传播要点|
|---|---|---|---|
|甲型病毒性肝炎|甲型肝炎病毒（HAV）|起病急，乏力、食欲减退、厌油腻，伴皮肤巩膜黄染、肝功能异常，多数患者可自愈，无慢性化倾向。|粪-口传播，食用被污染的食物（如毛蚶）、饮水为主要传播途径，接种甲肝疫苗可预防。|
|手足口病|肠道病毒71型（EV71）、柯萨奇病毒A16型|多见于5岁以下儿童，手足、口腔黏膜出现疱疹，伴低热、食欲不振，重症多由EV71感染引发，可累及神经系统。|粪-口、接触传播，玩具、餐具、衣物被污染后易造成传播，做好环境消毒可降低风险。|
|诺如病毒感染性腹泻|诺如病毒|突发呕吐、腹泻，粪便为水样便，伴低热、腹痛，病程短（1-3天），传染性极强。|粪-口传播，易在学校、托幼机构、集体食堂暴发，消毒以含氯消毒剂为主。|
### 2.1.3 血液/体液传播病毒性传染病

此类疾病主要通过血液、体液及母婴传播，多数可转为慢性感染，病程迁延，长期可累及脏器功能，需长期规范管理。

|典型疾病|核心病原体|核心临床特征|传播要点|
|---|---|---|---|
|乙型病毒性肝炎|乙型肝炎病毒（HBV）|急性期可无症状或伴乏力、肝区不适，慢性感染者可进展为肝硬化、肝癌，需定期监测肝功能及病毒载量。|血液、母婴、性接触传播，避免共用注射器、剃须刀，新生儿接种乙肝疫苗+免疫球蛋白可阻断母婴传播。|
|丙型病毒性肝炎|丙型肝炎病毒（HCV）|多数患者急性期无症状，易慢性化，长期感染可导致肝纤维化、肝硬化，目前可通过抗病毒药物治愈。|主要经血液传播，共用注射器、不安全输血为主要途径，性接触、母婴传播风险较低。|
### 2.1.4 其他传播途径病毒性传染病

除上述三类外，部分病毒性传染病通过虫媒、接触等特殊途径传播，发病具有地域聚集性，与媒介生物分布密切相关。

|典型疾病|核心病原体|核心临床特征|传播要点|
|---|---|---|---|
|流行性乙型脑炎|乙型脑炎病毒（JEV）|由蚊虫传播，起病急，高热、头痛、呕吐，重症可出现意识障碍、抽搐，病死率高，部分遗留后遗症。|蚊媒传播（主要为三带喙库蚊），夏秋季高发，接种乙脑疫苗、灭蚊是主要防控手段。|
|狂犬病|狂犬病病毒（RABV）|潜伏期长短不一，发病后表现为恐水、怕风、咽肌痉挛，最终因呼吸、循环衰竭死亡，病死率100%。|被感染动物（犬、猫等）咬伤、抓伤传播，暴露后及时接种狂犬病疫苗+免疫蛋白是唯一预防手段。|
细菌性传染病是由各类细菌感染引发的传染性疾病，多数可通过抗菌药物治愈，部分细菌易产生耐药性，且部分疾病传染性强、重症率高，是公共卫生防控的重要类别。根据传播途径可进一步细分亚型，各亚型核心特征及典型疾病如下：

### 2.1.5 呼吸道传播细菌性传染病

此类疾病主要通过呼吸道飞沫传播，冬春季高发，易在人群聚集场所暴发，儿童、老年人及免疫低下者为重症高发群体，多数需针对性使用抗生素治疗。

|典型疾病|核心病原体|核心临床特征|传播要点|
|---|---|---|---|
|肺结核|结核分枝杆菌|低热、盗汗、乏力、咳嗽咳痰，部分伴咯血，病程迁延，易转为慢性，重症可累及多脏器，需长期联合抗结核治疗。|飞沫传播为主，患者咳嗽、咳痰时病原体随飞沫扩散，密切接触者感染风险高，接种卡介苗可降低儿童发病风险。|
|流行性脑脊髓膜炎|脑膜炎双球菌|起病急骤，高热、剧烈头痛、喷射性呕吐，伴皮肤瘀点瘀斑，重症可出现休克、意识障碍，病死率高。|呼吸道飞沫传播，儿童及青少年易感，冬春季高发，接种流脑疫苗可有效预防，早用抗生素可降低重症率。|
|百日咳|百日咳杆菌|以阵发性痉挛性咳嗽为特征，咳嗽末期伴鸡鸣样吼声，病程长达数周，婴幼儿易并发肺炎、窒息。|飞沫传播，5岁以下儿童易感，接种百日咳疫苗（联合百白破疫苗）是主要预防手段，患者需隔离至症状消失。|
### 2.1.6 消化道传播细菌性传染病

此类疾病经粪-口途径传播，与不洁饮食、饮水、餐具污染及个人卫生习惯密切相关，夏秋季高发，多数起病较急，病程多为自限性，对症+抗菌治疗可缩短病程。

|典型疾病|核心病原体|核心临床特征|传播要点|
|---|---|---|---|
|细菌性痢疾|痢疾杆菌（志贺菌属）|腹痛、腹泻，粪便为黏液脓血便，伴里急后重、高热，重症可出现感染性休克、中毒性脑病。|粪-口传播，食用被污染的食物、饮水为主要途径，集体食堂、学校易暴发，做好饮食消毒可阻断传播。|
|伤寒|伤寒杆菌|起病缓慢，持续高热、相对缓脉，伴乏力、食欲减退、腹胀，部分患者胸腹部出现玫瑰疹，重症可并发肠出血、肠穿孔。|粪-口传播，污染的水源、食物为主要传播媒介，潜伏期长（7-23天），患者粪便需严格消毒处理。|
|副伤寒|副伤寒甲、乙、丙杆菌|症状与伤寒相似但较轻，发热时间短，玫瑰疹出现早，并发症少，病程较短，预后良好。|传播途径与伤寒一致，多因食用被污染的肉类、蛋类传播，预防措施同伤寒。|
### 2.1.7 虫媒/接触传播细菌性传染病

此类疾病通过虫媒叮咬、皮肤黏膜接触或接触污染物品传播，发病具有地域聚集性，部分为烈性传染病，需严格隔离及应急防控。

|典型疾病|核心病原体|核心临床特征|传播要点|
|---|---|---|---|
|鼠疫|鼠疫杆菌|烈性传染病，分腺型、肺型，腺型表现为局部淋巴结肿大、疼痛，肺型伴高热、咳嗽咯血，病死率极高。|主要经鼠蚤叮咬传播，肺型可通过飞沫传播，接触患者体液、尸体也可感染，发现后需立即上报并隔离。|
|炭疽|炭疽杆菌|分皮肤型、肺型、肠型，皮肤型表现为皮肤溃疡、焦痂，肺型、肠型病情凶险，可引发感染性休克。|接触病畜、病畜产品传播，皮肤黏膜破损时易感染，职业接触者（牧民、屠宰工）为高危人群，接种炭疽疫苗可预防。|
其他病原体传染病（真菌、寄生虫类）作为补充类别，核心特征如下：

|分型类别|典型疾病|核心病因（病原体）|核心临床特征及传播途径|
|---|---|---|---|
|真菌性传染病|念珠菌病、曲霉菌病、隐球菌病|白色念珠菌、曲霉菌、隐球菌|多见于免疫低下者，念珠菌病可引发口腔、阴道黏膜感染或内脏感染；曲霉菌病好发于肺部，表现为咳嗽、咯血；隐球菌病易累及中枢神经系统，伴头痛、呕吐，主要经呼吸道传播。|
|寄生虫病|疟疾、血吸虫病、蛔虫病、弓形虫病|疟原虫、血吸虫、蛔虫、弓形虫|疟疾经按蚊叮咬传播，表现为周期性寒战高热；血吸虫病经接触疫水传播，累及肝脏脾脏；蛔虫病经粪-口传播，儿童高发，伴腹痛、营养不良；弓形虫病经接触猫科动物粪便、生食肉类传播，孕妇感染可影响胎儿。|
## 2.2 流行病学与高危人群

- **高发人群**：儿童及青少年易感麻疹、手足口病、流脑，免疫系统发育不完善；老年人、慢性病患者（糖尿病、慢阻肺）、免疫缺陷者（化疗患者、艾滋病患者）易感流感、肺炎、肺结核，免疫力低下；医护人员、传染病密切接触者为职业暴露高危人群；有不洁饮食史、接触疫水史者易患肠道传染病、血吸虫病。
      

- **人群与地域特征**：呼吸道传染病（流感、新冠、麻疹）冬春季高发，传播速度快，易引发聚集性疫情；肠道传染病（甲肝、菌痢）夏秋季高发，与气温高、食物易变质相关；疟疾、血吸虫病多见于热带、亚热带及疫区，与媒介生物分布相关；经济欠发达、卫生条件差地区，传染病发病率较高，传播风险更高。
      

- **疾病负担**：传染病可快速传播引发暴发流行，占用大量医疗资源，影响社会生产生活；部分传染病（乙肝、肺结核）病程迁延，易转为慢性，导致劳动力丧失；烈性传染病（鼠疫、霍乱）可引发公共卫生危机，威胁人群生命安全；疫苗接种普及后，多数传染病发病率显著下降，但耐药菌株、新型病原体仍带来防控挑战。
      

# 三、常见传染病病因与发病机制

## 3.1 核心病因

常见传染病病因主要为病原体感染、传播途径暴露及机体免疫力低下，三者协同作用导致发病，核心病因可分为以下几类：

- **病原体因素**：病原体的毒力、数量及侵袭力是致病关键。毒力强的病原体（如脑膜炎双球菌）少量侵入即可发病；病毒易变异（如流感病毒），可逃避机体免疫，导致反复感染；细菌可产生毒素（如痢疾杆菌肠毒素），直接损伤组织器官引发症状。
      

- **传播途径因素**：不同传染病传播途径各异，呼吸道传染病经飞沫、气溶胶传播，易在密闭环境扩散；肠道传染病经粪-口传播，与不洁饮食、饮水相关；血液传播疾病（乙肝、丙肝）经输血、共用注射器、母婴传播；虫媒传染病（疟疾）经媒介生物叮咬传播，传播风险与媒介密度相关。
      

- **宿主因素**：机体免疫力低下是发病重要诱因，儿童、老年人、慢性病患者、免疫抑制药使用者，免疫防御功能弱，易受病原体侵袭；个体卫生习惯差（不洗手、生食食物）、暴露于疫区、缺乏疫苗保护，可显著增加感染风险；遗传因素影响机体对部分病原体的易感性，如部分人群对乙肝病毒抵抗力更强。
      

## 3.2 核心发病机制

常见传染病核心发病机制围绕病原体入侵、机体免疫应答及组织损伤展开，不同病原体机制各有侧重，共同病理过程如下：

病原体入侵与定植：病原体通过相应传播途径侵入机体后，在适宜部位定植繁殖，如呼吸道传染病病原体在呼吸道黏膜定植，肠道传染病病原体在肠道黏膜繁殖。部分病原体（如乙肝病毒）可侵入细胞内，逃避机体免疫清除，持续复制引发慢性感染；细菌可通过产生黏附因子附着于组织表面，进一步侵袭深层组织。

免疫应答与炎症反应：机体识别病原体后启动免疫防御，先天免疫（吞噬细胞、补体）快速清除病原体，后天免疫（抗体、T细胞）针对性杀伤病原体及感染细胞。免疫应答过强时，可引发过度炎症反应，导致组织损伤，如新冠重症患者的“细胞因子风暴”，累及肺部及多脏器；免疫功能低下时，无法有效清除病原体，导致感染迁延不愈，转为慢性。

毒素作用与组织损伤：部分细菌可产生外毒素（如百日咳毒素）、内毒素（如痢疾杆菌内毒素），外毒素直接作用于靶器官，引发特异性症状；内毒素激活机体炎症反应通路，导致发热、休克等全身症状。病毒感染可直接破坏宿主细胞，导致组织功能障碍，如肝炎病毒损伤肝细胞，引发肝功能异常；疟原虫寄生红细胞，导致红细胞破裂，引发贫血及高热。

# 四、常见传染病预防与健康管理

## 4.1 一级预防（病因预防）

针对普通人群及高危群体，通过综合干预阻断病原体传播、增强机体抵抗力，降低传染病发病风险，核心措施如下：

- **疫苗接种**：按国家免疫规划接种疫苗，如儿童接种麻疹、乙肝、流脑疫苗，老年人接种流感、肺炎疫苗，高危人群接种新冠疫苗、狂犬病疫苗；疫苗接种是预防病毒性传染病、部分细菌性传染病最经济有效的手段，可建立群体免疫屏障。
      

- **切断传播途径**：呼吸道传染病流行期减少聚集，佩戴口罩、勤通风、勤洗手；肠道传染病需注意饮食卫生，不生食、不喝生水，餐具消毒；血液传播疾病避免共用注射器、剃须刀，规范输血流程；虫媒传染病做好防护，使用驱蚊剂、避免接触疫水，消灭媒介生物（蚊虫、钉螺）。
      

- **增强机体抵抗力**：保持规律作息、均衡饮食，补充蛋白质、维生素，增强体质；适度运动，避免过度劳累；戒烟限酒，减少对免疫系统的损伤；慢性病患者规范治疗原发病，控制病情，降低感染风险。
      

## 4.2 二级预防（早筛查、早干预）

- **定期筛查与监测**：高危人群定期筛查，如乙肝患者家属、医护人员筛查乙肝五项，长期输血者筛查丙肝，疫区返回人员筛查疟疾、血吸虫病；医疗机构加强传染病监测，及时上报法定传染病，避免疫情扩散；个人出现疑似症状时，主动就医排查，不隐瞒旅居史、接触史。
      

- **早期信号识别与隔离**：警惕传染病早期信号，包括不明原因发热、咳嗽、腹泻、皮疹、乏力、头痛呕吐等；发现疑似病例及时隔离，避免接触他人，减少传播；呼吸道传染病患者咳嗽、打喷嚏时遮掩口鼻，肠道传染病患者做好粪便消毒，防止污染环境。
      

- **密切接触者管理**：对传染病密切接触者进行医学观察，如新冠密切接触者居家隔离观察，乙肝密切接触者接种疫苗并定期复查；观察期内出现症状及时就医，做到早发现、早诊断、早治疗，阻断传播链。
      

## 4.3 三级预防（慢病管理与并发症防治）

- **规范治疗与康复**：确诊传染病患者遵医嘱足量足疗程治疗，不可擅自停药，如肺结核需规律抗结核治疗6-9个月，乙肝需长期抗病毒治疗；病毒性传染病以对症支持治疗为主，细菌性传染病合理使用抗生素，避免耐药性产生；慢性传染病患者定期复查，监测病情变化，调整治疗方案。
      

- **并发症防治**：重症传染病患者及时住院治疗，监测脏器功能，预防呼吸衰竭、心力衰竭、感染性休克等并发症；如流感重症患者预防肺部感染，肝炎患者预防肝硬化、肝癌，疟疾患者预防脑型疟疾；康复期加强营养支持与护理，促进机体恢复，减少后遗症。
      

- **出院后管理**：慢性传染病患者出院后做好自我管理，养成良好卫生习惯，避免再次感染；定期随访，遵医嘱用药，不滥用药物；避免传染他人，如乙肝患者不与他人共用牙刷、餐具，肺结核患者治愈前避免密切接触儿童、老人。

# 六、常见传染病治疗方案

## 6.1 治疗原则

常见传染病治疗遵循“早诊断、早治疗、针对性干预、防止传播”原则。病毒性传染病以对症支持治疗为主，必要时使用抗病毒药物；细菌性传染病合理选用抗生素，足量足疗程使用，避免耐药性；寄生虫病使用抗寄生虫药物，同时对症处理症状；重症传染病加强脏器功能支持，预防并发症，降低病死率；治疗期间做好隔离，防止交叉感染。

## 6.2 药物治疗（核心干预手段）

药物治疗以杀灭/抑制病原体、缓解症状、控制感染为主，不同类型传染病用药重点不同，核心药物类别及应用如下：

## 6.3 综合治疗与护理

- **支持治疗**：卧床休息，保证充足睡眠，减少机体消耗；均衡饮食，补充水分、蛋白质、维生素，维持水电解质平衡，如腹泻患者补充口服补液盐，高热患者适当补液；免疫低下者可酌情使用免疫增强剂，辅助机体清除病原体。
      

- **对症护理**：高热患者采用物理降温（温水擦浴）或药物降温，避免高热惊厥；咳嗽咳痰患者翻身拍背，促进痰液排出；皮疹患者保持皮肤清洁，避免搔抓，防止继发感染；隔离患者做好心理疏导，缓解焦虑情绪，配合治疗。
      

- **隔离与消毒**：呼吸道传染病患者隔离至症状消失后3天，佩戴口罩，房间勤通风；肠道传染病患者隔离至粪便病原体检测阴性，粪便、呕吐物严格消毒；接触传染病患者后及时洗手、消毒，医疗器械专用，防止交叉感染。
      

# 七、鉴别诊断与转诊指征

## 7.1 常见疾病鉴别诊断

|鉴别疾病组合|核心鉴别点|
|---|---|
|流行性感冒与普通感冒|流感由流感病毒引起，高热、肌肉酸痛症状突出，全身症状重、呼吸道症状轻，可引发聚集性疫情；普通感冒多由鼻病毒引起，低热、鼻塞流涕症状明显，全身症状轻，传染性弱，病程自限。|
|细菌性痢疾与病毒性肠炎|菌痢由痢疾杆菌引起，伴黏液脓血便、腹痛明显，血常规白细胞升高，粪便培养可见痢疾杆菌；病毒性肠炎由轮状病毒、诺如病毒引起，粪便为水样便，无脓血，血常规白细胞正常或降低，病原学检测可确诊。|

## 7.2 转诊指征

出现以下情况，需及时转诊至上级医院感染科、传染病专科医院治疗：

- 高热持续不退（体温≥39℃）超过3天，伴剧烈头痛、呕吐、呼吸困难、意识模糊等重症表现，疑似重症传染病；

- 经基层治疗后症状无改善甚至加重，或出现多脏器功能异常（肝功能、肾功能、呼吸功能异常）；

- 疑似烈性传染病（鼠疫、霍乱、传染性非典型肺炎），需立即隔离并转诊至定点医院，同时上报疾控部门；

- 免疫低下者（艾滋病、化疗患者）感染传染病，病情进展快，易出现重症，需专科评估治疗；

- 传染病患者出现严重并发症（感染性休克、呼吸衰竭、心力衰竭、脑水肿），需重症监护治疗；

- 诊断不明确，无法排除疑难传染病，需上级医院进一步完善检查，明确诊断。
